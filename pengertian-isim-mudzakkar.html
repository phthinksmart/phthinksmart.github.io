14. Isim mudzakkar merubah ke jamak salim


1. Pengertian jamak mudzakkar salim

Jama’ mudzakkar salim adalah Jamak yang dibentuk dari isim mufrodnya yang digunakan untuk menunjukkan jenis laki-laki atau isim jama’ yang menunjukkan laki-laki dan beraturan dari mufrod yang menunjukkan laki-laki dan yang tidak ada ta’ marbutoh (ة ), yakni dengan menambahkan (ن و ) di akhirnya yang huruf sebelumnya dhomah, ataupun menambahkan (ين ) diakhirnya yang huruf sebelumnya kasrah.

Misalnya: مُؤْمِنٌ menjadi مُؤْمِنُوْنَ / مُؤْمِنِيْنَ tetapi jika tidak menunjukkan jenis laki-laki atau mudzakkar tidak bisa dibentuk menjadi jama’ mudzakkar salim.[3]

Contoh :


 مُؤْمِن : ٌmufrod

مُؤْمِنُوْنَ / مُؤْمِنِيْن : َJamak

Arti : Para laki-laki mukmin

كَافِر : ٌmufrod

كَافِرُوْنَ / كَافِرِيْن : َjamak

Arti : Para laki-laki kafir

مُسْلِم : ٌmufrod

مُسْلِمِيْنَ/ مُسْلِمُوْن : َjamak

Arti : Para laki-laki muslim

مَكْتَب : ٌmufrod

مَكْتَبُوْنِ/ مَكْتَبِيْن : َjamak

Arti : Banyak meja

مُدَرِّس : ٌmufrod

مُدَرِّسُوْنَ/ مُدَرِّسِيْن : َjamak

Arti : Para guru laki-laki atau guru-guru



2.      Cara pembentukan jama’ muadzakkar salim

Tata cara pembentukan jama’ mudzakkar salim adalah dengan mengubah isim mufrod yang menunjukkan laki-laki atau tidak ada ta’ marbutoh (ة ) diakhirnya dengan menambahkan (ن و ) di akhirnya yang huruf sebelumnya dhomah atau (ين ) diakhirnya yang huruf sebelumnya kasrah.

Contoh:


مُؤْمِنٌ (isim mufrod yang menunjukkan laki-laki) ditambahkan (ن و ) atau (ين )   sehingga diubah menjadi مُؤْمِنُوْنَ / مُؤْمِنِيْنَ.

مَوْقِدٌ (isim mufrod yang menunjukkan laki-laki) ditambahkan (ن و ) atau (ين ) sehingga diubah menjadi مَوْقِدُوْنَ/ مَوْقِدِيْنَ.

17. contoh isim mudzakkar dan muannats​


Contoh isim muzakkar :

- أَحْمَدُ

- مَكْتَبٌ

- دُرْجٌ

Contoh isim muannats :

- لَيْلَى

- كُرَّاسَةٌ

- كُتُبٌ

Pembahasan

Dalam bahasa Arab, isim berdasarkan jenisnya terbagi menjadi 2. Yaitu isim muzakkar dan isim muannats.

Ciri - Ciri Isim Muzakkar

Setiap nama - nama anak laki - laki adalah isim muzakkar.

Contoh : زَيْدٌ، عُمَرُ، عُثْمَانٌ

Tidak ada huruf ة diakhir katanya.

Contoh : كِتَابٌ، قَلَمٌ، مَكْتَبٌ

Berupa isim mufrod.

Contoh : -sama dengan contoh diatas-

Ciri - Ciri Isim Muannats.

Setiap nama - nama anak perempuan adalah isim muannats.

Contoh : زَيْنَبٌ، عَائِشَةُ، فَاطِمَةُ

Ada huruf ة diakhir katanya.

Contoh : مَدْرَسَةٌ، مِرْوَحَةٌ، طَلَّاسَةٌ

Isim jamak ghairu 'aaqil (kata benda yang tidak berakal dan jamak) adalah isim muannats.

Contoh : مَكَاتِبٌ (Jamak dari مَكْتَبٌ)

Ada beberapa isim yang tidak memiliki ciri ciri isim muannats, namun dianngap sebagai muannats.

Contoh : نَجْمٌ، شَمْسٌ، سَمَاءٌ

[ Untuk mengetahui isim apa saja yang dianggap muannats, kita perlu banyak membaca dan belajar dari berbagai sumber ]
