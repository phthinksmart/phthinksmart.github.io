9. Sebutkan contoh penggunaan variabel dan tipe data dalam algoritma!


Jawaban:

Tipe data adalah suatu kelompok yang mempunyai jenis-jenis tertentu. Dengan kata lain, tipe data adalah sebuah cara yang digunakan untuk menentukan jenis suatu data tersebut, kata lain dari hal ini ialah "deklarasi variabel". Dalam bahasa pemrograman mempunyai banyak jenis-jenis tipe data yang bisa digunakan. Namun setiap bahasa pemrograman mempunyai tipe data yang mungkin tidak ada di bahasa pemrograman lain. Namun secara umum, dalam bahasa pemrograman apapun mengenal 5 tipe data ini, yaitu integer, float, char, string, dan boolean. Masing masing tipe data ini mempunyai kegunaan masing-masing. Oleh karena itu, jika pada saat pengolahan data yang terdapat data yang mempunyai tipe data berbeda, maka harus dilakukan konversi terlebih dahulu. Berikut adalah penjelasan dari keempat tipe data.

Integer, tipe data integer ini digunakan untuk mendeklarasikan data/variable sebagai bilangan bulat. Misalkan nilai 10, 7,29,100, adn 22. Penggunaan tipe data integer biasanya digunakan untuk data yang akan dilakukan pengoprasian penjumlahan, pengurangan, pembagian, atau perkalian.

Float, seperti halnya integer, tipe data ini digunakan untuk data angka, namun untuk angka/nilai yang berupa pecahan. Jadi apabila ingin mendapatkan hasil yang terbaik pada suatu proses pengoprasian angka, maka kita bisa menggunakan tipe data float. Contohnya nilai 1.5, 3.5, 3.14, dan lain bilangan pecahan lainnya.

Char, char adalah tipe data yang digunakan untuk deklarasi variable berupa karakter. Karakter tersebut bisa berupa simbol, alfabet, atau angka.

String, khusus untuk string, tipe data yang digunakan untuk mendeklarasikan data yang berisi kalimat atau kata.

Boolean, tipe data ini sangat berbeda dengan yang lainnya. Karena tipe data boolean ini hanya digunakan untuk data yang mempunyai 2 nilai saja, yaitu true atau false.

14. Apa hubungan eis,data science,data mining dengan sistem informasi enterprise


Apa hubungan eis,data science,data mining dengan sistem informasi enterprise, sebelumnya mari kita bahas terlebih dahulu satu persatu istilah tersebut. EIS merupakan singkatan dari Executive Information System. EIS merupakan sistem yang dibuat untuk memudahkan seorang user untuk mengakses suatu data dan mendapatkan informasi dari data tersebut sehingga bisa mengambil keputusan berdasarkan informasi tersebut.


Sedangkan data science merupakan salah satu ilmu yang mempelajari bagaimana data tersebut diolah dan melakukan beberapa perubahan pada data untuk bisa mendapatkan informasi dari pengolahan data tersebut.


Untuk data mining sendiri merupakan salah satu ilmu yang mempelajari bagaiamana cara mendapatkan berbagai macam data yang tersebar di internet dengan cara yang seefektif mungkin.


Istilah yang terakhir adalah sistem informasi enterprise suatu aplikasi atau sistem yang dibuat untuk menginput mengolah dan menyajikan output bagi pemakaianya dan bersifat komersil.


Jadi apa hubungan eis,data science,data mining dengan sistem informasi enterprise adalah saling berhubungan dalam rangka mengolah data dan mendapatkan informasi dari berbagai macam data yang telah berhasil dikumpulkan. Semisal data piala dunia 2018, dengan berbagai macam teknik pengolahan data kita bisa memprediksi siapa yang akan menjadi juaranya.

#include <iostream>

#include <algorithm>

using namespace std;

float ratarata(int arr[], int n){

   float b = 0;

   for(int i = 0; i<n; i++){

          b = b + arr[i];

   }

   return b/n;

}

int terkecil(int arr[], int n){

  int i;

  int temp = arr[0];

  for(i = 1; i<n; i++) {

     if(temp > arr[i]) {

        temp = arr[i];

     }

  }

  return temp;

}

int terbesar(int arr[], int n)

{

   int i;

   int temp = arr[0];

   for (i = 1; i < n; i++)

       if (arr[i] > temp)

           temp = arr[i];

   return temp;

}

float median(int arr[], int n)

{

   sort(arr, arr+n);

 

   if (n % 2 != 0)

      return (float)arr[n/2];

     

   return (float)(arr[(n-1)/2] + arr[n/2])/2.0;

}
