15. Dari 40 siswa suatu madrasah, yang rutin sholat malam ada 22 siswa dan yang rutin sholat dhuha ada 23 siswa. Jika yang rutin sholat malam dan sholat dhuha ada 8 siswa, maka banyak siswa yang tidak rutin sholat malam dan sholat dhuha ada​


Materi : Himpunan

S = himpunan semesta ( 40 )

a = himpunan sholat malam ( 22 )

b = himpunan sholat dhuha ( 23 )

n = lakukan keduanya ( 8 )

ab^c = tidak lakukan keduanya ( ? )

\_______________________/

( a - n ) + ( b - n ) + n = S - ab^c

( 22 - 8 ) + ( 23 - 8 ) + 8 = 40 - ab^c

22 + 15 = 40 - ab^c

37 - 40 = - ab^c

- 3 = - ab^c

3 siswa = ab^c

30. Apa saja hadist tentang sholat Tahajud pada : -waktu tengah malam -sepertiga malam -ketika ayam jantan berkokok


Waktu tengah malam:
   Dari Abu Hurairah radhiyallahu anhu, ia berkata: Rasulullah shallallahu alaihi wasallam pernah ditanya: “Sholat apakah yang paling utama setelah sholat fardhu (yang lima waktu, pent) ?” beliau  menjawab: “Sholat yang paling utama  setelah sholat fardhu adalah sholat (sunnah) di tengah malam (sholat Tahajjud).”. (Diriwayatkan oleh imam Al-Bukhari dan Muslim).

   Sahabat bertanya:”ya Rosululloh…malam apakah yang paling didengar? “
