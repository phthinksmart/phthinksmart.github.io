3. (+) (-) father is not working today (/)


Jawaban:

~ Bahasa Inggris

Rumus Present Continuous :

+) S + to be + V - ing + ...

-) S + to be (not) + V - ing + ...

?) To be + S + V - ing + ...

Penyelesaian :

Father = S

is = to be

working = V - ing

today = keterangan waktu

+) Father is working today

-) Father is not working today

?) Is Father working today ?

===

Mapel : Bahasa Inggris

Materi : Past, present and future tenses

Kode : 9.5.5
