9. seorang penjahit mempunyai 60m kain sutra dan 40m kain katun.Penjahit tersebut akan membuat setelan baju muslim dan baju kerja untuk dijual. 1 setel baju muslim memerlukan 3 m kain sutra dan 1m katun sedangkan untuk kerja moal memerlukan 2m sutra dan 2m katun.keuntungan maksimum yang diperoleh jika harga 1 setel baju muslim rp.100.000 dan 1 setel baju kerja rp.60.000 adalah


dengan rumus keuntungan maksimum hasilnya adalah Rp 2.000.000. caranya : 3x + 2y = 60    Dengan Fungsi 100.000x + 60.000 y
1x + 2y = 40  
Petidakasamaanya, 3x + 2 y = 60 ..... x = 0; y = 30  -----> ( 0,30)
                                                            y =  0;  x = 20 -----> ( 20, 0 )
                                 1x + 2y = 40 .......x = 0;  y= 20 ---------> ( 0,20 )
                                                             y = 0;  x= 40 ---------> ( 40, 0 )
sehingga jika digamabarkan akan membentuk diagram dengan fungsi
 A. 100.000( 0 ) + 60.000 ( 30 ) = Rp 1800.000
B . 100.000 ( -20 ) + 60.000 ( 60 ) = Rp. 1600.000
c. 100.000 ( 20 ) + 60.000 ( 0 ) = Rp.2000.000. jadi keuntungan maksimum adalah Rp. 2000.000.

11. seorang pedagang pakaian ingin membeli pakaian pria dan pakaian wanita tidak lebih dari 15 stel. harga satu setel pakaian pria Rp.100.000,00 dan harga satu setel pakaian wanita Rp.150.000,00. jika uang uang tersedia hanya Rp.1.800.000,00, banyak pakaian pria yang dibeli x stel dan pkaian wanita y setel ,maka model matematika dari permasalahan di atas adalah


x = pakaian pria
y = pakaian wanita

x + y = 15
x = 15 - y

100000x + 150000y = 1800000
100000(15-y) + 150000y = 1800000
1500000 - 100000y+150000y= 1800000
50000y = 1800000 - 1500000
50000 y = 300000
y = 300000/50000
y = 6

x+y = 15
x + 6 = 15
x = 15 - 6
x = 9

pakaian pria = 9 stel
pakaian wanita = 6 stel

model matematika =
x + y ≤ 15
10x + 15 y ≤ 180

jadikan jawaban terbaik yaPria = x dan wanita = y
1) membeli pakaian pria dan wanita tidak lebih dari 15 step
x + y ≤ 15

2) harga
100.000x + 150.000y ≤ 1.800.000
10x + 15y ≤ 180
2x + 3y ≤ 36

3) x ≥ 0, y ≥ 0

jadi MODEL MATEMATIKA nya
x + y ≤ 15, 2x + 3y ≤ 36, x ≥ 0, y ≥ 0
