20. bagaimana katakana bed​


Huruf Katakana dari kata serapan 'Bed' adalah ベッド (Beddo).

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan

  Huruf Jepang ada 3 jenis yaitu Hiragana, Katakana, dan Kanji. Ketiga huruf ini merupakan huruf yang dipakai di Jepang. Selain ketiga huruf ini masih ada 1 jenis huruf lagi, tapi biasa hanya dipakai orang luar Jepang, yaitu Romaji. Huruf yang akan saya bahas kali ini hanya Katakana saja.

===

PENGERTIAN KATAKANA

  Katakana (カタカナ) merupakan salah satu dari tiga jenis huruf Jepang, yang digunakan untuk menulis kata-kata serapan luar Jepang. Katakana disebut juga 仮名 (Kana), sama seperti Hiragana. Katakana berawal dari Kanji, kemudian dikembangkan oleh rohaniawan Buddha dengan tujuan untuk melafalkan kitab suci Buddha yang bertuliskan Kanji Tiongkok.

---

FUNGSI KATAKANA

  Katakana memiliki beberapa fungsi, antara lain untuk :

Menulis kata-kata serapanMenulis onomatopeMemberi penekanan pada kata

---

CIRI DAN HURUF-HURUF KATAKANA

  Huruf-huruf Hiragana memiliki ciri yaitu garis-garis hurufnya yang kebanyakan lurus dan lebih kaku. Kemudian huruf Katakana juga dibagi menjadi Gojuuon, Dakuon, Handakuon, Youon, Sokuon, dan Chouon.

-

  1). 五十音 (Gojuuon)

  Gojuuon artinya "50 bunyi". Huruf-huruf Gojuuon terdapat 46 huruf yang nanti akan menjadi dasar untuk jenis-jenis huruf lainnya. Dinamakan 50 bunyi karena tabel huruf Gojuuon itu 5x10. Huruf-hurufnya antara lain sebagai berikut.

ア イ ウ エ オ (a - i - u - e - o)カ キ ク ケ コ (ka - ki - ku - ke - ko)サ シ ス セ ソ (sa - shi - su - se - so)タ チ ツ テ ト (ta - chi - tsu -te - to)ナ ニ ヌ ネ ノ (na - ni - nu - ne - no)ハ ヒ フ ヘ ホ (ha - hi - fu - he - ho)マ ミ ム メ モ (ma - mi - mu - me - mo)ヤ ユ ヨ (ya - yu - yo)ラ リ ル レ ロ (ra - ri - ru - re - ro)ワ ヲ (wa - wo)ン (n)

-

  2). 濁音 (Dakuon)

  Dakuon artinya "Bunyi keruh". Huruf Dakuon merupakan huruf Gojuuon deret [K, S, T, H] yang diberikan tanda seperti tanda kutip bernama Tenten di atas kanan huruf nya, untuk mengubah bunyi konsonan pada huruf tersebut menjad [G, Z, D, B]. Huruf Dakuon berjumlah 20 huruf. Huruf-hurufnya antara lain sebagai berikut.

ガ ギ グ ゲ ゴ (ga - gi - gu - ge - go)ザ ジ ズ ゼ ゾ (za - ji - zu - ze - zo)ダ ヂ ヅ デ ド (da - ji - dzu - de - do)バ ビ ブ ベ ボ (ba - bi - bu - be - bo)

-

  3). 半濁音 (Handakuon)

  Handakuon artinya "Bunyi setengah keruh". Huruf Handakuon merupakan huruf Gojuuon deret [H] yang diberikan tanda lingkaran kecil bernama Maru di atas kanan huruf nya, untuk mengubah bunyi konsonan pada huruf tersebut menjadi [P]. Huruf Handakuon berjumlah 5 huruf. Huruf-hurufnya antara lain sebagai berikut.

パ ピ プ ペ ポ (pa - pi - pu - pe - po)

-

  4). 拗音 (Youon)

  Youon artinya "Suara bengkok". Huruf Youon merupakan gabungan dari huruf-huruf berbunyi akhir 'i' dengan huruf 'ya', 'yu', atau 'yo' berukuran kecil. Huruf Youon berjumlah 21 huruf. Huruf-hurufnya antara lain sebagai berikut.

キャ キュ キョ (kya - kyu - kyo)ギャ ギュ ギョ (gya - gyu - gyo)シャ シュ ショ (sha - shu - sho)ジャ ジュ ジョ (ja - ju - jo)チャ チュ チョ (cha - chu - cho)ニャ ニュ ニョ (nya - nyu - nyo)ヒャ ヒュ ヒョ (hya - hyu - hyo)ビャ ビュ ビョ (bya - byu - byo)ピャ ピュ ピョ (pya - pyu - pyo)ミャ ミュ ミョ (mya - myu - myo)リャ リュ リョ (rya - ryu - ryo)

-

  5). 促音 (Sokuon)

  Sokuon merupakan huruf 'Tsu' berukuran kecil, untuk menggandakan konsonan berikutnya. Atau bisa juga untuk jeda sebelum membaca huruf berikutnya. Contohnya sebagai berikut.

 コク (Koku)  ||  コック (Kokku)

 サカ (Saka)  ||  サッカ (Sakka)

-

  6). 長音 (Chouon)

  Chouon merupakan bunyi vokal panjang dalam sebuah kata. Untuk huruf Katakana, cara memanjangkan suatu vokal yaitu dengan menambahkan tanda strip setelah huruf yang bunyi vokal nya ingin dipanjangkan. Contohnya sebagai berikut.

 サッカ (Sakka)  ||  サッカー (Sakkaa)

 ケキ (Keki)  ||  ケーキ (Keeki)

21. The man was a king.The negative sentence is……a. The man was not a kingb. The man is not a kingc. Was the man a king?d. The man did not a king​


Jawaban:

A. The man wasn't a king

Penjelasan:

Karena, kalau kalimat nya negatif tambahkan kata not di verb nya was jadi ''Wasn't''.

Was adalah bentuk V2 (past tense/lampau) dari is/am (to be), dan were adalah bentuk V2 dari are (to be). Was digunakan untuk subject yang tunggal (satu) dan untuk kata ganti I, he, she, it. Sedangkan were digunakan untuk subject yang jamak (lebih dari satu) dan kata ganti you, they, we.

30. jelaskan perbedaan sistem klasifikasi 4 king dom dan lima king dom​


jawaban:

1. Sistem 4 kingdom

Diperkenalkan oleh Herbert Faulkner Copeland, terdiri atas:  

a. Kingdom plantae, terdiri dari lumut, paku, tumbuhan biji, janmur, dan ganggang kecuali ganggang biru dan hijau.

b. Kingdom animalia

c. Kingdom protista, terdiri dari semua organisme uniseluler eukariotik (mempunyai membran inti sel)  

d. Kingdom monera  

Semua organisme bersel satu (uniseluler) tanpa selaput inti atau yang disebut dengan prokariotik, seperti bakteri, alga biru dan alga hijau.  

Kelebihan: sudah mampu membedakan antara organisme prokariotik dan eukariotik, sehingga masing-masing dikelompokkan dalam kingdom yang berbeda.

Kelemahan: belum bisa membedakan antara jamur (fungi) dengan tumbuhan. Fungi atau jamur bukan organisme autotrof layaknya tumbuhan melainkan organisme heterotrof yaitu tidak dapat mensintesis (membuat) makanannya sendiri.

2.Sistem 5 kingdom  

Ditemukan oleh Robert H. Whittaker menggunakan dasar tingkatan organisme, susunan sel, dan faktor nutrisinya, terdiri dari:  

a. Kingdom plantae, dibagi menjadi tubuhan berspora (lumut dan paku) dan tumbuhan berbiji (spermatophyta).

b. Kingdom animalia

c. Kingdom protista

d. Kingdom monera

e. Kingdom fungi

Memiliki ciri-ciri eukariotik, tidak berklorofil sehingga tidak berfotosintesis.

Contohnya: Mucor, Saccharomyces, Pleurotus (jamur tiram), Agaricus.

Kelebihan: jamur digolongkan kedalam kingdom tersendiri karena tidak mencerna  makanan seperti yang hewan lakukan, atau pun membuat makanan mereka sendiri seperti tumbuhan. Jamur   mengeluarkan enzim pencernaan di sekitar makanan mereka dan kemudian menyerapnya ke dalam sel.  

Sistem kingdom ini melengkapi sistem klasifikasi kingdom sebelumnya.

Kelemahan:  belum mampu mendefinisikan kingdom monera secara tepat sehingga didalam kelompok kingdom monera sendiri masih memiliki perbedaan yang cukup signifikan baik dalam hal RNA polymerase, RNA sequences, Introns, membran lipid dan lainnya.
