9. apa perbedaan web server, DNS server dan FTP server ?


Jawaban:

Web Server

Web Server dapat merujuk baik pada perangkat keras ataupun perangkat lunak yang menyediakan layanan akses kepada pengguna melalui protokol komunikasi HTTP atau HTTPS atas berkas-berkas yang terdapat pada suatu situs web dalam layanan ke pengguna dengan menggunakan aplikasi tertentu seperti peramban web.

DNS Server

DNS (Domain Name System) sendiri adalah sebuah sistem yang menyimpan informasi tentang nama host ataupun nama domain dalam bentuk basis data tersebar (distributed database) di dalam jaringan komputer, misalkan: Internet. DNS menyediakan alamat IP untuk setiap nama host dan mendata setiap server transmisi surat (mail exchange server) yang menerima surel (email) untuk setiap domain. Menurut browser Google Chrome, DNS adalah layanan jaringan yang menerjemahkan nama situs web menjadi alamat internet.

FTP Server

FTP Server adalah protokol internet yang digunakan untuk urusan pengiriman data dalam jaringan komputer, seperti upload dan download file yang dilakukan oleh FTP client dan FTP server.

24. Kasus atau contoh penerapan Server Softswitch PBX di Indonesia ?


Salah satu contoh penerapan Server Softswitch PBX di Indonesia ialah pada layanan telepon berbasis VOIP yang dimana biasanya terdapat di perkantoran. Selain itu konsep server softswitch ini sendiri memiliki peran sebagai penyedia layanan yang dimana dia digunakan oleh perusahaan dalam melayani komunikasi internet perusahaan yang bersangkutan ke perusahaan lain.

Pembahasan

Softswitch sendiri pada dasarnya dapat kita artikan sebagai suatu sekumpulan aplikasi dan juga perangkat protokol internet yang dimana dia memiliki fungsi untuk membantu kinerja perangkat lain untuk dapat mengakses layanan internet ataupun telekomunikasi secara umum pada sebuah jaringan IO.

PBX atau yang memiliki kepanjangan Private Branch Exchange sendiri pada dasarnya dapat kita artikan sebagai suatu sentral privat dengan fitur seperti sentral public.

VOIP atau yang memiliki kepanjangan Voice Over IP sendiri pada dasarnya dapat kita artikan sebagai suatu teknologi yang memungkinkan percakapan suara jarak jauh dengan cara melalui media internet.

SIP atau yang memiliki kepanjangan Session Intiation Protocol sendiri pada dasarnya dapat kita artikan sebagai suatu signalling protokol pada layer aplikasi yang berfungsi untuk membangun, serta memodifikasi, dan juga mengakhiri suatu sesi multimedia yang dimana dia mengikutsertakan satu atau beberapa pengguna.

Server adalah suatu sistem komputer yang menawarkan jenis layanan tertentu dalam suatu jaringan.

Web server adalah suatu perangkat lunak (software) yang memberikan perintah berbasis data yang berfungsi menerima permintaan dari http atau https pada klien web terkenal.
