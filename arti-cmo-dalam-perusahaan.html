Panjang sisi lainnya dari layang-layang dan kelilingnya adalah: c. 39 cm dan 130 cm

Penjelasan dengan langkah-langkahDiketahui: Perbandingan panjang sisi sebuah Layang-Layang adalah 2:3, ukuran panjang sisi yang lebih pendek 26 cmDitanya: Ukuran panjang sisi lainnya dan keliling layang-layang tersebut adalah?Jawab

Perbandingan sisi pendek dan sisi panjang layang-layang adalah  2 : 3, maka sisi panjang layang-layang adalah:

[tex]\frac{s.pendek}{s.panjang} =\frac{2}{3} \\\frac{26}{s.panjang} =\frac{2}{3} \\s.panjang=\frac{26}{2} (3)\\s.panjang=39cm[/tex]

Rumus untuk menentukan keliling layang-layang adalah:

(2 x sisi panjang) + (2 x sisi pendek)

= (2 x 26) + (2 x 39)

= 52 + 78

= 130 cm

Opsi E

Pembahasan :

penurunan pendapatan sebesar x% dari y.

pendapatan =(100%- x%)×y.

Berapa % peningkatan dari x ke y.

(y-x)/x ×100%

22. Dikethui untuk array memiliki banyak elemen A[23,17]. tentukan : a. Alamat untuk A[7,8] dengan lebar elemen 5 secara RMO dan CMO


Jawaban:

Untuk menentukan alamat A[7,8] dalam array A[23,17] dengan lebar elemen 5 secara RMO (Row-Major Order), kita dapat menggunakan rumus berikut:

Alamat RMO = Alamat Awal + [(Baris - 1) x Jumlah Kolom + (Kolom - 1)] x Lebar Elemen

Alamat Awal dalam hal ini adalah alamat pertama dalam array A, yaitu A[1,1]. Jumlah Kolom adalah jumlah kolom dalam array A, yaitu 17. Lebar Elemen adalah 5.

Alamat RMO = A[1,1] + [(7 - 1) x 17 + (8 - 1)] x 5

= A[1,1] + [6 x 17 + 7] x 5

= A[1,1] + (102 + 7) x 5

= A[1,1] + 109 x 5

Sedangkan untuk CMO (Column-Major Order), rumusnya sedikit berbeda:

Alamat CMO = Alamat Awal + [(Kolom - 1) x Jumlah Baris + (Baris - 1)] x Lebar Elemen

Alamat CMO = A[1,1] + [(8 - 1) x 23 + (7 - 1)] x 5

= A[1,1] + [7 x 23 + 6] x 5

= A[1,1] + (161 + 6) x 5

= A[1,1] + 167 x 5

Perlu diketahui bahwa Alamat Awal dalam hal ini harus diketahui atau diberikan dalam konteks yang lebih spesifik, karena tergantung pada bagaimana array A diinisialisasi dan disimpan dalam memori.

25. In 2019, Company A has suffered from a 10% decrease in its revenue compared to the previous year. Two years ago, the revenue was only Rp 57,000,000, which is half of the revenue from 2018. The CMO is targeting Rp 130,000,000 sales in 2020. How much increase in % does Company A aim for 2020? A 18,67% B 25,77% C 19,6% D 26,71% E 22,3%


The correct answer is option D (26.71%)

Penggunaan tanda baca seperti: koma, titik, tanda sambung, dan sebagainya dalam bahasa Inggris berbeda dengan bahasa Indonesia.

Langkah 1

Menerjemahkan soal kedalam bahasa Indonesia dari bahasa Inggris

Adapun hasil terjemahan nya ialah:

Di tahun 2019, Perusahaan A menderita penurunan 10% dari pendapatannya dibandingkan tahun sebelumnya

Dua tahun sebelumnya pendapatannya hanya Rp.57.000.000,- dimana itu adalah setengah dari pendapatan  tahun 2018.

Kepala Pemasaran ditarget penjualannya Rp.130.000.000,-  pada tahun 2020. Berapa banyak peningkatan dalam persentase yang perusahaan A target pada 2020 ?

A 18,67%

B 25,77%

C 19,6%

D 26,71%

E 22,3%

Langkah 2

Mengerjakan soal yang sudah diterjemahkan pada langkah 1

Diketahui: Pendapatan 2019 = pendapatan 2018 – 10%

                  Pendapatan 2017 = (1/2) pendapatan 2018

                                                = Rp.57.000.000,-

Ditanya : Kenaikan target pendapatan dalam persentase

Dijawab :

Langkah 2.1

Mencari nilai pendapatan ditahun 2017, 2018, dan 2019

Pendapatan 2017 = Rp.57.000.000,-

Pendapatan 2018 = 2 x pendapatan 2017

                                = 2 x Rp.57.000.000,-

                                 = Rp.114.000.000,-

Langkah 2.2

Mencari selisih peningkatan antara target 2020 dengan pendapatan 2019

Pendapatan 2019 = pendapatan 2018 – 10%

                               = Rp. 114.000.000 – Rp.114.000.000,- x 10%

                               = Rp.114.000.000,- – Rp. 11.400.000,-

                               = Rp.102.600.000,-

Langkah 2.3

Mencari selisih peningkatan antara target dan pendapatan 2019 yang sudah didapatkan pada langkah 2

Selisih peningkatan   = target 2020 – pendapatan  2019

                                   = Rp.130.000.000,- - Rp.102.600.000,-

                                   = Rp.27.400.000,-

Langkah 2.4

Mencari nilai persentase peningkatan target

Peningkatan target (%) = (selisih peningkatan/pendapatan 2019) x 100%

                                      = (Rp.27.400.000/Rp.102.600.000) x 100%

                                      =  26,705%
