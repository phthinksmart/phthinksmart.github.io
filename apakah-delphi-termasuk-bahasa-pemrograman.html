1. Bahasa pemrograman tingkat tinggi yang di gunakan delphi adalah


Bahasa pemrograman tingkat tinggi yang di gunakan delphi adalah Object Pascal.

Pembahasan

Object Pascal merupakan bahasa pemrograman yang diturunkan dari bahasa pemrograman Pascal. Object Pascal pertama kali dikenalkan ketika Delphi menggunakan Object Pascal sebagai bahasa pemrogramman tingkat tinggi utama dari IDE tersebut. Dengan adanya Object Pascal, Delphi sudah mendukung pemrograman berorientasi obyek dengan berdasarkan class dan interface. IDE atau Integrated Development Environment adalah aplikasi yang digunakan untuk mengembangkan suatu perangkat lunak.

4. Konsep dasar yang digunakan pada pemrograman delphi adalah


Konsep dasar pada pemrograman delphi adalah

Project, form, dan unit

Penjelasan:

KONSEP DASAR

Didalam delphi ada beberapa pengertian yang memerlukan penjelasan untuk memahami bagaimana dan apa saja komponen yang ada dalam delphi, adapun diantaranya adalah:

PROJECT

Project delphi adalah sekumpulan file yang akan membangun aplikasi atau library yang dapat disebarkan kepada pemrogram lain. Pada saat memulai delphi, delphi akan membuat sebuah project baru. Setelah itu terserah pengguna, apakah ingin memodifikasi projek terasebut, membuat projek baru lagi atau mungkin membuka projek yang sudah ada.

Project aplikasi terdiri dari :

1.File project : file ini tersimpan dengan ekstensi ".Dpr", hanya ada satu file projek untuk satu projek.

2.File form : file form tersimpan dengan ekstensi ".dfm", adalah file biner yang berisi gambaran grafis dari form. Setiap file .dfm selalu pararel dengan satu unit file yang berekstensi .pas.

3.Unit (source) : file unit tersimpan dengan ekstensi ".pas", dimana setiap unit berupa file source code dalam bahasa pascal. Tidak setiap file unit memiliki atau terhubung file .dfm

File projek menggabung semua file diatas yang digunakan oleh delphi untuk melakukan kompilasi dan membuat file target (.exe atau .dll).

FORM

Form merupakan tempat kita merancang program. Jika seorang pemrogram diibaratkan sebagai seorang pelukis maka form inilah yang menjadi kanvasnya. Form yang dirancang dalam delphi bisa digunakan kembali oleh projek delphi yang lain. Form dapat juga disimpan dalam bentuk dll, sehingga dapat digunakan oleh aplikasi yang lain seperti c++, paradox, atau dbase. Delphi menawarkan kemudahan pembuatan form melalui model template.

UNIT

Unit merupakan modul yang terpisah yang dapat dikompilasi tersendiri, yang berisi deklarasi bagian publik (antarmuka) dan bagian privat (implementasi). Setiap form dalam delphi mempunyai satu unit yang bersesuaian. Source code unit tersimpan dengan ekstensi pas, sedangkan yang sudah terkompilasi tersimpan dengan ekstensi .dcu. proses link akan menggabung file .dcu menjadi satu file exe atau dll.

HTML adalah singkatan dari Hypertext Markup Language. Pengertian HTML sering kali dikaitkan dengan bahasa pemrograman. Akan tetapi, sejatinya, HTML bukan sebuah bahasa pemrograman (programming languange), melainkan sebuah markup languange.

17. apakah pengulanagan dalam bahasa pemrogaman delphi sama dengan pengulangan di bahasa pemrograman lain?dan jelaskan alasannya


Jawaban:

Pengertian

Looping/pengulangan adalah sebuah perintah yang berfungsi untuk mengulang suatu statement, banyaknya statement yang diulang tergantung pada nilai yang telah dimasukan dan statement akan berhenti mengulang apabila jumlah statemen yang ditampilkan tidak lebih dari sama dengan nilai yang dimasukan.

Looping dalam bahasa pemrograman delphi dibagi kedalam 3 mecam, yang pertama For To do, kedua while do, dan yang ketiga repeat until.

1. For To Do

Pengertian Looping/Pengulangan dengan menggunakan for to do adalah sebuah pengulangan yang mana suatu nilai akan berhenti apabila nilai tersebut sudah bernilai true atau sudah memenuhi persyaratan yang sudah ditentukan.

Contoh

Var i:integer;

Begin

For i:=1 to 5 do

Begin

Memo1.Text:=IntToStr(i);

End;

End.

Dalam syntax diatas bisa disimpulkan bahwa pengulangan akan terus berlanjut sampai angka 5 ditampilkan pada listbox dimana angka 5 adalah nilai yang bernilai true atau persyaratan yang telah ditentukan. Jadi, proses looping/pengulangan akan berhenti apabila angka 5 sudah muncul pada memo.

2. While Do

Pengertian Looping/pengulangan dengan menggunakan while do sebenarnya hampir sama dengan for to do hanya saja pada pengulangan while do nilai awalnya harus dicantumkan terlebih dahulu. Selain itu, dalam pengulangan while do bisa menggunakan operasi aritmatika seperti lebih besar(>), lebih kecil(<), dll.

Contoh

Var i:integer;

Begin

I:=0;

While i>5 do

Begin

I:=i+1;

Memo1.Text:=IntToStr(i);

End;

End.  

3. Repeat Until  

Pengertian looping/pengulangan dengan menggunakan repeat until adalah sebuah pengulangan yang mana suatu statemen/nilai yang ada dalam repeat tersebut akan berhenti dieksekusi jika nilai yang ada pada until sudah bernilai false.

Contoh

var

i : integer;

begin

i := 1;

repeat

i := i + 1;

Memo1.Text := inttostr(i);

until i = 5;

end;
