1. harga kaos adalah 3/5 dari harga celana. Harga kaos dan celana tersebut adalah Rp160.000. Berapa harga setiap kaos dan celana?


kaos = x
celana = y
harga kaos (x) = 3/5 y
harga x+y = 160,000

x+y = 160,000
3/5y+y = 160,000
3/5y+5/5y = 160,000
8/5y = 160,000
y = 160,000 : 8/5
y = 160,000 x 5/8
y = 100,000

x+y = 160,000
x + 100,000 = 160,000
x = 160,000-100,000
x = 60,000

jadi harga kaos adalah 60,000 dan harga celana adalah 100,000

18. Harga label sebuah kaos oblong adalah Rp42.000. Jika kaos tersebut diskon 10%, maka harga kaos oblong itu menjadi …. *​


Penyelesaian :

Harga label = Rp42.000

Diskon = 10%

Harga setelah diskon :

= Rp42.000 × (100% - 10%)

= Rp42.000 × 90%

= Rp37.800

Jadi jika mendapatkan diskon 10% maka harga kaos tersebut menjadi Rp37.800

Jawab: Rp. 37.800

Penjelasan dengan langkah-langkah:

Diskon : 10/100×42.000 = 4.200

Harga setelah di diskon : 42.000-4.200 = 37.800

Semga bermanfaat terimakasih.
