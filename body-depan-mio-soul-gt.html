5. Sintak yang benar untuk menambahkan warna background atau latar belakang dibawah ini adalah... a. <body =’background-:color:red’>b. <background>red</background>c. <body style=’background-color:red’>d. <body style=background-red>​


Sintaks yang benar untuk menambahkan warna background atau latar belakang adalah:

<body style=’background-color:red’>

Jadi pilihan yang tepat adalah C.

Pembahasan

Di dalam HTML sintaks yang benar untuk melakukan perubahan warna background adalah:

<body style=’background-color:red’>

Sintaks tersebut adalah sintaks yang sesuai dengan panduan HTML yaitu:

style="background-color:" ke elemen <body>.

Pilihan lainnya salah karena:

a. <body =’background-:color:red’> tidak sesuai dengan sintaks HTML.

b. <background>red</background> tidak sesuai dengan sintaks HTML.

d. <body style=background-red>​ tidak sesuai dengan sintaks HTML.

Sintaks HTML atau sintaks dalam bahasa pemrograman apa pun adalah bentuk perintah baku yang harus diikuti agar baris sintaks tersebut dapat dikenali dan dieksekusi oleh komputer. Bila salah walau satu karakter saja maka program tidak bisa mendeteksi baris perintah tersebut dan membuat baris kode yang kita buat tidak bisa dijalankan.

Baris kode di atas adalah salah satu contoh baris kode dalam HTML-CSS yang disebut inline CSS. Kita dapat menerapkan pedoman style ke komponen HTML tertentu dengan menggunakan inline CSS. Alih-alih menggunakan CSS eksternal, menyisipkan CSS melibatkan penyertaan CSS dalam file HTML. Meskipun penggunaannya dibatasi, inline CSS berguna untuk menentukan karakteristik unik karena memungkinkan penerapan gaya khusus ke satu elemen HTML.

a × a × a × ....... a × a = aⁿ

a disebut bilangan pokok (basis)

1). a^m × aⁿ = a^(m + n)

2). a^m ÷ a^n = a^(m - n)

3). 1 ÷ aⁿ = a^(-n)

4). (a^m)ⁿ = am × n

5). a⁰ = a ÷ a

6). aⁿ × bⁿ = ab

7). aⁿ ÷ bⁿ = (a ÷ b)ⁿ

14. 6. Untuk membuat judul yang kemudian akan tampil pada bagian atas / judul dari sebuah web browser, maka kita dapat menggunakan tag: ....... *A. <head></head>B. <html></html>C. <body></body>D. <title></title>​


B. <html></html>

HTML menyediakan tag khusus untuk membuat judul atau di dalam HTML lebih di kenal dengan istilah: heading. Heading dirancang terpisah dari paragraf. Tag Heading biasanya digunakan untuk judul dari paragraf, atau bagian dari text yang merupakan judul.

Tag heading di dalam HTML terdiri dari 6 tingkatan, yaitu <h1>, <h2>, <h3>, <h4>, <h5>, dan <h6>. Tag heading secara default akan ditampilkan oleh web browser dengan huruf tebal (bold). Tampilan dari tag header juga dibuat bertingkat. Tag header <h1> memiliki ukuran huruf paling besar, sedangkan <h6> terkecil.

Biasanya dalam sebuah halaman akan ada hanya 1 tag <h1>, dan beberapa tag <h2> dan <h3> dengan beberapa tag <p>. Berikut adalah contoh struktur artikel HTML sederhana dengan menggunakan tag <p> dan beberapa tag heading.

Biasanya semakin tinggi tingkat heading, semakin tinggi pula penekanan search engine. Tag <h1> dianggap lebih bernilai dari pada <h2>. Tag <h1> umumnya digunakan untuk judul sebuah konten / artikel, sehingga dianggap dapat mewakili keseluruhan artikel.

29. 5. Informasi mengenai sebuah dokumen HTML, seperti judul dan pengarang / pembuat dokumen HTML tersebut terdapat di antara tag: .......... *A. <head></head>B. <html></html>C. <body></body>D. <title></title>​


B. <html></html>

HTML menyediakan tag khusus untuk membuat judul atau di dalam HTML lebih di kenal dengan istilah: heading. Heading dirancang terpisah dari paragraf. Tag Heading biasanya digunakan untuk judul dari paragraf, atau bagian dari text yang merupakan judul.

Tag heading di dalam HTML terdiri dari 6 tingkatan, yaitu <h1>, <h2>, <h3>, <h4>, <h5>, dan <h6>. Tag heading secara default akan ditampilkan oleh web browser dengan huruf tebal (bold). Tampilan dari tag header juga dibuat bertingkat. Tag header <h1> memiliki ukuran huruf paling besar, sedangkan <h6> terkecil.

Biasanya dalam sebuah halaman akan ada hanya 1 tag <h1>, dan beberapa tag <h2> dan <h3> dengan beberapa tag <p>. Berikut adalah contoh struktur artikel HTML sederhana dengan menggunakan tag <p> dan beberapa tag heading.

Biasanya semakin tinggi tingkat heading, semakin tinggi pula penekanan search engine. Tag <h1> dianggap lebih bernilai dari pada <h2>. Tag <h1> umumnya digunakan untuk judul sebuah konten / artikel, sehingga dianggap dapat mewakili keseluruhan artikel.
