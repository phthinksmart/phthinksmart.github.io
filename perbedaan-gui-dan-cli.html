1. perbedaan CLI dan GUI​


Jawaban:

GUI adalah jenis antarmuka pengguna yang memungkinkan pengguna berinteraksi dengan perangkat elektronik melalui ikon grafis dan indikator visual. CLI adalah antarmuka bagi pengguna untuk mengeluarkan perintah dalam bentuk baris teks atau baris perintah berturut-turut untuk melakukan tugas komputer.


2. perbedaan GUI sama CLI?


GUI adalah tipe antarmuka yang digunakan oleh pengguna untuk berinteraksi dengan sistem operasi melalui gambar-gambar grafik, ikon, menu, dan menggunakan perangkat penunjuk (pointing device) seperti mouse atau track ball. Elemen-elemen utama dari GUI bisa diringkas dalam konsep WIMP (window, icon, menu, pointing device).

CLI adalah tipe antarmuka dimana pengguna berinteraksi dengan sistem operasi melalui text terminal. Pengguna menjalankan perintah dan program di sistem operasi tersebut dengan cara mengetikkan barisbaris tertentu.

18. jelaskan kekurangan dan kelebihan dari CLI, GUI,dan VUI!​


Kelebihan CLI:

Membutuhkan spesifikasi hardware yang relatif lebih rendah.Dapat melakukan berbagai perintah spesifik.Akses yang cepat.Dapat berpindah dari satu layar ke layar lain tanpa terjadi kehilangan informasi.

Kekurangan CLI:

Tampilannya kurang menarik.Harus menghafalkan berbagai perintah dan kode.Resiko kehilangan file yang cenderung lebih besar.

Kelebihan GUI:

Tampilannya menarik dan mudah dimengerti.Resolusi gambar yang tinggi.Tidak perlu menghafalkan perintah dan kode.

Kekurangan GUI:

Membutuhkan spesifikasi hardware yang lebih tinggi.Performa lebih berat.

Kelebihan VUI:

Mudah digunakan.Faktor keamanan biasanya jauh lebih tinggi.Efisien dan cepat.

Kelemahan VUI:

Implementasi terbatas.

Pembahasan

GUI merupakan tipe antarmuka/interface yang digunakan untuk berinteraksi antara pengguna dengan sistem operasi melalui gambar dan grafik. CLI merupakan tipe interface yang digunakan untuk berinteraksi namun menggunakan menu baris atau teks. VLI merupakan tipe antarmuka yang menggunakan teknologi pengenalan suara/ucapan (voice).

20. Apa perbdaan GUI dan CLI


Dua tipe interface sistem operasi : 1. Command Line Interface (CLI) – berbasis teks, yang jika melakukan perintah harus dengan mengetikan perintah. 2.Graphical User Interface (GUI) – berbasis grafis, dgn memiliki tampilan berupa menu dan icon-2. Dua Jenis lisensi sistem operasi : 1. Propietary (kepemilikan), yaitu untuk menggunakannya dengan meminta ijin kepada sang pemiliki, yang biasanya dengan membayar sejumlah cost dan biasanya propietary itu closed source (tertutup). 2. Free Software (open source), yaitu suatu lisensi Free (bebas) dan terbuka yang memungkinkan pengguna untuk memiliki, mengcopy, menggandakan, mendistribusikan tanpa ada batasan.

26. Berikan contoh VUI, GUI dan CLI​


Contoh VUI (Voice User Interface) adalah perangkat pintar yang dapat dioperasikan dengan suara, seperti Amazon Echo atau Google Home. Pengguna dapat mengoperasikan perangkat tersebut dengan memberikan perintah suara untuk mengakses layanan atau aplikasi yang tersedia.

Contoh GUI (Graphical User Interface) adalah antarmuka pengguna pada perangkat komputer, seperti desktop, laptop, atau smartphone. Pengguna dapat mengoperasikan perangkat tersebut dengan menggunakan input seperti mouse, keyboard, atau layar sentuh.

Contoh CLI (Command Line Interface) adalah antarmuka pengguna yang menggunakan perintah teks untuk mengakses sistem atau aplikasi, seperti Command Prompt pada sistem operasi Windows atau Terminal pada sistem operasi macOS atau Linux. Pengguna dapat mengetik perintah teks untuk mengakses layanan atau aplikasi yang tersedia.
