20. tuliskan kode program pada aplikasi visual basic agar tampil tulisan "hello word"​


Jawaban:Biasanya untuk mengawali belajar programing saya memulainya dengan membuat program dengan output Hello world seperti pembahasan program sebelumnya dalam java dan c kita di awal pembalajaran membuat script dengan output hello world . terdapat menu bar, Tool bar, Form (lingkarang hijau), Tool box (lingkarang Merah), Project Explorer (Lingkaran hitam), Property (lingkarang biru), dan Form layout (seperti gambar monitor di properti bawah) . kemudian pilih Exe standar , setelah itu anda akan di suguhkan tampilan kemudian kita tinggal mendesain tampilan untuk setiap perbaikan objek kita lakukan di properti . setelah itu akan tampil kode form proyek. Text1 (dalam program) diambil dari tex box yang kita buat pertama sehingga namanya text1 (bisa anda rubah jika mau), karena di tex box itu akan keluar kata Hello world , sehingga dalam program harus di tuliskan objek tersebut di kenali dimana letak teks akan tampil bila tombol tombol klik di klik.

* Aligenment = untuk perataan kalimat. dalam skrip ini saya menggunakan pusat sehingga teks yang tampil pada contoh kotak tex berada di tengah.

* perintah Text = untuk menampilkan kata / hasil (output) pada objek dalam contoh ini yaitu pada (tex box)

* perintah ForeColor = untuk memberikan warna pada Objek dalam contoh ini yaitu kata ”Hello world” dengan ketentuan RGB atau Red (merah), Green (hijau), Blue (biru). pada contoh script diatas saya memberikan angka 255 pada R / Red sehingga nanti keluarannya (Hello world) akan berwarna merah.

setelah selesai coba anda menjalankan program anda dengan keinginan tombol f5 , nanti hasilnya seperti ini. kemudian klik tombol button klik maka akan tampil kata ” Hello world ”. Maaf kalau salah:) tolong jadikan jawaban tercerdas

Kode Program:

Imports System

Module ModuleKu

   Sub Main()

       Console.WriteLine("Hello World!")

   End Sub  

End Module

Penjelasan:

Ayo Kita Bahas Satu Persatu Artinya Ya.

1. Imports System

Imports System Berarti Kita Mengimpor Semua Kelas, Fungsi Dan Konstanta / Variabel Didalam Namespace system Yang Sudah Disediakan Oleh NET. Framework.

2. Module ModuleKu

Module ModuleKu Berarti Kita Mendefinisikan Module Yang Bernama "ModuleKu". Module Ini Berisi Semua Variabel Dan Fungsi Yang Kita Butuhkan.

3. Sub Main()

Ini Berarti Kita Mendefinisikan Fungsi Main() Didalam Module ModuleKu. Sub Main() Artinya Fungsi Utama. Program Akan Berjalan Mulai Dari Sini.

4. Console.WriteLine("Hello World!")

Console.WriteLine("Hello World!") Berarti Kita Akan Menulis Output "Hello World!" Ke Konsol.

5. End Sub

End Sub Menandakan Akhir Dari Fungsi Yang Kita Buat, Dalam Hal Ini Adalah Fungsi Main().

6. End Module

End Module Menandakan Akhir Dari Module Yang Kita Buat, Dalam Hal Ini Adalah Module ModuleKu.
