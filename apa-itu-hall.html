Ada beberapa alasan penting mengapa formulir – formulir digunakan : 1. untuk keseragaman atau pembakuan kerja ( work standarization ) 2. untuk mempermudah dalam penggolongan ( classification ) data. 3. untuk mempermudah pemberkasan ( filling ) baik sistem maupun tempatnya.

Bagian-bagian teks formulir terdiri dari: Kepala (heading), bagian ini berisi tentang informasi dasar seperti nama, alamat, hingga nomor telepon. Instruksi, bagian ini berisi informasi tentang cara pengisian formulir. Badan, bagian ini berisi data keterangan yang harus diisi secara lengkap.

72 =2(3*breadth + breadth)

27. There were 445 pupils in Hall A and 279 pupils in Hall B. A teacher transferred some pupils from Hall A to Hall B so that both halls had an equal number of pupils. How many pupils were there in each hall after the transfer? step by step​


Jawaban:

83

Penjelasan dengan langkah-langkah:

for this question, we first need to know how many pupils should be in one hall so that halls A and B have an equal number of pupils. this could be done by finding the mean (average) of the 2. here's how it goes:

→ (445+279)/2

→ 724/2

→ 362

now that we know the mean number of pupils in the halls, we subtract the number of pupils in hall A by the average. here's how they go:

445-362=83

so the answer is 83. hope that helps :D
