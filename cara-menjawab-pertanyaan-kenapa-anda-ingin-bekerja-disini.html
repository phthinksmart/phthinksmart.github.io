(f•g)(x) = f(x³) = x³-2

((f•g)•h)(x) = 64x³ - 2

y = 64x³ - 2

y + 2 = 64x³

<=> 64x³ = y + 2

=> x³ = (y + 2)/64

=> x = ³√( (y + 2)/64 )

Jawaban:

Nilai x adalah 2

B. 2

Penjelasan dengan langkah-langkah:

fungsi komposisi

================

f(x) = x – 3, dan g(x) = x² + 5

mencari nilai x dari :

(f ⁰ g)(x) = (g ⁰ f)(x)

================

cari fungsi (fog)(x)

= (fog)(x)

= f(g(x))

= (x² + 5) – 3

= x² + 2

================

cari fungsi (gof)(x)

= (gof)(x)

= g(f(x))

= (x – 3)² + 5

= x² – 6x + 9 + 5

= x² – 6x + 14

===============

sehingga,

(fog)(x) = (gof)(x)

x² + 2 = x² – 6x + 14

x² + 2 - x² + 6x - 14 = 0

6x - 12 = 0

6x = 12

x = 12/6

x = 2

16. Jawablah pertanyaan diatas dengan cara kerja yang benar!​


Jawaban:

nilai n adalah -15

A. –15

Penjelasan dengan langkah-langkah:

fungsi komposisi

================

f(x) = 3x - 10, dan g(x) = 4x + n

maka,

(fog)(x) = (gof)(x)

f(g(x)) = g(f(x))

3(4x + n) - 10 = 4(3x - 10) + n

12x + 3n - 10 = 12x - 40 + n

12x - 12x + 3n - n - 10 + 40 = 0

2n + 30 = 0

2n = -30

n = -30/2

n = -15
