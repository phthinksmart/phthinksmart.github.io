1. Apa Perbedaan PNG dan JPEG?​


Jawaban:

JPEG

JPEG merupakan singkatan dari Joint Photographics Expert Group. Format yang memiliki ekstensi file *.JPEG atau *.JPG. ini mulai dikembangkan pada tahun 1986. Keunggulan JPEG adalah memiliki tingkat kompresi tinggi namun tetap menggunakan warna _true colour_ (24 bit). Sehingga walaupun ukuran file-nya kecil, warna yang ditampilkan tetap akurat, sehingga kualitas gambar dapat terjaga.

Format ini dapat digunakan di hampir semua sistem operasi seperti Windows, Mac, Linux, dan sebagainya. JPEG menjadi format standar untuk kamera digital atau kamera hp. Namun JPEG tidak mendukung transparansi. Sehingga jika gambar kamu tidak full, akan terdapat background berbentuk kotak warna putih. Misalnya kalau Jaka masukkan ke dalam tulisan di Microsoft Word seperti ini.

Jpg

Format JPEG paling tepat digunakan untuk foto dan gambar yang memiliki kompleksitas warna tinggi, atau terdapat gradasi dengan perbedaan warna yang tipis, serta nuansa terang dan gelap.

PNG

PNG adalah kependekan dari Portable Network Graphic dan memiliki ekstensi file *.PNG. Pengembangan format PNG dimulai pada tahun 1995. Keunggulan PNG adalah mampu menyimpan gambar yang memiliki transparansi, misalnya _background_ transparan atau bagian gambar yang semi transparan. Kamu bisa memasukkan gambar PNG transparan ke dalam program lain, misalnya Microsoft Word tanpa ada _background_ putih yang mengganggu seperti format JPEG.

Png

Format PNG juga mampu menyimpan gambar yang semi transparan seperti pada gambar berikut. Coba perhatikan tulisan "Jalan" yang seolah tembus pandang.

Png2

Format PNG paling tepat digunakan untuk foto dan gambar yang memiliki transparansi atau efek fading (luntur), misalnya logo. PNG juga sangat baik untuk menyimpan gambar yang akan digunakan sebagai bahan-bahan membuat desain selanjutnya. Namun ukuran file gambar berformat PNG cenderung lebih besar.

PNG : untuk menangkap gambar seutuhnya dan namun memiliki ukuran yang lebih besar.

JPEG merupakan singkatan pendek : Joint Photographics Expert Group .

JPEG : Cenderung membuat kompresi yang ukuran lebih kecil penurunan kualitas gambar .

11. 5.Sebutkan perbedaan format gambar GIF, PNG, JPEG, dan SVG!Jawab:​


Jawaban:

perbedaan format pada image karena penggunaannya berbeda-beda

Penjelasan:

GIF : format gambar yang bisa dianimasikan (bergerak). Mode gambar RGB sehingga penggunaannya untuk ditampilkan dilayar saja (bukan format cetak). Beberapa jenis tidak bergerak (statis), bisa memiliki backround transparan atau tidak. Berlisensi untuk penggunaannya

PNG : hampir sama seperti GIF namun tidak bisa dianimasikan (bergerak) dan tidak berlisensi. Mode gambar RGB untuk penggunaan di layar monitor (bukan format cetak). Paling sering digunakan karena memiliki background yang transparant, beberapa jenis tidak memiliki background transparant.

JPEG/JPG : Format standart image yang paling banyak dipakai. Mode gambar RGB untuk penggunaan di layar monitor, dan Mode gambar CMYK untuk keperluan cetak.

SVG : format gambar vector yaitu berupa pola garis (outline) dan isi warna (fill). Format gambar untuk keperluan editing ulang. Memiliki background transparant

NB: semua format diatas bertipe BITMAP kecuali SVG bertipe VECTOR

21. Apakah perbedaan JPEG & RAW?


Gambar RAW, atau dikenal sebagai “negatif digital” ini merupakan file hampir seluruhnya masih diproses langsung dari sensor kamera. File RAW benar-benar “mentah”, seperti bahan makanan yang perlu dipersiapkan dan dimasak sebelum dapat dipakai. RAW ialah format esklusife yang terkait dengan produsen kamera dan sensor, karena itu tidak didukung oleh semua produk software. file RAW menjaga jumlah sebagian besar informasi tentang gambar yang umumnya mengandung lebih banyak warna dan jangkauannya yang lebih dinamis dari pada gambar JPEG. Untuk memungkinkan kamera digital dapat menampilkan gambar yang direkam pada LCD, file RAW biasanya terdiri dari dua bagian yakni data RAW aktual dari sensor kamera serta preview JPEG yang diproses. Banyak aplikasi yang dipakai untuk melihat gambar, termasuk LCD kamera, kemudian gunakan pratinjau JPEG ini tertanam ke dalam file RAW untuk menampilkan gambar.

JPEG ini format gambar yang paling populer untuk foto-foto pada masa sekarang ini, mampu menampilkan jutaan warna dalam file dengan sangat padat. Metode kompresi yang dipakai dalam format JPEG ialah “lossy”, yang berarti bahwa informasi tertentu akan dihapus dari gambar. Berbagai tingkat kompresi (dalam persentase) bisa diterapkan pada gambar JPEG, yang berdampak pada kualitas serta ukuran gambar. Semakin banyak detail gambar yang di pertahankan, maka ukurannya semakin besar.

Pembahasan

Perbedaan dari RAW dan JPEG

RAW :

Gambar dari format RAW mengandung jumlah sebagian besar informasi dengan 4.096 nuansa atau Merah, Hijau dan Biru ( setara dengan 68 miliar warna!) serta lebih tinggi. Pada kamera Nikon D700 bisa merekam 14-bit file RAW, yang setara dengan kira-kira 4,3 triliun warna mungkin.  File dari format RAW berisi rentang informasi yang paling dinamis (rasio antara maksimum dan minimum intensitas cahaya yang terukur cahaya putih dan hitam) dan kemudian dapat dipakai untuk memulihkan atau memperbaiki gambar yang kurang terang atau gambar underexposed atau pada bagian dari suatu gambar yang perlu di perbaiki. Saat gambar RAW yang dihasilkan, semua pengaturan kamera, termasuk kamera khusus dan informasi produsen-spesifik, hanya ditambahkan ke dalam file. Yang berarti bahwa gambar itu sendiri tetap tidak dimodifikas, pengaturan ini hanya disediakan sebagai referensi dan pengguna kamera dapat membuat perubahan nanti dalam aplikasi pengolahan pasca foto seperti Lightroom dan Photoshop.  File dari format RAW memanfaatkan kompresi lossless, yang berarti mereka tidak kehilangan data gambar saat di kompresi. Tidak ada gambar penajaman dilakukan pada file RAW, yang berarti bahwa pengguna dapat menggunakan algoritma penajaman lebih baik serta lebih kompleks untuk foto pengguna di PC yang digunakan. File RAW dapat dipakai sebagai bukti kepemilikan atas foto dan keaslian, gambar tampa di modifiaksi.

JPEG :

Gambar JPEG ini sepenuhnya diproses dalam kamera dan semua pengaturan seperti White Balance, Color Saturation, Tone Curve, ISO serta kecepatan Shutter sudah diterapkan pada gambar. Hanya perlu melakukan ekstrak gambar dari kartu memori dan siap untuk digunakan. Ukuran Gambar JPEG jauh lebih kecil dari gambar RAW karena mengkonsumsi kurang banyak penyimpanan serta tidak membutuhkan pemrosesan yang tinggi. Karena ukuran yang lebih kecil, kamera dapat menulis serta membaca file JPEG jauh lebih cepat, meningkatkan jumlah gambar yang bisa ditampung dalam buffer kamera sementara, dapat berpotensi menjepret pada frame yang lebih tinggi per detik untuk waktu yang cukup lama. Kebanyakan perangkat modern maupun paket perangkat lunak mendukung gambar JPEG, membuat format yang sangat kompatibel. Kamera digital menyediakan kompresi serta ukuran yang berbeda pilihan untuk menyimpan gambar JPEG, memberikan fleksibilitas serta pilihan atas kualitas gambar dan ukurannya. Ukuran yang lebih kecil berarti lebih cepat dan backup serta lebih efisien.Pelajari lebih lanjut

25. pengertian format gambar PSD,EPS,JPEG,GIF,PNG,TIFF,BMP


JPEG (Joint Photographic Expert Group) adalah standar kompresi file yang dikembangkan oleh Group Joint Photographic Experts; menggunakan kombinasi DCT dan pengkodean Huffman untuk mengkompresikan suatu file citra. JPEG adalah suatu algoritma kompresi yang bersifat lossy, (yang berarti kualitas citranya agak kurang bagus).

Graphics Interchange Format(GIF) adalah sebuah format yang sering digunakan dalam dunia web maupun dalam dunia citra digital. Format ini sering digunakan karena ukurannya yang relati kecil dan juga banyaknya software editor gambar yang telah mendukung citra ini. GIF berukuran kecil karena membatasi jumlah warnanya sebanyak 256 warna sehingga dapat menghemat ukuran berkas. 

BMP merupakan file gambar yang tidak terkompresi. Dan karena itu ukuran file BMP biasanya besar. File berukuran besar tidak tepat untuk transfer di internet karena akan memenuhi bandwidth dan menjadi lambat.

The TIFF (Tagged Image File Format) adalah format gambar yang fleksibel biasanya menyimpan 16-bit per warna - merah, hijau dan biru untuk total 48-bit - atau 8-bit per warna - merah, hijau dan biru untuk total 24-bit - dan menggunakan nama file atau perpanjangan TIFF TIF. TIFF yang kedua adalah fleksibilitas fitur, dan kutukan, dengan tidak ada satu pembaca semua mampu menangani berbagai jenis file TIFF. TIFF dapat lossy atau lossless. 

(Portable Network Graphics) adalah salah satu format penyimpanan citra yang menggunakan metode pemadatan yang tidak menghilangkan bagian dari citra tersebut (Inggris lossless compression).

26. Sebutkan perbedaan format gambar GIF, PNG, JPEG, dan SVG​


Jawaban:

perbedaan format pada image karena penggunaannya berbeda-beda

Penjelasan:

GIF : format gambar yang bisa dianimasikan (bergerak). Mode gambar RGB sehingga penggunaannya untuk ditampilkan dilayar saja (bukan format cetak). Beberapa jenis tidak bergerak (statis), bisa memiliki backround transparan atau tidak. Berlisensi untuk penggunaannya

PNG : hampir sama seperti GIF namun tidak bisa dianimasikan (bergerak) dan tidak berlisensi. Mode gambar RGB untuk penggunaan di layar monitor (bukan format cetak). Paling sering digunakan karena memiliki background yang transparant, beberapa jenis tidak memiliki background transparant.

JPEG/JPG : Format standart image yang paling banyak dipakai. Mode gambar RGB untuk penggunaan di layar monitor, dan Mode gambar CMYK untuk keperluan cetak.

SVG : format gambar vector yaitu berupa pola garis (outline) dan isi warna (fill). Format gambar untuk keperluan editing ulang. Memiliki background transparant

NB: semua format diatas bertipe BITMAP kecuali SVG bertipe VECTOR

Jawaban:

- format gambar GIF (Graphical Interchange Format)

Merupakan format paling sesuai untuk menyimpan gambar sederhana (seperti garis & bentuk dasar) yang tak mengandung banyak warna/ gradasi.

- format gambar PNG (Portable Network Grahic)

Merupakan format yang mampu menyimpan gambar dengan transparansi (misalnya background transparan) atau bagian gambar yang semi transparan.

- format JPEG (Joint Photographic Expert Group)

Merupakan format paling tepat dipakai untuk file foto dan gambar yang memiliki kompleksitas warna tinggi.

- format SVG (Scalable Vector Graphic)

Merupakan format untuk menyimpan file gambar berbasis vektor.

Pembahasan :

PNG

PNG (diucapkan "ping" atau "pi en ji") mulai dikembangkan pada pertengahan 90-an sebagai alternatif format GIF dan JPEG tetapi masih memiliki keunggulan dari kedua format tersebut. PNG merupakan kependekan dari Portable Network Graphic dan memiliki ekstensi .png.

PNG memiliki 2 bentuk yaitu PNG-8 dan PNG-24. PNG-8 memiliki kesamaan sifat seperti GIF, yaitu mendukung hanya 256 warna (indexed color) serta mendukung transparansi 1-bit. Namun terkadang format PNG-8 ini memiliki ukuran file yang lebih kecil dibandingkan dengan GIF.

Kemudian PNG-24, format ini memiliki kesamaan dengan JPEG yaitu mendukung pewarnaan hingga 24-bit (true color) dimana memiliki kedalaman warna hingga 16 juta warna. Perbedaan dengan JPEG adalah PNG menggunakan kompresi loosless, artinya gambar yang dihasilkan memiliki kualitas yang luarbiasa namun ukuran file yang lebih besar dibanding JPEG.

Baik PNG-8 ataupun PNG-24 mendukung transparansi dimana tingkat transparansi nya dapat diatur dari mulai gelap (pekat) hingga transparan atau dapat disebut juga alpha channel. Format PNG dapat dipakai di berbagai platform dan browser ataupun aplikasi pengolah gambar, namun untuk beberapa browser jadul akan mendapati masalah dalam menampilkan PNG karena browser tersebut belum mendukung alpha channel.

Format PNG akan sangat cocok digunakan untuk gambar logo pada website yang terkadang dibutuhkan tranparansi/alpha channel. Selain itu PNG juga cocok untuk gambar fotografi berkualitas tinggi dimana ukuran file tidak menjadi pertimbangan.

Dengan menggunakan format gambar yang bervariasi dan sesuai dengan penggunaanya, tentu saja akan berpengaruh pada ukuran file gambar sehingga space pada hosting web pun dapat dihemat selain itu waktu loading pun dapat di efektifkan.
