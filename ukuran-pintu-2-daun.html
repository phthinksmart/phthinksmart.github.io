4. 2. Lebar daun pintu pada gambar di samping = 80 cm. Saat pintu dibuka, jarak ujung pintu dengan posisi pintu 40 cm. Jika sudut yang dibentuk daun pintu dengan pintu adalah sudut Q, tentukan nilai dari sin Q, cos Q, dan tan Q. pintu Sumber: dokumen penerbit​


Dalam soal tersebut setelah dilakukan penyelesaian maka didapat hasil sebagai berikut:

Sehingga nilai sin Q adalah sqrt(2)/2, nilai cos Q adalah 2 sqrt(2)/5, dan nilai tan Q adalah 1/2.

Penjelasan dengan langkah-langkah:

Berikut adalah ilustrasi dari deskripsi soal:

     40 cm     |

    <------->     |

      ___        |   pintu

     |       |       |   __

     |       |       |  |       |

     |       |       |  |       |

     |__ _|      |  |__ _|

        80 cm        |

Untuk menentukan nilai sin Q, cos Q, dan tan Q, kita perlu mengetahui panjang sisi miring segitiga yang terbentuk oleh daun pintu dan jarak ujung pintu saat dibuka. Dalam hal ini, panjang sisi miring segitiga adalah akar dari jumlah kuadrat dari lebar pintu (80 cm) dan jarak ujung pintu saat dibuka (40 cm), yaitu:

c = sqrt(80^2 + 40^2) = sqrt(6400 + 1600) = sqrt(8000) = 20 sqrt(20) cm

Setelah mengetahui panjang sisi miring segitiga, kita dapat menentukan nilai sin Q, cos Q, dan tan Q sebagai berikut:

sin Q = jarak tegak / sisi miring

     = 40 / (20 sqrt(20))

     = sqrt(2)/2      

cos Q = jarak alas / sisi miring

     = 80 / (20 sqrt(20))

     = 2 sqrt(2)/5    

tan Q = jarak tegak / jarak alas

         = 40 / 80

        = 1/2

Sehingga nilai sin Q adalah sqrt(2)/2, nilai cos Q adalah 2 sqrt(2)/5, dan nilai tan Q adalah 1/2.

6. sebuah daun pintu berbentuk pergsegi panjang dengan panjang (3h - 2) cm dan lebar (2h + 15) cm. Berapa luas daun pintu?


Luas Persegi Panjang :
L = p × l
L = (3h - 2)(2h + 15)
L = 6h² + 45h - 4h - 30
L = 6h² + 41h - 30 cm
