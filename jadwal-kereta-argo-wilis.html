diket =

Argo Wilis
- 105 km/jam

Jawaban =
A. * 95 km × 3 = 285 km
* 105 km × 3 = 315
Jarak antara kedua kereta =
285 km + 315 km = 600 km

B. 400 = ( 95 × X ) + ( 105 × X )

[tex] \frac{400}{x} = 95 + 105 \\ \frac{400}{x} = 200 \\ x = \frac{400}{200} [/tex]
X = 2
Jadi lama waku yg diperlukan agar keduanya berjarak 400 km adalah 2 jam

Soal :
Selama 8 jam kereta argo murah dapat menempuh jarak 320 km. dengan kecepatan yang sama, jarak tempuh kereta argo murah selama 14 jam adalah 

Di Ketahui :
* Jarak = 320 Km 
* Waktu = 8 Jam
* K = 14 Jam

Di Tanya :
Jarak Tempuh Kereta Argo Murah selama 14 Jam => .. ?

** Rumus Mencari Jarak Tempuh Kereta Selama 14 Jam.
Yaitu :
{{ J => K × W }}

Pembahasan :

-> Langkah 1 
{{ J     => K × W }}
{{ 320 => K x 8 Jam }}
{{ K     => 320 / 8 Jam }}
{{ K     => 40 Km/Jam }}

-> Langkah 2 
{{ J     => K x W }}
{{ J     => 40Km/Jam x 14Jam }}
{{ J     => 560Km }}

Jadi, Jarak Tempuh Kereta Argo Murah selama 14 Jam => 560 Km
