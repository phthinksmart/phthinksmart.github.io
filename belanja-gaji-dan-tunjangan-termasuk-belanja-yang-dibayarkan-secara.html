10. gaji upah tunjangan honorarium komisi dan bonus termasuk ke dalam jenis penghasilan apa


Gaji, upah, tunjangan, honorarium, komisi, serta bonus termasuk jenis penghasilan yang menjadi objek pajak.  Hal ini tercantum pada UU Harmonisasi Pajak Nomor 7 Tahun 2021, Bab III Pasal 4.

Pembahasan

Dijelaskan dalam UU HPP  Bab III Pasal 4 Ayat 1 bahwa objek pajak adalah penghasilan yaitu setiap tambahan kemampuan ekonomis yang diterima atau diperoleh wajib pajak. Tambahan ini dapat dipakai untuk konsumsi atau menambah kekayaan wajib pajak bersangkutan, dengan nama apapun.

Secara detail yang menjadi objek pajak adalah:

Imbalan berkenaan dengan pekerjaan atau jasa yang diterima: gaji, upah, tunjangan, honorarium, komisi, bonus, gratifikasi, uang pensiuan dan lainnya termasuk natura. Hadiah undian, hadiah pekerjaan, hadiah suatu kegiatan dan penghargaan.Laba usaha.Keuntungan karena penjualan atau pengalihan harta. Penerimaan kembali pembayaran pajak yang dibebankan sebagai biaya dan pembayaran tambahan pengembalian pajak. Bunga termasuk premi, diskonto dan imbalan karena jaminan pengembalian utang. Dividen dan royalti.Sewa dan penghasilan lain yang berkaitan dengan penggunaan harta.Penerimaan atau pembayaran berkala.Keuntungan pembebasan utang dan keuntungan selisih kurs. Premi asuransi. Iuran yang diterima dalam menjalankan usaha.Tambahan kekayaan neto yang berasal dari penghasilan belum dikenai pajak. Penghasilan syariah. Imbalan bunga sesuai aturan perpajakan dan surplus Bank Indonesia.

11. Untuk mengisi Tabel gaji pokok, tunjangan istri, tunjangan anak, gaji kotor, pajak dan gaji bersih menggunakan fungsi …​


Jawaban:

Penjelasan:

Kolom Gaji Pokok:

Untuk mengisi kolom ini, cukup dengan memasukkan nilai gaji pokok pada masing-masing baris.

Kolom Tunjangan Istri:

Jika tunjangan istri diberikan sebesar 10% dari gaji pokok, maka rumusnya sebagai berikut: =Gaji_Pokok*10%

Kolom Tunjangan Anak:

Jika tunjangan anak diberikan sebesar Rp. 500.000,- per anak, maka rumusnya sebagai berikut: =500000*Jumlah_Anak

Kolom Gaji Kotor:

Gaji kotor adalah jumlah dari gaji pokok, tunjangan istri, dan tunjangan anak. Maka rumusnya sebagai berikut: =Gaji_Pokok+Tunjangan_Istri+Tunjangan_Anak

Kolom Pajak:

Jika pajak dikenakan sebesar 5% dari gaji kotor, maka rumusnya sebagai berikut: =Gaji_Kotor*5%

Kolom Gaji Bersih:

Gaji bersih adalah hasil dari gaji kotor dikurangi dengan pajak. Maka rumusnya sebagai berikut: =Gaji_Kotor-Pajak


12. Buatlah program untuk menghitung gaji pegawai Input : nama, golongan Proses : · Jika golongan A maka gaji pokok =Rp 1.500.000, tunjangan =5% * gaji pokok .jika gol.B maka gaji pokok = Rp.1.700.000,tunjangan = 10%* gaji pokok .jika gol.C maka gaji pokok = Rp.2.000.000, tunjangan = 15%*gaji pokok Output : Nama,gaji pokok,tunjangan serta gaji bersih Petunjuk : gunakan array dan petualangan


program untuk menghitung gaji pegawai, program ditulis dalam bahasa C

#include <stdio.h>

int main()

{

    char nama[50];

    char golongan;

    int gaji[3] = { 1500000, 1700000, 2000000 };

    int persentunjangan[3] = { 5, 10, 15 };

    int gajibersih, tunjangan;

    printf("Masukan nama Anda : ");

    scanf("%s", nama);

    printf("Masukan golongan (A, B, atau C) : ");

    scanf("%c", &golongan);

    golongan = golongan - 'A';

    tunjangan = gaji[ golongan ] * persentunjangan[ golongan ] / 100;

    gajibersih = gaji[ golongan ] + tunjangan;

    printf("Nama : %s\n", nama);

    printf("Gaji pokok : %i\n", gaji[ golongan ]);

    printf("Tunjangan : %i\n", tunjangan);

    printf("Gaji bersih : %i\n", gajibersih);

    return 0;

}

Pembahasan

Ada bagian yang aneh pada soal, "gunakan array dan petualangan". Array adalah bentuk variabel dengan alokasi memory untuk banyak nilai. Yang saya bingung adalah istilah "petualangan" disini. Setahu saya tidak ada istilah "petualangan" dalam bahasa pemrograman, mohon dikoreksi jika memang terjadi typo.

Pada solusi pemrograman yang dibuat, kita memanfaatkan golongan untuk menjadi index dari Array.

Pada potongan code berikut golongan = golongan - 'A';

Berfungsi untuk mengubah karakter A menjadi nilai 0. Perlu diketahui karakter adalah sebuah huruf yang memiliki nilai angka pada komputer, atau ASCII. Sehingga bisa dioperasikan menggunkana operator aritmatika seperti penjumlahan pengurangan, dan lainnya.

Maka jika golongan adalah A dan dikurangi 'A' maka hasilnya adalah 0

jika golongan adalah B dan dikurangi 'A' maka hasilnya adalah 1

jika golongan adalah C dan dikurangi 'A' maka hasilnya adalah 2

Nilai ini bisa kita gunakan sebagai index dalam array gaji yang sudah didefinisikan sebelumnya.

Sehingga kita bisa akses secara dinamis array gaji menggunakan golongan sebagi indexnya.

= 5,000,000 - 100,000

= 4,900,000

29. Seorang karyawan digaji harian, yaitu sebesar Rp. 100.000 untuk setiap hari kerja. Apabila karyawan tersebut tidak masuk kerja, maka gaji harian tidak dibayarkan dan dia didenda Rp. 20.000. Selain gaji harian, karyawan tersebut memperoleh tunjangan sebesar Rp 200.000. Pada akhir hari ke-25 ia mendapat gaji kotor Rp. 2.580.000. Berapa harikah dia bekerja?


Jawaban:

= 24 hari.

Penjelasan dengan langkah-langkah:

diket :

gaji harian (GH) = 100.000

denda (D) = 20.000

tunjangan (T) = 200.000

total hari kerja (THK) = 25

total gaji didapat (TGD) = 2.580.000

dit : berapa hari kerja (BHK) ?

jawab :

Pertama, tentukan dulu besarnya potongan bila tidak masuk kerja :

potongan (P) = GH + D

P = 100.000 + 20.000

P = 120.000

Kedua, tentukan total gaji yang semestinya (TGS) didapat dalam 25 hari kerja :

TGS = (GH x THK) + T

= (100.000 x 25) + 200.000

= 2.500.000 + 200.000

= 2.700.000

Ketiga, cari selisih (S) total gaji didapat (TGD) dengan total gaji yang semestinya didapat (TGS) :

= TGS - TGD

= 2.700.000 - 2.580.000

= 120.000

Keempat, cari berapa hari kerja (BHK) :

BHK = THK - (S : P)

= 25 - (120.000 : 120.000)

= 25 - 1

= 24

Jadi, karyawan tersebut hanya kerja sebanyak 24 hari.
