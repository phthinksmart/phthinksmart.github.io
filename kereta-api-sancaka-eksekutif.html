1. kereta api sancaka kelas eksekutif berangkat dari yogyakarta pukul 23.45 dan tiba di surabaya pukul 04.35. kereta api ekonomi berangkat dari yogyakarta pukul 06.30 dan tiba di surabaya pukul 11.27. selisih waktu tempuh kedua kereta api tersebut adalah?​


Kereta api sancaka kelas eksekutif berangkat dari yogyakarta pukul 23.45 dan tiba di surabaya pukul 04.35. kereta api ekonomi berangkat dari yogyakarta pukul 06.30 dan tiba di surabaya pukul 11.27. selisih waktu tempuh kedua kereta api tersebut adalah?

Penyelsaian  

Waktu merupakan interval atau jarak antara dua atau lebih kejadian atau keaadaan, atau juga bias dikatakan sebagai lamanya suatu kejadian atau keadaan berlangsung.

Diketahui KA Sancaka eksekutif brangkat pukul = 23.45 dan sampai 04.35 sehingga kereta tersebut menghabiskan waktu tempuh perjalanan sebesar 3 jam 50 menit. sedangkan kereta KA Sancaka ekonomi brangkat pukul = 06.30 dan sampai 11.27 sehingga kereta tersebut menghabiskan waktu tempuh perjalanan sebesar 4 jam 57 menit.

Dengan demikian selisish waktu tempuh kedua kereta tersebut adalah  3 jam 50 menit -  4 jam 57 menit =  1 jam 07 menit.

Waktu merupakan interval atau jarak antara dua atau lebih kejadian atau keaadaan, atau juga bias dikatakan sebagai lamanya suatu kejadian atau keadaan berlangsung.

• Satuan waktu

Berdasarkan pada satuan internasioanal (SI), satuan standar yang dimiliki oleh waktu yaitu detik atau sekon second.

• Konversi satuan waktu  

1 jam = 60 menit

1 menit = 60 detik

1 detik = 1000 milidetik

6. kereta api sancaka berangkat dari terminal A pada pukul 09.20 menit. kereta api tersebut akan menuju stasiun B tanpa henti di stasiun yang di lewatinya jika kereta api tersebut tiba di stasiun B pada pukul 12.20 dan jarak antara kedua stasiun 252km kecepatan kereta api adalahA.68km/jamB.72km/jamC.80km/jamD.84km/jam​


Kereta api berangkat dari terminal A pada pukul 09.20 dan tiba diterminal B pada pukul 12.20. Jika jarak kedua terminal adalah 252 km. Maka kecepatan kereta api adalah 84 km/jam (D).

Penyelesaian Soal :

Diketahui : Jam berangkat = 09.20

                  Jam tiba = 12.20

                   Jarak (s) = 252 km

Ditanya : Kecepatan (V) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung waktu tempuh (t) kereta api dengan menggunakan cara sebagai berikut :

waktu tempuh (t) = Jam tiba - Jam berangkat

                            = 12.20 - 09.20

                            = 03.00  atau   3 jam

LANGKAH KEDUA (II)

Hitung kecepatan (V) kereta api dengan menggunakan cara sebagai berikut :

Hitung selisih jarak dilihat berdasarkan kecepatan Tono dengan menggunakan cara sebagai berikut :

V = [tex]\frac{s}{t}[/tex]

keterangan : v = kecepatan (km/jam)

                      s = jarak yang ditempuh (km)

                     t = waktu tempuh (jam)

Maka caranya yaitu :

V = s/t

   = 252 km / 3 jam

   = 84 km/jam

∴Kesimpulan kecepatan kereta api tersebut adalah 84 km/jam.

S = 400 KM

V = 40 KM/Jam

T = S÷V

= 400 ÷ 40 = 10 Jam

Kereta Api Exekutif

S = 400 KM

V = S ÷ T

= 400 ÷ 8 = 50 Km/Jam

28. Quis 1. Paman melakukan perjalanan dari Surabaya ke Jember . Ayah berangkat pukul 07.00 mengedarai mobil dengan kecepatan rata-rata 60 km/jam. Jarak Surabaya – jember 120 km . Pukul berapa ayah sampai di Jember ?2. Kereta Api Sancaka berangkat dari Surabaya ke Jogjakarta berkecepatan rata-rata 80 km/jam .Jarak Surabaya – Jogjakarta 160 km. Kereta tersebut sampai pukul 09.00 . Pukul berapa kereta Sancaka berangkat dari Surabaya ?​


Jawab:

Penjelasan dengan langkah-langkah:

1. Diketahui:

  Jarak (s) =  120 km

  Kecepatan (v) = 60 km/jam

  Jam berangkat = 07.00

  Ditanya:

  Jam tiba ayah sampai Jember...........??

  Jawab:

   Menghitung waktu tempuh ayah dari Surabaya ke Jember dengan   menggunakan rumus sebagai berikut

   waktu tempuh (t) = [tex]\frac{s}{v}[/tex]

                                 = [tex]\frac{120 km}{60 km/jam}[/tex]

                                 = 2 jam

   Maka ayah akan sampai di Jember pukul 07.00 + 02.00  = 09.00

2. Diketahui:

   Jarak (s) =  160 km

   Kecepatan (v) = 80 km/jam

   Jam tiba = 09.00

   Ditanya:

   Jam kereta Sancaka berangkat...........??

   Jawab:

   Menghitung waktu tempuh kereta Sancaka dari Surabaya ke Jogja dengan   menggunakan rumus sebagai berikut

   waktu tempuh (t) = [tex]\frac{s}{v}[/tex]

                                 = [tex]\frac{160 km}{80 km/jam}[/tex]

                                 = 2 jam

Maka, kereta Sancaka berangkat dari Surabaya pukul 09.00 - 02.00  = 07.00
