12. ketika kita hendak menginstal os windows pada komputer, biasanya clien (pelanggan) meminta agar partisi d dan e tidak boleh di format. lalu dalam pembagian partisi ketika proses penginstalan, apa yang boleh di hapus dan bagaimana langkah-langkahnya?


Jawaban:

Ketika Anda mencoba untuk menginstal Windows Vista, Windows 7 atau Windows Server 2008 R2, Anda mungkin mengalami satu atau lebih gejala berikut ini:

Perangkat keras di mana Anda ingin menginstal Windows Vista, Windows 7 atau Windows Server 2008 R2is tidak terdaftar.

Anda tidak dapat memilih partisi cakram keras untuk menginstal Windows Vista, Windows 7 atau Windows Server 2008 R2.

Anda tidak dapat memformat partisi hard disk atau partisi.

Anda tidak dapat menetapkan ukuran yang benar untuk partisi hard disk.

Anda menerima pesan galat berikut:

Windows tidak dapat menemukan volume sistem yang memenuhi kriteria yang untuk penginstalan

Penyebab

Masalah ini dapat terjadi karena salah satu dari alasan berikut ini:

Windows tidak kompatibel dengan kontroler penyimpanan massal atau pengandar penyimpanan massal.

Kontroler penyimpanan massal atau pengandar penyimpanan massal telah kedaluwarsa.

Perangkat keras di mana Anda ingin menginstal Windows Vista, Windows 7 atau Windows Server 2008 R2 adalah disk dinamis.

Kabel data di komputer longgar, atau masalah perangkat keras lain telah terjadi.

Perangkat keras atau sistem file Windows rusak.

Anda mencoba untuk memilih partisi FAT32 atau jenis lain dari partisi yang tidak kompatibel dengan Windows Vista, Windows 7 atau Windows Server 2008 R2.

Pemecahan masalah

Untuk memecahkan masalah ini, gunakan salah satu atau lebih metode berikut.

Metode 1: Memverifikasi bahwa partisi yang kompatibel dengan Windows

Anda tidak dapat menginstal Windows di partisi FAT32. Selain itu, Anda harus benar mengkonfigurasi disk dinamis untuk digunakan dengan Windows. Untuk memverifikasi bahwa partisi kompatibel dengan Windows Vista, Windows 7 atau Windows Server 2008 R2, ikuti langkah-langkah berikut:

Untuk disk dinamis yang memiliki volume sederhana, menggunakan utilitas Diskpart.exe untuk mengkonfigurasi disk sebagai cakram yang aktif. Untuk informasi selengkapnya tentang cara menggunakan utilitas Diskpart.exe, klik nomor artikel berikut ini untuk melihat artikel di Pangkalan Pengetahuan Microsoft:

300415 penjelasan tentang utilitas baris perintah Diskpart.exe

Untuk FAT32 partisi, memformat partisi, atau mengubah partisi untuk partisi sistem file NTFS dengan menggunakan perintah Convert.exe .

Catatan Saat Anda memformat partisi, semua data yang dihapus dari partisi. Data ini mencakup semua berkas di partisi. Untuk informasi selengkapnya tentang cara menggunakan perintah Convert.exe, klik nomor artikel berikut ini untuk melihat artikel di Pangkalan Pengetahuan Microsoft:

314097 bagaimana cara menggunakan Convert.exe untuk mengubah partisi ke sistem berkas NTFS

16. Jika anda memiliki harddisk dengan kapasitas 80 GB Jelaskan cara melakukan partisi sebelum menginstal OS Windows 7 yang memerlukan minimal 10 GB partisi​


Jawaban:

Anda dapat melakukan partisi saat anda akan melakukan proses instalasi windows 7 tersebut, yaitu setelah anda masuk proses instalasi windows 7 (setelah memasukan CD Instalasi/USB Instalasi, dan melakukan setting untuk BIOS agar melakukan boot ke CD/USB instalasi, lalu rebooting, maka anda akan masuk ke proses instalasi Windows 7). Pertama setelah masuk ke menu tampilan awal Instalasi windows 7, klik Next di pojok bawah kanan, klik Install Now, klik untuk mencentang I Accept the license terms, klik next.

Nah disini ada 2 pilihan yaitu Upgrade dan Custom (advance), pilihlah Custom (Advance) agar anda dapat melakukan proses partisi sebelum melakukan instalasi windows 7 ke harddisk anda.

Nantinya setelah anda masuk ke menu Custom (advance) anda akan masuk ke menu Install Windows dengan tulisan "Where Do you want to install Windows", disini anda dapat melakukan configurasi harddisk sebelum anda melakukan installasi windows 7.

Jika harddisk yang anda gunakan adalah benar-benar baru, maka disana akan tampil nama harddisk anda yang masih Unallocated, untuk membuat partisi pada harddisk anda caranya adalah

Klik terlebih dahulu Drive Option (Advance) jika tombol New tidak ada pada tombol dibawah, setelah itu klik Newnantinya anda akan disuruh memasukan besaran untuk partisi baru anda dalam ukuran MB, untuk membuat partisi sebesar 10 GB, masukan lah nilai 10240 MB (karena 1024 MB = 1 GB)Klik ApplyAnda dapat melanjutkan membuat partisi selanjutnya sesuai kebutuhan

Sampai disini proses pembuatan partisi yang dapat dilakukan sebelum instalasi windows 7.

22. perbedaan antara bios legacy dan bios uefi dalam instalasi windows 7 partisi MBR, apa ya? tlong di bantu​


Jawaban:

bios legacy:

BIOS, singkatan dari Basic Input Output System, dalam sistem komputer IBM PC atau kompatibelnya merujuk kepada kumpulan rutin perangkat lunak yang mampu melakukan hal-hal berikut: Inisialisasi serta pengujian terhadap perangkat keras Memuat dan menjalankan sistem operasi.

bios uefi

UEFI adalah pembaruan teknologi terbaru dari BIOS yang berfungsi mengupdate fungsi BIOS yang sudah ketinggalan zaman. Awal mulanya UEFI muncul pada pertengahan tahun 1990an dimulai dari hadirnya standar EFI yang ada pada komputer server. Saat itu yang sudah menggunakan firmware EFI adalah perusahaan Intel dan HP.
