Jawaban:

A + B = 500.....(1)

8000A + 6000B = 3.250.000

Bagi kedua sisi dengan 2000

= 4A + 3B = 1625.....(2)

Eliminasi Variabel B

4A + 3B = 1625

(A + B = 500)×3

• 4a + 3b = 1625

3a + 3b = 1500

______________(-)

A = 125

Substitusi nilai A ke dalam Persamaan (1)

A + B = 500

125 + B = 500

B = 500-125 = 375

Jadi Materai A terjual sebanyak 125 buah Materai B terjual sebanyak 375 buah
