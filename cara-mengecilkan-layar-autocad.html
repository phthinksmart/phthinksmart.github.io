3. jelaskan fungsi tool bar pada layar autocad


1. Line = Membuat object dengan garis
2. Polyline = Membuat object dengan garis menyatu.
3.Splin = Membuat object dengan garis melengkung.
4.Elipse = Membuat object Elipse.
5. Cirxle = Membuat object Lingkaran.
6. Rectangle = Membuat object Persegi.
7.Region = Menyatukan object.
8.Extrude = Memberikan ketebalan pada object.
9.Move = Memindahkan object.
10.Mirror = Mengcopy object disebelah kiri, kanan, atas, dan bawah.
11.Trim = Memotong object.
12. Chamfer = Membantu object persegi menjadikan sudut object tidak lancip.
13. Fillet = Membuat object menjadikan sudut tidak lancip atau melengkung.
14. Union = Menggabungkan object.
15. Substract = Menghilangkan object 3D.
16. Ofset = Menggadakan object didalam atau diluar object.
17. Extrude Faces = Menambahkan ketebalan yang sudah dibuat.

4. Cara menggambar rumah di Autocad


Jawaban:

Membuat kapling/denah biasa terlebih dahulu.

Banyak cara yang di gunakan dalam membuat denah rumah ini bisa dengan membuat layernya terlebih dahulu atau dengan command multiline, tapi saya akan menggunakan command yang sudah saya berikan.

command yang di pakai : Line (L), Offset (O)

Gambarlah desain awalnya terlebih dahulu sesuai dengan ukuran yang sudah ada di gambar ini

Untuk ukuran dinding 15 mm

Membuat pintu dan jendelanya di tempat/space yang terpisah dari denah yang di buat tadi untuk lebih gampang di pindahkan nantinya.

command yang di pakai : Rectangle (Rec), Arc, Copy (Co)

Gambarlah sesuai dengan ukuran yang sudah di sediakan seperti gambar di bawah ini

Untuk gambar jendela yang di kotak merah

Memindahkan object pintu yang sudah di buat tadi ke object denah/kapling yang akan di buatkan pintu.

command yang di pakai : Copy (Co), Mirror (Mi), Rotasi (Ro)

Ubah tampilan object pintunya seperti yang di kotak merah dengan menggunakan command Mirror dan Rotasi

Lalu pindahkan object pintu ke space yang dimana akan di jadikan tempat keluar masuk seperti yang di lingkaran merah dengan menggunakan command Copy

sekarang giliran Memindahkan object jendelanya ke space yang menurut kalian cocok untuk di pasang jendela.

command yang di pakai : Copy (Co)

pindahkan object jendela yang sudah di buat sebelumnya (dalam kotak merah) ke space yang di inginkan (dalam lingkaran merah)

anda bisa membuat garis bantu agar mendapatkan midpoint/basepointnya untuk menyesuaikan letaknya

Membuat toilet.

Agar lebih mudah kita membuat toiletnya berada di luar denah, jadi nanti tinggal di pindahkan saja

command yang di pakai : Rec (R), (Offset (O), Arc, Mirror (Mi)

Gambarlah toilet sesuai ukuran seperti gambar di bawah ini dengan menggunakan command2 tersebut atau sesuai keinginan anda juga ga apa2

memindahkan toilet yang sudah di buat tadi ke kamar mandi.

command yang di pakai : Move (M)

pindahkan toilet tadi seperti gambar di bawah ini dengan menggunakan command Move

Membuat bak mandi.

command yang di pakai : Rectangle (Rec)

buatlah bak mandi dengan ukuran yang anda inginkan

membuat teras depan dan belakang dan lantai kamar mandi.

command yang di pakai : Line (L), Hatch (Ha)

membuat areanya terlebih dahulu dengan menggunakan command Line

Lalu memberi tampilan seperti lantai dengan menggunakan command Hatch dengan pengaturan pattern = NET, scale = 300, dan add pick points sesuai gambar di bawah ini

Untuk lantai kamar mandi ukuran scale = 150

Membuat carport/garasi.

command yang di pakai : Line (L), Offset (O), Copy (Co)

Buatlah garasi/carport dengan command2 tersebut seperti gambar di bawah ini atau sesuai keinginan anda

Memberi nama pada masing-masing bagian.

command yang di pakai : Dtext (Dt)

10. Bagaimana cara meng-import gambar dari autocad ke sketchup?


Copy gambar yang ingin anda import ke file baru agar gambar yang tidak perlu tidak memberatkan sketchup. seperti pada gambar saya hanya mengambil gambbar denah.Hapus layer selain layer tembok dan kolom.Explode semua gambar hal ini sangat penting agar memudahkan langkah selanjutnya di sketchup. Selain itu jika anda biasanya membuat tembok dengan multi line, jika tidak anda explode maka tidak akan terimport di sketchup.Bukua sketchup anda kemudian klik pada menu File>ImportSetelah muncul window seperti diatas pilih jenis filenya yaituAutoCAD files (*.dwg, *.dfx).Sebelum menekan open klik pada Options terlebih dahulu. Pada bagian Scale pilih unit yang anda gunakan dalam file DWG tersebut. Saat pertama kali unit defaultnya adalah feet, jika tidak anda atur maka ukuran pada sketchup tidak akan sesuai dengan ukuran di AutoCAD.Kemudian tekan Open maka gambar akan terimoprt ke sketchup.Gambar yang anda import akan berada dalam satu grup. Langkah berikutnya adalah membuat face atau bidang pada denah tersebut. Pertama download dulu plugin make_faces.rb kemudian instal. cara menginstalnya bisa anda lihat disini. setelah itu masuk ke group dan pilih semua lalu klik kanan dan pilih make face, jika tidak ada pilihan make face anda bisa membuat shortcutnya caranya bisa anda lihat disini.setelah selesai hasilnya akan seperti berikut.
lebih jelasnya bisa kunjungi http://www.tutorialsekecap.blogspot.co.id/2014/09/cara-mengimport-gambar-denah-dari.html

pada name:  pilih none ( untuk pengaturan print ) atau previous plot untuk melanjutkan print dengan pengaturan yang sudah dibuat sebelumnya, kali kita pilih saja none , langkah berikutnya
1.  pilih jenis printer yang kita gunakan untuk mencetak gambar autocad kita
2. pengaturan garis pada plot style table ( bisa dicoba-coba dengan klik gambar pensil yang ada disebelah kanan) secara default bisa langsung kita pilih salah satu sajaNone ( hasil output printer berwarnan )Mochrome.ctb (hasil output printer hitam putih tebal)Grayscale.ctb ( hasil output printer hitam tipis)untuk menghasilkan output prin sesuai selera dapat dimodifikasi melalui menu ini
3.  pilih ukuran kertas pada menu paper size
4. pilih hasi output object pada shade plot optionWireframe ( hasil output garis tembus pandang/ semua garis ter print)Hidden ( hasil output print object yang langsung tampak oleh mata hanya untuk gambar 3 dimensi )Rendered ( hasil output gambar 3 dimensi komplit dengan pencahayaan, warna material gambar 3 dimensi yang sudah dimodifikasi sebelumnya)pilihan lainya, fast, draft, normal silahkan dicoba-coba
5. pilih posisi kertas pada menu drawing orientationPotrait ( posisi kertas berdiri )Landscape ( posisi kertas tidur )
6. pilih are yang di print pada menu what to plotwindow ( hasil print sesuai dengan area yang dipilih ) untuk menggunakan pilihan ini klik window kemudian klik berikan kotak area yang akan diprint )display ( hasil print adalah gambar yang tampak pada layar )
7.pengaturan skala gambar pada menu plot scalefit to paper ( hasil print skala gambar menyesuaikan ukuran kertas ) untuk menggunakan pilihan ini berikan centang pada fit to papaer1:… untuk memilih skala yang diinginkancustom untuk membuat ukuran skala sendiri

24. bagaimana cara merubah scale di Autocad​


Jawaban:

1. Ketik sc pada layar AutoCAD untuk mengaktifkan skala perintah

2. Lalu pilih objek atau pilih objek yang ingin skala, lalu tekan spasi atau enter.

3. Pilihlah titik dasar atau titik referensi untuk memperbesar atau memperbesar gambar.

4. Selanjutnya, ketiklah skala yang kalian inginkan. Jika ingin memperbesar ketiklah angka> 1, tetapi jika Anda ingin memperbesar gambar ketiklah angka <1. Misalkan kalian ingin membuat gambar dengan skala 1: 100, maka kita bagi saja 1 dibagi 100 sama dengan 0,01 maka ketiklah angka 0,01 pada skala sehingga jadilah gambar dengan skala 1: 100.

27. apa saja icon yang ada di aplikasi AutoCAD?


Ada berbagai icon yang terdapat dalam aplikasi AutoCAD. Berikut ini adalah beberapa icon yang sering digunakan dan fungsinya.

Pull-down menus: menu-menu AutoCAD yang disusun secara menurun.Properties toolbar: mengakses toolbar untuk properti ruang kerja.Standard toolbar: mengakses toolbar dasar.Layer toolbar: mengakses toolbar untuk later.Edit toolbar: mengakses fitur edit gambar.Draw toolbar: mengakses fitur menggambar.Save: menyimpan file yang sudah pernah disimpan sebelumnya.Save as: menyimpan file baru.Open: membuka dokumen AutoCAD.Close: menutup lembar kerja AutoCAD.Export: mengubah jenis ekstensi file hasil kerja.Page setup: mengatur ukuran kertas, orientasi, dan hal-hal lain yang berhubungan dengan kertas lembar kerja.Zoom: memperbesar maupun memperkecil ukuran tampilan lembar kerja.Text style: mengubah jenis huruf yang digunakan.Line: membuat garis lurus sambung.Circle: membuat lingkaran.Erase: menghapus objek yang tidak diinginkan dari lembar kerja..

Pembahasan

AutoCAD merupakan perangkat lunak (software) yang digunakan untuk menggambar, baik gambar dua dimensi maupun gambar tiga dimensi. Program AutoCAD ini paling sering digunakan untuk membuat desain teknik yang diperuntukkan bagi pembangunan gedung, tata kota, dan desain teknik lainnya.
