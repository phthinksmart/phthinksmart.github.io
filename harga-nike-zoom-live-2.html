29. Nia membeli sepasang sepatu Nike dan 2 pasang Adidas seharga Rp 4.000.000 Nita membeli sepasang Nike dan sepasang Adidas Rp 2.500.000 hitunglah:a.Karin membeli sepasang Nike b.Tania membeli 3 pasang adidas

( Nike = x ) dan ( Adidas = y )

\_____________________/

x + 2y = Rp 4.000.000

x + y = Rp 2.500.000

__________________-

Eliminasi ( y )

2y - y = Rp 4.000.000 - Rp 2.500.000

[ y = Rp 1.500.000 ]

()()()()()()()()()()()()()()()()()

x + Rp 1.500.000 = Rp 2.500.000

x = Rp 2.500.000 - Rp 1.500.000

[ x = Rp 1.000.000 ]

Hasil / Result :

( Sepatu Nike ) = x = Rp 1.000.000

( Sepatu Adidas ) = y = Rp 1.500.000

Maka x + 3y = Rp 5.500.000
