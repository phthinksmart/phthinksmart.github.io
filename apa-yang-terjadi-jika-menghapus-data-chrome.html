1. Mau tanya apakah hapus data chrome ngaruh di google form?


Jawaban:

Tidak

Penjelasan:

Jika kamu menghapus data di Chrome, maka kemungkinan besar data yang ada di Google Form tidak akan hilang, karena data tersebut sudah dikirim kepada admin yang membuat form tersebut.

Begitu juga dengan Google Drive, Gmail dan Akun Google, datanya tidak akan terhapus, tergantung situs web nya. Namun aplikasi atau web selain Google yang login menggunakan akun Google mungkin akan otomatis log out.

11. bagaimana caranya dari menyimpan file dokumen dari google chrome, sampai mengubah destinationnya menjadi PDF?


simpan file dari google chrome, lalu masukan file yang akan dirubah dengan cara mengklik “Browse”, kemudian sorot file yang akan dirubah formatnya, kemudian klik “convert to pdf”, kemudian kita tunggu prosesnya. Setelah proses berakhir akan muncul pernyataan “Your files are ready for download” kemudian klik nama file yang telah di akhiri dengan .pdf maka akan munculah file anda dengan format pdf, dan yang terakhir save-lah file pdf.
