2. nilai sin phi/3 + cos phi/4 + tan phi/3.. berapa??


[tex]Answer[/tex]

Sin π/3 + Cos π/4 + Tan π/3

Nilai π dalam Materi Trigonometri yaitu 180. Jadi, substitusi nilai π tersebut!

Sin 180/3 + Cos 180/4 + Tan 180/3

Sin 60 + Cos 45 + Tan 60

Dari sini, kita harus tahu sudut - sudut istimewa. Karena sangat dibutuhkan dalam pelajaran trigonometri.

~ Sinus (Sin) ~

Sin 0 = 0

Sin 30 = ½

Sin 45 = ½ √2

Sin 60 = ½ √3

Sin 90 = 1

~ Cosinus ~

(Kebalikan Dari Sin)

Cos 0 = 1

Cos 30 = ½ √3

Cos 45 = ½√2

Cos 60 = ½

Cos 90 = 0

~ Tangen ~

Tan 0 = 0

Tan 30 = ⅓ √3

Tan 45 = 1

Tan 60 = √3

Tan 90 = (Tak Terdefenisi)

Jadi,

=> Sin 60 + Cos 45 + Tan 60

=> ½ √3 + ½ √2 + √3

=> 3/2 √3 + ½ √2

PEMBAHASAN

Persamaan trigonometri merupakan bentuk persamaan yang memuat variabel dan fungsi trigonometri. Adapun beberapa cara penyelesaian dari fungsi trigonometri adalah sebagai berikut.

1. Fungsi Sinus

ㅤ[tex]\boxed{\boxed{\begin{array}{l}\sf{sin\:f(x)=sin\:\alpha}\\\\\sf{penylesaiannya :}\\\sf{\:i)\:f(x)=\alpha+{k.360}^{o}}\\\sf{ii)\:f(x)=({180}^{o}-\alpha)+{k.360}^{o}}\end{array}}}[/tex]

ㅤ

2. Fungsi Cosinus

ㅤ [tex]\boxed{\boxed{\begin{array}{l}\sf{cos\:f(x)=tan\:\alpha}\\\\\sf{penylesaiannya :}\\\sf{\:i)\:f(x)=\alpha+{k.360}^{o}}\\\sf{ii)\:f(x)=-\alpha+{k.360}^{o}}\end{array}}}[/tex]

ㅤ

3. Fungsi Tangen

ㅤ [tex]\boxed{\boxed{\begin{array}{l}\sf{tan\:f(x)=tan\:\alpha}\\\\\sf{penylesaiannya :}\\\sf{f(x)=\alpha+{k.180}^{o}}\end{array}}}[/tex]

ㅤ

ㅤ

Diketahui:

[tex]\sf{tan\:3x-tan\:\dfrac{4}{3}\pi=0}[/tex]

ㅤ

Ditanyakan:

Nilai x yang memenuhi persamaan [tex]\sf{tan\:3x-tan\:\dfrac{4}{3}\pi=0}[/tex] adalah …

ㅤ

Jawab:

[tex]\begin{array}{lll}\sf{tan\:3x-tan\:\dfrac{4}{3}\pi}&=&\sf{0}\\\\\sf{tan\:3x}&=&\sf{tan\dfrac{4}{3}\pi}\\\\\sf{\:\:\:\:\:\:\:3x}&=& \sf{\dfrac{4}{3}\pi+{k.180}^{o}}\\\\ \sf{\:\:\:\:\:\:\:3x}&=&\sf{{240}^{o}+{k.180}^{o}}\\\\\sf{\:\:\:\:\:\:\:\:\:x}&=&\sf{{80}^{o}+{k.60}^{o}}\end{array}[/tex]

ㅤ

Karena syarat tidak disebutkan, saya anggap syaratnya 0° ≤ x ≤ 360°.

[tex]\sf{k = 0,\:x={80}^{o}}\\\sf{k=1,\:x={140}^{o}}\\\sf{k=2,\:x={200}^{o}}\\\sf{k=3,\:x={260}^{o}}\\\sf{k=4,\:x={320}^{o}}\\\sf{k = 5,\:x={380}^{o}\:\:\:(tidak \: memenuhi)}[/tex]

ㅤ

Jadi nilai x yang memenuhi persamaan [tex]\sf{tan\:3x-tan\:\dfrac{4}{3}\pi=0}[/tex] adalah [tex]\boxed{\sf{x={80}^{o}+{k.60}^{o}}}[/tex] atau [tex]\boxed{\sf{x=\{{80}^{o},\:{140}^{o},\:{200}^{o},\:{260}^{o},\:{320}^{o}\}}}[/tex] dengan syarat 0° ≤ x ≤ 360°.

7. Tan(phi per 4 - A) per tan(phi per 4 +A) = 1 per 4


[tex] \frac{tan(\frac{\pi}{4})-A}{tan(\frac{\pi}{4})+A} = \frac{1}{4} \\ \frac{tan(45-A)}{tan(45+A)} = \frac{1}{4} \\ 4(tan(45-A)) = tan(45+A) \\ 4(\frac{tan(45)-tan(A)}{1+tan(45)tan(A)} = \frac{tan(45)+tan(A)}{1-tan(45)tan(A)} \\ 4(\frac{1-tan(A)}{1+(1)tan(A)} = \frac{1+tan(A)}{1-(1)tan(A)} \\ \frac{4-4tan(A)}{1+tan(A)} - \frac{1+tan(A)}{1-tan(A)} = 0 \\ \frac{4-4tan^2(A)}{1-tan(A)} = 0 \\ 4-4tan^2(A) = 0 \\ tan^2(A) = 1 \\ tan(A) = +-1 [/tex]

Namun, tan(A) = 1 tidak memungkinkan, karena [tex] \frac{1 + tan(A)}{1 - tan(A)} [/tex] akan menjadi tidak real.

Jadi, penyelesaiannya
tan(A) = -1
tan(A) = tan 135°
A = 135° ± k.180°
