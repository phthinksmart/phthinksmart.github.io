Jawaban:

Shared hosting merupakan jenis layanan hosting yang sifatnya paling basic, di mana sebuah account hosting diletakan bersama dengan beberapa account hosting lainnya dalam satu cloud server yang sama. Kelebihan : Tidak perlu membeli Kekurangan : Fitur terbatas Akses terbatas Kinerja blog tidak bisa maksimal Tidak dapat melakukan backup data Tidak mendapat layanan customer service Tidak diberikan notifikasi ketika layanan akan ditutup (sebagian diberikan pemberitahuan) Data tidak aman Tidak bisa memasang iklan di blog Sebagian hosting gratisan mencantumkan brand hosting tersebutkelebihan dan kekurangan web hosting berbayar dan web hosting Gratisan :

• Layanan pada web hosting berbayar lebih lengkap jika dibandingkan dengan layanan yang terdapat pada web hosting gratisan. Untuk ituwebhosting indonesia atau luar yang berbayar sangat cocok bagi pemilik situs web profesional karena memiliki kompleksitas layanan yang lebih yang dapat digunakan untuk mengembangkan web.

11. Sebutkan tahapan-tahapan dalam menggunakan hosting ?​


Saya asumsikan yang dimaksud di sini adalah shared hosting ya, bukan yang vps atau cloud computing.

Pertama yang jelas kamu order dulu di provider preferensi kamu dan bayar. Biasanya kalau kamu order di provider yang "bonafide", mereka punya virtual account yang bisa langsung mengaktifkan hosting kamu setelah membayar tagihannya.

Ketika akun kamu sudah aktif, kamu akan punya akses ke cpanel. Biasanya kalau pakai cpanel "cPanel", kamu bisa mengaksesnya via "namadomain.com/cpanel". Tapi ada beberapa yang pakai software dari providernya sendiri, seperti Hostinger.

Jika kamu ingin mengupload website-mu, cari "File Manager" di cpanel. Tapi sebelum itu, pastikan semua file-nya dicompress terlebih dahulu biar mudah. Di file manager, masuk ke direktori yang bernama "public_html". Upload file website kamu yang berupa zip di situ, kemudian ekstrak. Dan voila! Website kamu online!

Apabila kamu menemui kesulitan yang lebih advance, kamu bisa kok tanya langsung ke Customer Service. Mereka siap melayani kamu. Tapi biasanya tergantung service yang kamu bayar. Semakin mahal semakin baik pelayanannya. Jika kamu bayar yang paling mahal, jam 2 dini hari pun kamu bisa dilayani.

21. tolong bantu jawab, buat besok sebutkan perbedaan dari shared hosting, dedicated hosting, cloud hosting, vps dan dedicated server​


Jawaban:

Perkembangan teknologi informasi terutama internet merubah cara hidup semua orang yang ada diseluruh dunia. Kebiasaan-kebiasaan yang dahulu masih dilakukan dengan cara yang kuno berganti dengan cara yang jauh lebih modern. Perubahan cara hidup semua orang yang ada di dunia tak lepas dari pesatnya perkembangan teknologi yang menunjang infakstruktur internet agar lebih optimal.

Salah satu contoh teknologi yang berkembang adalah cloud computing. Perkembangan cloud computing sangat membantu setiap orang karena dengan cloud computing, akses data ataupun berkas yang biasa tersimpan didalam hard drive dapat disimpan didalam cloud computer dan bisa diakses perangkat apa saja, dimana saja tanpa ada batasan seperti menyimpan data didalam hard drive.

Di Indonesia, banyak perusahaan baik itu perusahaan berskala kecil maupun menengah yang sudah memanfaatkan teknologi internet untuk usahanya, baik itu penggunaan aplikasi yang berbasis cloud, website dan menggunakan layanan koneksi internetcepat.

Baca juga: Pasang Internet Wifi Unlimited Di Kota Medan

Shared Hosting

Shared hosting adalah sebuah layanan web hosting dimana sebuah akun hosting akan ditaruh bersama dengan beberapa akun hosting lain didalam satu server yang sama. Sehingga dengan kata lain, shared hosting adalah sebuah server yang digunakan oleh banyak user atau akun hosting, menggunakan resource server yang sama baik itu prossesor, ram maupun storage. Tentu saja dengan banyaknya akun hosting yang menggunakan resource sama, maka website yang anda miliki tidak akan selalu bisa berjalan dengan stabil, tidak jarang website yang anda miliki akan sering down karena disebabkan user atau akun hosting lain sedang mendapat traffic pengunjung yang tinggi. Oleh karena itu, shared hosting tidak cocok untuk digunakan bagi anda yang memiliki usaha besar.

Cloud Hosting

Cloud Hosting dengan Shared Hosting adalah sebuah layanan web hosting namun terdapat perbedaan yang membuat teknologi cloud hosting jauh lebih unggul serta lebih sering digunakan oleh penyedia layanan hosting terbaik dibanding dengan shared hosting yang sudah mulai ditinggalkan.

Baca juga: Tips Cara Memilih Web Hosting Terbaik Untuk Website Anda

Hal ini karena teknologi cloud dianggap jauh lebih efesien ketimbang dengan shared hosting. Cloud hosting adalah sebuah layanan web hosting yang memiliki teknologi dimana sebuah akun hosting dapat terhubung pada banyak server, sehingga pengguna akun hosting tersebut tidak perlu takut dengan performa hostingnya down karena ketika hosting anda mendapatkan trafik pengguna yang tinggi, maka hosting akan otomatis berpindah ke server yang tidak memiliki trafik tinggi, alhasil hosting anda pun tidak akan bermasalah karena banyaknya trafik yang masuk ke website anda.

Virtual Private Server (VPS)

VPS adalah Virtual Private server. VPS dapat dikatakan sebagai titik tengah antara shared hosting dengan dedicated server. VPS adalah sebuah server yang menggunakan teknologi virtual untuk membagi hardware fisik menjadi beberapa virtual server.

Baca juga: Colocation Server Murah atau Dedicated Server Murah

Setiap VPS dapat menjalankan sistem operasinya masing-masing dan juga dapat di restart atau reboot terpisah meski didalam satu server terdapat beberapa virtual machine didalam server. Anda mendapatkan akses penuh terhadap VPS yang anda miliki, namun tetap saja anda harus berbagi resource dengan user lain, apalagi ketika user lain menggunakan VPS dengan sangat boros, maka anda akan ikut terkena imbasnya.

Dedicated Server

Sementara itu, dedicated server adalah sebuah layanan penyewaaan sebuah server kepada seorang user atau pengguna. Sehingga hanya anda sendiri yang akan menggunakan dan menempati sebuah server tersebut. Anda memiliki hak yang jauh lebih tinggi jika dibandingkan dengan VPS. Dedicated Server sangat cocok digunakan oleh anda yang memiliki
