3. berapakah luas maksimum trapesium tersebut (dalam cm2)? tuliskan langkah penyelesaiannya


Gambar dibawah adalah sebuah trapesium ABCD dengan AD = CD = BC = m cm, ∠DAB = ∠CBA = 2α  dan α adalah sudut lancip. Luas maksimum trapesium tersebut adalah ¾ √3 m² cm². Hasil tersebut diperoleh dengan menggunakan turunan pertama untuk mencari nilai stasionernya. Nilai stasioner diperoleh jika f’(x) = 0. Titik stasioner ada 3 jenis yaitu

Titik balik maksimum diperoleh jika f”(x₁) < 0 Titik balik minimum diperoleh jika f”(x₁) > 0 Titik belok diperoleh jika f”(x₁) = 0

Pembahasan

Diketahui

AD = CD = BC = m cm ∠DAB = ∠CBA = 2α α = Sudut lancip

Ditanyakan

Luas maksimum trapesium ABCD = ... ?

Jawab

Luas trapesium = ½ × (jumlah sisi yang sejajar) × tinggi

Luas trapesium = ½ × (DC + AB) × DP

Perhatikan trapesium pada gambar

AP = QB, segitiga APD siku-siku di P

cos A = [tex]\frac{AP}{AD}[/tex]

cos 2α = [tex]\frac{AP}{m}[/tex]

AP = m cos 2α

sin A = [tex]\frac{DP}{AD}[/tex]

sin 2α = [tex]\frac{DP}{m}[/tex]

DP = m sin 2α

QB = AP = m cos 2α

PQ = DC = m

AB = AP + PQ + QB

AB = m cos 2α + m + m cos 2α

AB = 2m cos 2α + m

Jadi luas trapesium adalah

L = ½ × (DC + AB) × DP

L = ½ × (m + 2m cos 2α + m) × m sin 2α

L = ½ × (2m + 2m cos 2α) × m sin 2α

L = (m + m cos 2α) × m sin 2α

L = m² sin 2α + m² sin 2α cos 2α

L = m² sin 2α + m² ½ sin 4α

L = m² sin 2α + ½ m² sin 4α

Agar diperoleh luas maksimum maka L’ = 0 (diturunkan terhadap α)

L’ = m² . 2 cos 2α + ½ m² . 4 cos 4α

L’ = 2m² cos 2α + 2m² cos 4α

L’ = 0

2m² cos 2α + 2m² cos 4α = 0

2m² (cos 2α + cos 4α) = 0

2m² ≠ 0 karena m adalah panjang sisi trapesium, maka  

(cos 2α + cos 4α) = 0

cos 2α + (2 cos² 2α – 1) = 0

2 cos² 2α + cos 2α – 1 = 0

Misal cos 2α = a

2a² + a – 1 = 0

(2a – 1)(a + 1) = 0

(2a – 1) = 0 atau (a + 1) = 0

a = ½                   a = –1  

Untuk a = ½  

cos 2α = ½    

cos 2α = cos 60ᵒ

2α = 60ᵒ  

α = 30ᵒ  

Untuk a = –1  

cos 2α = –1    

cos 2α = cos 180ᵒ

2α = 180ᵒ  

α = 90ᵒ

Karena α sudut lancip maka α = 30ᵒ

Luas maksimum dari trapesium ABCD adalah

L = m² sin 2α + ½ m² sin 4α

L = m² sin 2(30ᵒ) + ½ m² sin 4(30ᵒ)

L = m² sin 60ᵒ + ½ m² sin 120ᵒ

L = m² (½ √3) + ½ m² (½ √3)

L = ½ √3 m² + ¼ √3 m²

L = ¾ √3 m²

Jadi luas maksimum trapesium tersebut adalah ¾ √3 m² cm²

7. Gambar di bawah adalah sebuah trapesium ABCD dgn AD=CD=BC= m cm, dan DAB= CBA = 2a (alpha adalah sudut lancip). berapakah luas maksimum trapesium tsb (dalam m cm)? tuliskan langkah penyelesaian


Luas maksimum trapesium tersebut adalah [tex]\frac{3}{4} \sqrt{3}\: m^{2}[/tex].

Pembahasan

Trapesium merupakan bangun datar dua dimensi dengan 4 sisi dan 2 sisi tak sama panjang namun sejajar. Untuk mencari luas maksimum kita masuk ke materi aplikasi turunan.

PenyelesaianDiketahui :

AD = CD = BC = m cm

∠DAB = ∠CBA = 2α

α = Sudut lancip

Ditanyakan :

Luas maksimum trapesium ABCD = ... ?

Jawaban :

[tex]\boxed {Luas\: trapesium = \frac{1}{2} \times (a + b) \times tinggi}[/tex]

[tex]Luas\: trapesium = \frac{1}{2} \times (DC + AB) \times DP[/tex]

AP = QB sebab memiliki letak yang berhadapan, segitiga APD siku-siku di P

[tex]\cos A = \frac{AP}{AD}[/tex]

[tex]cos\: 2\alpha = \frac{AP}{m}[/tex]

[tex]\boxed {AP = m.\: cos\: 2\alpha}[/tex]

[tex]sin\: A = \frac{DP}{AD}[/tex]

[tex]sin\: 2\alpha = \frac{DP}{m}[/tex]

[tex]\boxed {DP = m.\: sin\: 2\alpha}[/tex]

[tex]QB = AP = m.\: cos\: 2\alpha[/tex]

[tex]PQ = DC = m[/tex]

[tex]\boxed {AB = AP + PQ + QB}[/tex]

[tex]AB = m.\: cos\: 2\alpha + m + m.\: cos\: 2\alpha[/tex]

[tex]AB = 2\:m.\: cos\: 2\alpha + m[/tex]

Jadi luas trapesium yang terbentuk adalah

[tex]\boxed {Luas\: trapesium = \frac{1}{2} \times (DC + AB) \times DP}[/tex]

[tex]Luas\: trapesium = \frac{1}{2} \times (m + 2\:m.\: cos\: 2\alpha + m) \times m.\: sin\: 2\alpha[/tex]

[tex]Luas\: trapesium = \frac{1}{2} \times (2\:m + 2\:m.\: cos\: 2\alpha) \times m.\: sin\: 2\alpha[/tex]

[tex]Luas\: trapesium = (m + m.\: cos\: 2\alpha) \times m.\: sin\: 2\alpha[/tex]

[tex]Luas\: trapesium = m^{2}\: sin\: 2\alpha + m^{2}\: sin\: 2\alpha\: cos\: 2\alpha[/tex]

[tex]Luas\: trapesium = m^{2}\: sin\: 2\alpha + m^{2}\: \frac{1}{2}\: sin\: 4\alpha[/tex]

[tex]Luas\: trapesium = m^{2}\: sin\: 2\alpha + \frac{1}{2}\: m^{2}\: sin\: 4\alpha[/tex]

Agar diperoleh luas maksimum maka l’ = 0 (diturunkan terhadap α)

[tex]I^{'} = m^{2}\: 2\: cos\: 2\alpha + \frac{1}{2}\: m^{2}.\: 4\: cos\: 4\alpha[/tex]

[tex]I^{'} = 2\:m^{2}\: cos\: 2\alpha + 2\: m^{2}.\: cos\: 4\alpha[/tex]

[tex]I^{'} = 0[/tex]

[tex]2\:m^{2}\: (cos\: 2\alpha + cos\: 4\alpha) = 0[/tex]

[tex]2\:m^{2} \neq 0[/tex]

maka selanjutnya

[tex](cos\: 2\alpha + cos\: 4\alpha) = 0[/tex]

[tex]cos\: 2\alpha\: + (2\:cos^{2} \: 2\alpha - 1) = 0[/tex]

[tex]2\:cos^{2} \: 2\alpha + cos\: 2\alpha - 1 = 0[/tex]

[tex](2\:cos^{2} \: 2\alpha - 1) (cos\: 2\alpha + 1) = 0[/tex]

[tex](2\:cos^{2} \: 2\alpha - 1) = 0\: atau\: (cos\: 2\alpha + 1) = 0[/tex]

[tex]\boxed {cos\: 2\alpha = \frac{1}{2} \:atau\:cos\: 2\alpha = -1}[/tex] 

Untuk cos 2α = 1/2  

[tex]cos\: 2\alpha = \frac{1}{2}[/tex]

[tex]cos\: 2\alpha = cos\: 60^{o}[/tex]

[tex]2\alpha = 60^{o}[/tex]

[tex]\alpha = \frac{60^{o}}{2}[/tex]

[tex]\boxed {\alpha = 30^{o}}[/tex]

Untuk a = -1  

[tex]cos\: 2\alpha = -1[/tex]

[tex]cos\: 2\alpha = cos\: 180^{o}[/tex]

[tex]2\alpha = 180^{o}[/tex]

[tex]\alpha = \frac{180^{o}}{2}[/tex]

[tex]\boxed {\alpha = 90^{o}}[/tex]

Karena soal meminta α sebagai sudut lancip, maka α = 30ᵒ. Luas maksimum dari trapesium ABCD adalah

[tex]Luas = m^{2}.\: sin\: 2\alpha + \frac{1}{2}\: m^{2}.\: sin\: 4\alpha[/tex]

[tex]Luas = m^{2}.\: sin\: 2\: (30^{o}) + \frac{1}{2}\: m^{2}.\: sin\: 4\: (30^{o})[/tex]

[tex]Luas = m^{2}.\: sin\: 60^{o} + \frac{1}{2}\: m^{2}.\: sin\: 120^{o}[/tex]

[tex]Luas = m^{2}.(\frac{1}{2}\: \sqrt{3}) + \frac{1}{2}\: m^{2}.(\frac{1}{2}\: \sqrt{3})[/tex]

[tex]\boxed {Luas = \frac{3}{4} \sqrt{3}\: m^{2}}[/tex]

21. Trapesium ABCD dengan AD=CD=BC = m cm dan ∆DAB=∆CBA = 2a (a adalah sudut lancip). Berapa luas maksimum trapesium tersebut (dalam m cm²)


Gambar dibawah adalah sebuah trapesium ABCD dengan AD = CD = BC = m cm, ∠DAB = ∠CBA = 2α  dan α adalah sudut lancip. Luas maksimum trapesium tersebut adalah ¾ √3 m² cm². Hasil tersebut diperoleh dengan menggunakan turunan pertama untuk mencari nilai stasionernya. Nilai stasioner diperoleh jika f’(x) = 0. Titik stasioner ada 3 jenis yaitu

Titik balik maksimum diperoleh jika f”(x₁) < 0 Titik balik minimum diperoleh jika f”(x₁) > 0 Titik belok diperoleh jika f”(x₁) = 0

Pembahasan

Diketahui

AD = CD = BC = m cm ∠DAB = ∠CBA = 2α α = Sudut lancip

Ditanyakan

Luas maksimum trapesium ABCD = ... ?

Jawab

Luas trapesium = ½ × (jumlah sisi yang sejajar) × tinggi

Luas trapesium = ½ × (DC + AB) × DP

Perhatikan trapesium pada gambar

AP = QB, segitiga APD siku-siku di P

cos A = [tex]\frac{AP}{AD}[/tex]

cos 2α = [tex]\frac{AP}{m}[/tex]

AP = m cos 2α

sin A = [tex]\frac{DP}{AD}[/tex]

sin 2α = [tex]\frac{DP}{m}[/tex]

DP = m sin 2α

QB = AP = m cos 2α

PQ = DC = m

AB = AP + PQ + QB

AB = m cos 2α + m + m cos 2α

AB = 2m cos 2α + m

Jadi luas trapesium adalah

L = ½ × (DC + AB) × DP

L = ½ × (m + 2m cos 2α + m) × m sin 2α

L = ½ × (2m + 2m cos 2α) × m sin 2α

L = (m + m cos 2α) × m sin 2α

L = m² sin 2α + m² sin 2α cos 2α

L = m² sin 2α + m² ½ sin 4α

L = m² sin 2α + ½ m² sin 4α

Agar diperoleh luas maksimum maka L’ = 0 (diturunkan terhadap α)

L’ = m² . 2 cos 2α + ½ m² . 4 cos 4α

L’ = 2m² cos 2α + 2m² cos 4α

L’ = 0

2m² cos 2α + 2m² cos 4α = 0

2m² (cos 2α + cos 4α) = 0

2m² ≠ 0 karena m adalah panjang sisi trapesium, maka  

(cos 2α + cos 4α) = 0

cos 2α + (2 cos² 2α – 1) = 0

2 cos² 2α + cos 2α – 1 = 0

Misal cos 2α = a

2a² + a – 1 = 0

(2a – 1)(a + 1) = 0

(2a – 1) = 0 atau (a + 1) = 0

a = ½                   a = –1  

Untuk a = ½  

cos 2α = ½    

cos 2α = cos 60ᵒ

2α = 60ᵒ  

α = 30ᵒ  

Untuk a = –1  

cos 2α = –1    

cos 2α = cos 180ᵒ

2α = 180ᵒ  

α = 90ᵒ

Karena α sudut lancip maka α = 30ᵒ

Luas maksimum dari trapesium ABCD adalah

L = m² sin 2α + ½ m² sin 4α

L = m² sin 2(30ᵒ) + ½ m² sin 4(30ᵒ)

L = m² sin 60ᵒ + ½ m² sin 120ᵒ

L = m² (½ √3) + ½ m² (½ √3)

L = ½ √3 m² + ¼ √3 m²

L = ¾ √3 m²

Jadi luas maksimum trapesium tersebut adalah ¾ √3 m² cm²
