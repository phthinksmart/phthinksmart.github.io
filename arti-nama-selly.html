6. perbandingan uang selly dan agung 5:9 apabila selisih uang mereka adalah 28000 maka banyak uang selly adalah


selly = s
agung = a

diketahui :
s : a = 5x : 9x
a - s = 28000

ditanya :
s = ?

dijawab
a - s = 28000
9x - 5x = 28000
4x = 28000
x = 7000

s = 5x = 5×7000 = 35000

jadi, banyak uang selly = Rp. 35000

banyak uang
selly
28.000/ (9 - 5) x 5 =
28.000/4 x 5 = Rp. 35.000Mapel : Matematika
Kelas : VII SMP
Bab : Perbandingan

Pembahasan :
Misal Ju = Jumlah Uang
Su = Selisih Uang
Jp = Jumlah Perbandingan
Sp = Selisih Perbandingan

Jp/Sp = Ju/Su
(9 + 5)/(9 - 5) = Ju/28.000
14/4 = Ju/28.000
7/2 = Ju/28.000
Ju = 7/2 x 28.000
Ju = 98.000

Ju = 98.000
5x + 9x = 98.000
14x = 98.000
x = 7.000

Banyak Uang Selly = 5x
Uang Selly = 5(7.000)
Uang Selly = Rp35.000
