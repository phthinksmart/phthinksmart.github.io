13. Kabel sepanjang 10m, memiliki luas penampang 2,5mm² dan hambatannya 25 ohm,hitunglah hambatan jenis kabel tersebut


Jawaban:

25 ohm

Penjelasan:

Untuk menghitung hambatan jenis kabel, kita perlu menggunakan rumus:

R = (ρ x L) / A

dimana:

R adalah hambatan kabel (dalam ohm)

ρ (rho) adalah hambatan jenis bahan (dalam ohm.m)

L adalah panjang kabel (dalam meter)

A adalah luas penampang kabel (dalam m²)

Untuk kabel dengan panjang 10m, luas penampang 2,5mm², dan hambatan 25 ohm, kita perlu mencari nilai ρ.

ρ = (R x A) / L

ρ = (25 x 0.0000025) / 10

ρ = 0.00000625 ohm.m

Setelah itu, kita dapat menghitung hambatan jenis kabel dengan rumus di atas:

R = (ρ x L) / A

R = (0.00000625 x 10) / 0.0000025

R = 25 ohm

Jadi, hambatan jenis kabel tersebut adalah 25 ohm.


14. sebuah penampang kabel memiliki panjang 8m dan ruas penampang 3cm dengan hambatan kawat 3,35 ×10-5 berapa hambatan tersebut​


Jawaban:

Untuk menghitung hambatan kawat, kita menggunakan rumus:

Hambatan = (Hambatan kawat per satuan panjang) × (Panjang kawat) / (Luas penampang)

Dalam hal ini, hambatan kawat per satuan panjang adalah 3,35 × 10^(-5) ohm/m, panjang kawat adalah 8 m, dan luas penampang adalah 3 cm² (atau 0,03 cm² setelah dikonversi ke meter persegi).

Mari kita hitung hambatan tersebut:

Hambatan = (3,35 × 10^(-5) ohm/m) × (8 m) / (0,03 m²)

Hambatan = 0,000268 ohm / 0,00003 m²

Hambatan = 8,93 ohm

Jadi, hambatan kawat tersebut adalah 8,93 ohm.

Penjelasan:

Panjang kabel mula-mula L dan luasnya A

Konstanta elastisitas sebanding dgn A tetapi berbanding terbalik dgn L.

Kabel dipotong dgn perbandingan 3 : 4

Konstanta kabel yang pendek menjadi k₁

k₁ : k = 1 / (3/7. L) : 1 / L = 7 : 3

k₁ = 7/3. k = 7k/3

Konstanta kabel yang panjang menjadi k₂

k₂ = 7k/4

Kabel yang panjang disayat menjadi dua dgn perbandingan luas 2:3

Konstanta kabel yang luasnya lebih kecil misalnya k₃.

k₃ : k₂ = 2/5. A : A = 2 : 5

k₃ = 2/5. k₂ = 2/5. 7k/4

k₃ = 7k/10

Kabel pendek disambung kabel panjang yang disayat dgn luas lebih kecil

1/ks = 1/k₁ + 1/k₃

1/ks = 1 / (7k/3) + 1 / (7k/10)

1/ks = 3/7k + 10/7k

1/ks = 10/7k

ks = 7k/10 = 0,7k

24. Rumus mencari konduktivitas pada kabel


Formula arus listrik adalah: 

I = Q/t (ampere)

Dimana:
I = besarnya arus listrik yang mengalir, ampere
Q = Besarnya muatan listrik, coulomb
t = waktu, detik

kuat arus dan waktu:

Q = I x t
I = Q/t
t = Q/I

Dimana : 
Q = Banyaknya muatan listrik dalam satuan coulomb 
I = Kuat Arus dalam satuan Amper. 
t = waktu dalam satuan detik.

Rumus untuk menghitung besarnya tahanan listrik terhadap daya hantar arus:

R = 1/G
G = 1/R

Dimana :
R = Tahanan/resistansi [ Ω/ohm] 
G = Daya hantar arus /konduktivitas [Y/mho]

26. Luas penampang dari sebuah kabel sebesar A, sedangkan hambatannya R. Kabel itu berbentuk silinder pejal. Jika dibuat lubang sepanjang sumbu kabel, hambatan kabel berlubang itu menjadi sebesar 5R/2. Berapakah luas penampang lubang itu? (A) 2A/5 (D) 2A/3 (B) 3A/5 (C) A/3 (E) A/5


Jawaban:

Luas penampang lubang adalah 3A/5.

(Opsi B)

Penjelasan:

Diketahui :

A1 = Luas penampang mula - mula = A

R1 = Hambatan mula - mula = R

R2 = Hambatan akhir = 5R/2

Ditanyakan :

Luas penampang akhir = ....

Penyelesaian :

[tex]\boxed{R=\frac{\rho.L}{A}}[/tex]

Hitung luas penampang akhir (A2) :

R1 x A1 = R2 x A2

R x A   = 5R/2 x A2

[tex]A2=\frac{R\times{A}}{\frac{5R}{2}}[/tex]

[tex]A2=R\times{A}\times\frac{2}{5R}[/tex]

[tex]A2=A\times\frac{2}{5}[/tex]

[tex]A2=\frac{2A}{5}[/tex]

Luas Lubang :

= A1 - A2

[tex]=A-\frac{2A}{5}[/tex]

[tex]=\frac{A}{1} -\frac{2A}{5}[/tex]

[tex]=\frac{5A}{5} -\frac{2A}{5}[/tex]

[tex]=\frac{3A}{5}[/tex]
