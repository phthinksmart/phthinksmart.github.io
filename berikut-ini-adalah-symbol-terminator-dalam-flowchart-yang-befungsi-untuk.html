1. Jelaskan dan gambar dari symbol flowchart proses dan terminator!


●》 Terminator, simbol untuk menunjukkan awal atau akhir dari aliran proses. Umumnya, diberi kata-kata ‘Start’, ‘End’, ‘Mulai’, atau ‘Selesai’.

●》 Process, simbol untuk menunjukkan sebuah langkah  proses atau operasi.

jawab

1. Buat Flowchart Program untuk menghitung luas lingkaran, di dalam flowchart tersebut definisilan Phi dengan symbol Preparation.

berikut adalah flowchart program untuk menghitung luas lingkaran dengan menggunakan simbol preparation untuk phi:

Start

Input r

Preparation: Phi = 3.14

Set L = Phi * r^2

Output L

Stop

penjelasan:

-program diawali dengan start

-user diminta untuk memasukkan nilai jari" atau r

-dilakukan pendefinisian nilai phi dengan menggunakan simbol preparation

-luas lingkaran (L) dihitung dengan rumus Phi * r^2

-luas lingkaran (L) ditampilkan menggunakan output

-program diakhiri dengan stop

2. buatlah flowchart yang sesuai dimana harus memiliki symbol decision untuk memilih volume dan luas balok tersebut.

berikut flowchart yg sesuai untuk memilih antara menghitung volume atau luas permukaan balok:

start

|

v

input panjang balok (p)

|

v

input lebar balok (l)

|

v

input tinggi balok (t)

|

v

decision: "hitung volume atau luas?"

|

|--- hitung volume

|     |

|     v

|     output "volume balok = p x l x t"

|     |

|     v

|     stop

|

|--- hitung luas permukaan

|     |

|     v

|     output "luas permukaan balok = 2(pl + pt + lt)"

|     |

|     v

|     stop

pada flowchart diatas pengguna diminta untuk memasukkan panjang (p), lebar (l), dan tinggi (t) dari balok terlebih dahulu. setelah selesai program akan meminta pengguna memilih antara menghitung volume atau luas permukaan balok dengan menggunakan symbol decision.

apabila menghitungnya dan mengeluarkan output "]Volume Balok = p x l x t"] sebelum berakhir.

]namun jika pengguna memilih untuk menghitung luas permukaan maka program akan menghitung dengan rumus Luas Permukaan Balok = 2(pl + pt + lt) dan mengeluarkan output sebelum berakhir.

Jawaban:

a. dokumen

Penjelasan :

Pengertian Flowchart adalah urutan proses kegiatan yang digambarkan dalam bentuk simbol. Flowchart (bagan alir) juga didefinisikan sebagai diagram yang menyatakan aliran proses dengan menggunakan anotasi- anotasi semisal persegi, panah, oval, wajik dll.

Penggunaan flowchart sangat penting, Bahkan Dr. Kauro Ishikawa seorang ahli teori organisasi, menjadikannya sebagai 1 dari 7 alat kualitas dasar (7 basic quality tools) yang harus dikuasai para anggota Quality Control Circle atau gugus kendali kualitas.

Melalui flowchart, kita bisa melihat langkah-langkah proses secara mendetail, lengkap dengan aktivitas yang terjadi. Flowchart banyak dipergunakan sebagai alat komunikasi dan dokumentasi.

17. 1. Buatlah gambar flowchart program menggoreng telur dadar dengan mengikuti langkah-langkah dibawah ini. Petunjuk : Perhatikan langkah-langkah dibawah, ada 8 langkah, artinya ada 8 symbol untuk membuat flowchart tersebut. Dari tiap langkah, silahkan dicocokkan dengan gambar symbol yang sesuai. ​

Pembahasan

Flowchart adalah diagram atau bagan alur yang menampilkan tahapan atau langkah-langkah dalam suatu proses. Setiap langkah yang terdapat di dalamnya dihubungkan dengan garis atau panah.

Ada beberapa jenis flowchart, yaitu:

Flowchart dokumen, berfungsi untuk menelusuri alur form.Flowchart program, berfungsi untuk menggambarkan prosedur.Flowchart proses, berfungsi untuk menggambarkan analisis langkah dalam prosedur.Flowchart sistem, berfungsi untuk menguraikan urutan dalam prosedur.Flowchart skematik, berfungsi untuk menampilkan alur prosedur menggunakan simbol-simbol tertentu.
