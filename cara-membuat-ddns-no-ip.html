Setiap angka dalam no IP tersebut akan dikonversi ke dalam bentuk biner dengan cara mengkonversi setiap angka yang ada ke dalam bentuk biner. cara menghitung first host address :
1. konversikan Ip address ke biner
2. pada host bit, jadikan semua bit = 0, kecuali yg terakhir = 1
3. konversikan kembali ke decimal

60 . 253 . 117 . 12 /16
00111100 . 11111101 . 01110101 . 00001100  ---> Ip address
                                   00000000 . 00000001  ---> host bit
sehingga = 60 . 253 . 0 . 1

cara menghitung last host address :
langkah ny sama, kecuali pada host bit, jadikan semua = 1, kecuali bit terakhir = 0

60 . 253 . 117 . 12 /16
00111100 . 11111101 . 01110101 . 00001100

15. apa yg dimaksud dengan no ip traffic


Apa yg dimaksud dengan no ip traffic?Jawaban Pendahuluan

Sebelum membahas mengenai apa yang dimaksud dengan no ip traffic, kita akan membahas secara singkat terlebih dahulu apa yang dimaksud dengan IP. IP adalah alamat pada jaringan komputer. Alamat IP biasanya terdiri dari kombinasi beberapa angka. Dengan pemberian IP ini, komputer akan bisa berkomunikasi dengan komputer lainnya.

Pembahasan

Setelah mengetahui apa itu IP, kita akan membahas mengenai no ip traffic, no ip traffing adalaj kondisi dimana komputer tidak bisa terhubung dengan komputer lainnya dapat suatu jaringan komputer. No IP Traffic bisa diakibatkan karena adanya kerusakan pada kabel LAN maupun dari sistem operasi atau perangkat lunak yang berhubungan dengan jaringan komputer.

Kesimpulan

Demikian pembahasan mengenai No Ip Traffic, No IP Traffic disebabkan oleh beberapa hal, baik dari hardware maupun software. Dengan mengetahui penyebabnya, kita bisa segera menanganinya agar komputer dapat berkomunikasi kembali dengan komputer lainnya. 17. Bagaimana anda menentukan kelas ip pada client dan pembuatan dokumentasi ip nya ?


Jawaban:

Ada dua cara yang biasa dilakukan untuk memberikan alamat IP address terhadap komputer dalam sebuah jaringan.

banner-content-image

1. Secara Manual (Statis)

Pengertian manual dalam hal ini adalah user dengan level administrator melakukan konfigurasi terhadap komputer satu persatu, menentukan, mengisi IP address, subnetmask, default gateway, dan DNS. Dengan kata lain semua konfigurasi dilakukan oleh administrator. Dalam hal ini berarti IP address yang telah dikonfigurasi bersifat statik, yang artinya setiap terjadi perubahan, maka sang administrator harus melakukan konfigurasi ulang secara manual dan satu persatu. IP yang dikonfigurasi secara manual ini sering diistilahkan sebagai IP Static.

2. Secara Dinamis

Metode ini menggunakan protocol Dynamic Host Configuration Protocol disingkat DHCP. Pada protokol ini, sistem konfigurasi ip address pada klien dalam sebuah jaringan dilakukan secara otomatis dan dinamis. Secara otomatis, itu berarti metode pengalokasian IP address, subnetmask, default gateway, DNS dan WNS dilakukan secara otomatis oleh sistem tanpa dilakukan manual oleh sang administrator. Secara dinamis berarti setiap jangka waktu tertentu konfigurasi dari IP

address dicek dan diubah sesuai dengan kondisi yang diinginkan oleh

administrator jaringan dan dilalukan secara otomatis tanpa melibatkan

peran dari si admin komputer.Metode pengalamat IP address semacam ini dikenal dengan istilah IP DHCP

Penjelasan:

Jadi, dalam pemberian ip address terhadap sebuah komputer dalam jaringan komputer terdapat dua metode yaitu, secara manual dan secara dinamis. Semoga bermanfaat.

23. Sebuah No IP 11.12.13.14 /8 tentukan network ID pada kelas no IP tersebut


Jawaban:

Kelas no IP tersebut adalah kelas B, dan network ID-nya adalah 11.0.0.0/8. Network ID adalah bagian pertama dari sebuah no IP yang menunjukkan jaringan di mana komputer tersebut berada. Pada kelas B, network ID terdiri dari dua blok pertama dari no IP, yaitu 11 dan 0. Jadi, network ID pada no IP 11.12.13.14/8 adalah 11.0.0.0/8.

25. Berikut pernyataan yang benar mengenai MASQUERADE adalah a. Suatu cara menghubungkan ip public menjadi ip private b. Membuat ip private c. Membuat ip public d. Mentranslasi ip public menjadi ip private e. Merubah ip menjadi biner​

Penjelasan:

MASQUERADE adalah teknik yang digunakan dalam jaringan komputer untuk menyembunyikan alamat IP asli dari host yang terhubung ke internet. Saat menggunakan MASQUERADE, alamat IP publik dari jaringan lokal akan diubah menjadi alamat IP privat sebelum data dikirim ke internet.
