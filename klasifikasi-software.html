2. jelaskan macam macam klasifikasi software berdasarkan jenisnya​


Secara umum, software diklasifikasikan menjadi empat jenis, yaitu

Sistem operasi, merupakan jenis software yang menangani langsung antara hardware dengan user atau pengguna. Sistem operasi harus diinstall pada hardware komputer. Contoh sistem operasi, mulai dari Windows 7 hingga Linux.Program Aplikasi (Aplication Programs), software yang kedua merupakan software yang dibuat untuk menyelesaikan tugas yang spesifik, mulai dari Microsoft Office untuk mengerjakan pekerjaan kantor, hingga aplikasi Corel Draw yang digunakan untuk membuat desain grafis.Bahasa Pemrograman atau yang dikenal dengan Programming Utility ini digunakan untuk membuat aplikasi. Secara umum bahasa pemrograman akan menerjemahkan perintah dalam bahasa pemrograman ke bahasa mesin atau biner sehingga dapat dijalankan oleh komputer.Utility Program, software ini merupakan software yang membantu pada beberapa tugas yang diberikan kepada komputer, mulai dari membersihkan file sampah hingga membantu melakukan troubleshooting hardware.

Pembahasan

Software atau yang dikenal dengan perangkat lunak dapat diartikan sebagai program yang dibuat dan berbentuk digital serta memiliki fungsi tertentu. Software terbagi menjadi beberapa jenis berdasarkan lisensinya yaitu

Komersial, pada jenis ini, pengguna harus membayar sejumlah harga untuk bisa menggunakan software secara resmi dan mendapatkan dukungan bantuan dari pembuat software.Trial Software, jenis lisensi ini sering dijumpai pada demo aplikasi atau program, semua fitur biasanya akan diberikan dalam beberapa hari, setelah masa trial habis, maka pengguna diharuskan membayar lisensi komersial untuk tetap bisa menggunakan software tersebut.Freeware, merupakan jenis lisensi yang tidak memungut biaya bagi penggunanya, namun pengguna tetap mendapatkan semua fitur dan akses penuh terhadap software tersebut.Shareware, lisensi ini hampir mirip dengan lisensi trial, namun pada shareware tidak memiliki batasan waktu penggunaan.

Untuk jenis freeware hingga shareware, meskipun tidak memungut biaya saat aplikasi mereka digunakan, biasanya pembuat aplikasi membuka donasi untuk mengumpulkan dana pengembangan software.

18. sebagai sarana interaksi antara pengguna dengan hardware software dapat diklasifikasi menjadi dua yaitu​


Jawaban:

1. Sistem Operasional

2. Aplikasi

Penjelasan:

System Operation (Operasi Sistem): Operasi sistem mengacu pada tindakan dan proses yang terlibat dalam menjalankan, mengelola, dan memelihara sistem komputer secara keseluruhan. Ini melibatkan operasi yang lebih mendasar dan fundamental yang diperlukan untuk menjalankan komputer dan menjaga keberfungsian sistem. Beberapa contoh operasi sistem termasuk:Booting: Proses memulai sistem komputer dengan memuat sistem operasi ke dalam memori dan menjalankannya.Instalasi dan pembaruan sistem operasi: Menginstal atau memperbarui sistem operasi komputer.Manajemen memori: Mengatur dan mengalokasikan ruang memori kepada program dan proses yang berjalan.Manajemen file: Mengelola file dan direktori dalam sistem komputer.Manajemen jaringan: Mengatur dan mengelola koneksi jaringan dan sumber daya jaringan.Application (Aplikasi): Aplikasi mengacu pada program perangkat lunak yang dirancang untuk memenuhi kebutuhan dan tujuan spesifik pengguna. Aplikasi berjalan di atas sistem operasi dan memberikan fungsi dan fitur yang lebih khusus dan spesifik sesuai dengan keperluan pengguna. Contoh aplikasi meliputi:Aplikasi perkantoran: Seperti pengolah kata, spreadsheet, presentasi, dan program pengelolaan data.Aplikasi desain grafis: Seperti program pengedit gambar, perangkat lunak desain grafis, dan perangkat lunak animasi.Aplikasi multimedia: Seperti pemutar media, editor video, dan perangkat lunak pengolahan suara.Aplikasi permainan: Permainan komputer dan perangkat lunak hiburan.Aplikasi web: Aplikasi yang berjalan di web browser, termasuk situs web interaktif, toko online, dan layanan daring lainnya.

Jawaban:

1. Antarmuka Pengguna (User Interface/UI)
2. Antarmuka Perangkat (Device Interface)

Penjelasan:

Sebagai sarana interaksi antara pengguna dengan hardware dan software, interaksi tersebut dapat diklasifikasikan menjadi dua kategori utama:

1. Antarmuka Pengguna (User Interface/UI): Antarmuka pengguna adalah bagian dari sistem yang memungkinkan pengguna berinteraksi dengan perangkat keras (hardware) dan perangkat lunak (software). Antarmuka pengguna mencakup elemen-elemen seperti tombol, ikon, menu, jendela, dan elemen visual lainnya yang digunakan oleh pengguna untuk mengoperasikan perangkat atau program. Antarmuka pengguna dapat berupa antarmuka grafis (GUI - Graphical User Interface) atau antarmuka berbasis teks (CLI - Command Line Interface).

2. Antarmuka Perangkat (Device Interface): Antarmuka perangkat mengacu pada hubungan dan komunikasi antara perangkat keras (hardware) dan perangkat lunak (software). Ini mencakup protokol, driver, dan perangkat penghubung yang memungkinkan perangkat keras berkomunikasi dengan perangkat lunak yang sesuai. Antarmuka perangkat juga dapat mencakup konektor fisik, kabel, atau jaringan yang digunakan untuk menghubungkan perangkat keras dengan sistem komputer atau perangkat lainnya.

Dengan adanya antarmuka pengguna dan antarmuka perangkat, pengguna dapat berinteraksi dengan perangkat keras dan perangkat lunak secara efektif. Antarmuka pengguna menyediakan lingkungan visual atau teks yang memungkinkan pengguna menjalankan perintah, mengakses fitur, dan mengelola aplikasi atau sistem. Antarmuka perangkat, di sisi lain, memungkinkan perangkat keras dikenali, diakses, dan dikontrol oleh perangkat lunak yang relevan.

Kedua jenis antarmuka ini bekerja bersama-sama untuk memfasilitasi interaksi yang efektif antara pengguna, perangkat keras, dan perangkat lunak, sehingga memungkinkan pengguna untuk menggunakan dan mengendalikan sistem komputer atau perangkat lainnya dengan mudah dan efisien.

Berikut pembahasan terhadap perbedaan Open Source Software dan Closed Source Software:

Open Source Software merupakan sistem operasi atau software yang mengizinkan pengguna atau orang selain developer (pembuat) untuk menyalin, memodifikasi atau bahkan menghapus source code sesuai dengan keinginan pengguna. Sistem operasi Open Source Software ini menyediakan pengembangan yang bebas bagi penggunanya. Hal ini dapat diterapkan karena akses untuk source code sistem operasi dapat dibuka dan dilihat, sehingga bagi kalian yang hanya sekedar melihat source code pada software yang digunakan atau suka dengan memodifikasi suatu sofware dapat menumpahkan kreatifitas tanpa perlu takut menyalahi lisensi yang sudah dibuat, tentu dengan menyesuaikan aturan dan hukum yang ada agar tidak digunakan untuk hal-hal yang negatif.

Contoh Open Source Software adalah Android dan Linux

Sementara Closed Source atau Prorietary Source Software merupakan sistem operasi yang yang source code-nya tidak dibuka untuk umum. Closed Source sistem operasi juga merupakan sistem operasi yang mempunyai lisensi yang artinya hanya dapat dikembangkan oleh pemilik atau developer terkait dan kebanyakan untuk mendapatkan atau menggunakannya dikenakan biaya (berbayar). Source code pada Closed Source tidak dapat dilihat atau dirubah sehingga pengguna atau orang lain yang tidak membeli atau bahkan orang yang sudah membeli software ini tidak diperbolehkan untuk menyalin, mengubah, hingga menghapus source code-nya

Contoh Closed Source atau Prorietary Source Software adalah Windows dan MacOs

29. Berdasarkan fungsinya, software dikategorikan sebagai .... a. Application software dan System software b. System software dan Compilers c. System software dan Compilers d. System software dan assemblers


Berdasarkan fungsinya, software yang digunakan dalam komputer dikategorikan menjadi Application software dan System software

Pembahasan:

Dalam penggunaan komputer, kita membutuhkan software atau perangkat lunak agar bisa mengoperasikan komputer. Tanpa adanya perangkat lunak, maka suatu komputer tidak bisa digunakan secara maksimal. Dilihat dari fungsinya, software secara garis besar dibedakan menjadi dua macam yaitu application software dan system software.

Application software merupakan perangkat lunak yang memiliki fungsi dan tujuan khusus. Kekhususannya ini membuat perangkat lunak ini efektif dalam mengerjakan suatu tugas secara spesifik, namun secara general tidak bisa. Fungsi spesifik antara lain untuk pengolahan kata bisa menggunakan microsoft word, untuk pengolahan gambar bisa menggunaka Adobe photoshop, dan pengolah angka bisa menggunakan Microsoft exel.

System software merupakan perangkat lunak yang membuat suatu komputer bisa bekerja secara umum seperti menyimpan file, mengorganisit file, hingga menggandakan suatu file. System software juga menjadi tempat untuk instalasi dari application software. Contoh dari system software antara lain Microsoft Windows 10, Ubuntu 19.04, Machintos.
