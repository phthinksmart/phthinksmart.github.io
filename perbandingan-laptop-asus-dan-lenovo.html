12. Mouse dom touchpad pada laptoptidak berfungsi bersamaan, (laptop Lenovo)langkah yangharus dilakukan ?​


Jawaban:

Aktifkan touchpad yang disable

Pada beberapa laptop biasanya sudah dilengkapi tombol khusus di keyboard untuk mengaktifkan dan menonaktifkan touchpad. Jadi bisa saja tanpa sengaja kamu menekan tombol tersebut sehingga membuat touchpad tidak berfungsi. Untuk mengatasinya, carilah tombol di keyboard yang memiliki ikon bergambar touchpad. Biasanya tombol tersebut berada di antara tombol F1 – F12. Jika berhasil ketemu, kamu hanya perlu melakukan tekan kombinasi Fn + tombol tersebut untuk mengaktifkan kursor yang freeze, misalnya Fn + F6 atau Fn + F9.

Cara lainnya, kamu bisa mengaktifkan touchpad dari Mouse Setting menggunakan mouse eksternal. Untuk Windows 7 atau 8, dari menu Start, lalu ketik Mouse. Setelah jendela Mouse Setting terbuka, lalu pilih opsi Enable Touchpad, klik Apply, dan OK. Sementara pada Windows 10, klik tombol Windows, lalu masuk ke Setting dan klik Device. Setelah itu, pilih Mouse & Touchpad untuk masuk ke dalam jendela pengaturan touchpad. Lihat opsi di bagian bawah dan klik Additional Mouse Options untuk mengaktifkan touchpad.

23. kekurangan laptop asus(?)


Kekurangan Laptop Asus

1. Asus terlalu cepat dalam mengup-date produknya, sehingga produk buatan sebelumnya menjadi cepat ketinggalan. Hampir setiap bulan Asus mengeluarkan produk terbaru, sehingga variannya terlalu banyak akibatnya hal ini bisa membingungkan pembeli. Di samping itu, kondisi ini juga sering membuat para pengguna yang telah membeli produk sebelumnya merasa kecewa dan menyesal.
2. Asus masih belum memiliki jumlah service center yang cukup di beberapa wilayah. Hal ini menjadi penyebab sering terjadinya penumpukan barang servisan. sehingga saat ada masalah maupun kerusakan, Anda harus menunggu antrian dan ini biasanya memakan waktu yang cukup lama.
3. Dalam produk netbooknya tidak disertai dengan Softcase dan tas, sehingga pembeli harus membeli lagi softcase dan tas untuk membawa netbooknya kemana-mana.

27. bagai mana cara menscreen pada laptop asus ?


Menggunakan Tombol Print Screen

Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Selanjutnya, Gadgeter dapat menekan tombol prt sc sysrq yang berada di keyboard

Nantinya secara otomatis gambar screenshot akan dihasilkan

Kemudian, Gadgeter dapat membuka aplikasi Microsoft Paint atau Adobe Photoshop atau pengolah gambar lainnya

Setelah aplikasi terbuka maka Gadgeter tinggal melakukan paste dengan kombinasi Ctrl + V di keyboard

Nantinya gambar hasil screenshot akan muncul dan dapat langsung diedit atau disimpan dengan format jpg, png, bmp, dan masih banyak lagi di Harddisk.

Menggunakan Kombinasi Tombol Win + Print screen Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Kemudian, Gadgeter bisa menekan tombol berlogo Windows + prt sc sysrq secara bersamaan dalam waktu beberapa detik

Hasilnya secara otomatis akan disimpan dalam folder Screenshots yang berada di dalam folder Pictures yang bisa diakses dengan Windows Explorer

Menggunakan software Snipping Tool

Snipping Tool merupakan software atau tool bawaan yang sudah hadir dalam Windows. Dengan menggunakan tool ini, Gadgeter dapat melakukan berbagai macam model screenshot dibandingkan kedua cara di atas.

Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Selanjutnya, Gadgeter dapat mencari software Snipping Tool di Start Menu

Setelah software ini dibuka, Gadgeter akan dihadapkan pada menu Snipping Tool

Untuk memulai screenshot, Gadgeter dapat memilih menu mode dan akan dihadapkan pada 4 mode sekaligus yang terdiri dari Free-form Snip, Rectangular Snip, Windows Snip, dan Full-screen Snip

Setelah memilih mode screenshot yang diinginkan maka Gadgeter dapat langsung melakukan screenshot

Hasilnya akan muncul di software Snipping Tool

Nantinya Gadgeter hanya tinggal memilih menu File > Save as dan hasil screenshot dapat langsung disimpan di folder yang telah ditentukan
