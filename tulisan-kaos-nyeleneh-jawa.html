Jawab:

10% x 140.000 = 10/100 x 140.000 = 14.000

Rp 140.000 - 14.000 = Rp 126.000

Jadi Harga Kaos 30% + 10% = Rp 126.000

22. plis help jangan ngasal/nyeleneh mau di kumpul​


Persamaan kuadrat umum

ax² + bx + c = 0 dengan a ≠ 0

(1) x² = -x² + 3x

<=> x² + x² - 3x = 0

<=> 2x² - 3x = 0 → pers. Kuadrat

(2) x² - 2x = x² - 3

<=> x² - x² - 2x + 3 = 0

<=> -2x + 3 = 0 → bukan pers. Kuadrat

(3) 2x - 3 = 2x² - x + 1

<=> 2x² - x - 2x + 1 + 3 = 0

<=> 2x² - 3x + 4 = 0 → pers. Kuadrat

(4) 16 - x² = 2x² + 5

<=> 2x² + x² + 5 - 16 = 0

<=> 3x² - 11 = 0 → pers. Kuadrat

Kesimpulan

Pernyataan yang menyatakan persamaan kuadrat adalah 1, 3 dan 4

Yang merupakan persamaan kuadrat adalah 3.

Bukti:

1. x² = -x² + 3x

x² + x² - 3x = 0

2x² - 3x = 0 (salah, karena tidak ada konstantanya)

2. x² - 2x = x² - 3

x² - x² - 2x + 3 = 0

-2x + 3 = 0 (salah, karena tidak ada koefisien x²)

3. 2x - 3 = 2x² - x + 1

0 = 2x² - x - 2x + 1 + 3

0 = 2x² - 3x + 4 (benar)

4. 16 - x² = 2x² + 5

0 = 2x² + x² + 5 - 16

0 = 3x² - 11 (salah, karena tidak ada koefisien x)
