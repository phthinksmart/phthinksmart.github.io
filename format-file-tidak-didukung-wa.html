6. Jelaskan perbedaan format file sistem komputer (OS) dan format file dokumen!


Format file system merupakan file-file yang dibutuhkan oleh sistem operasi ketika saat melakukan pekerjaannya. Biasanya file atau folder ini terletak di C:\Windows dimana difolder ini beberapa file atau folder tidak dapat dihapus atau dimodifikasi dengan sembarangan

Sedangkan file dokumen ialah file yang dibuat oleh user, seperti contoh file dokumen microsoft word, microsoft excel, file gambar, file video, dsb
