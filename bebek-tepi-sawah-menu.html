Tekanan

Diketahui:

F = gaya = 22.5 N A = luas bidang tekan = 0.1 m²

Ditanya:

P = tekanan ?

Jawaban

P = F/A

P = 22.5/0.1

P = 225 N/m² ✔

A = 0,1 m²

F = 22,5 N

tekanan

P = F/A

P = 22,5 N/0,1 m²

P = 22,5/0,1

P = 225 N/m²
