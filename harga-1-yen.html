7. Harga total riket masuk sebuah museum seni di jepang adalah 1.550 yen untuk 1 dewasa dan 3 peserta didik smp, serta 2.750 yen untuk 2 dewasa dan 5 peserta didik smp. Cari lah harga tiket masuk untuk masing masing 1 dewasa dan 1 peserta didik smp


" Jadilah orang yang jujur "

Diketahui :

Dewasa diibaratkan x

Peserta didik SMP diibaratkan y

1.550 yen = x + 3y

2.750 yen = 2x + 5y

Ditanya :

Berapa harga tiket masuk untuk masing masing 1 dewasa dan 1 peserta didik smp?

Jawab :

Eliminasi terlebih dahulu :

1.550 = x + 3y ..... ( Dikali 2)

2.750 = 2x + 5y

3.100 = 2x + 6y

2.750 = 2x + 5y

____________ _ (Dikurangkan agar x habis)

(3.100 - 2.750) = ( 2x - 2x ) + ( 6y - 5y )

350 = 0 + y

350 = y ( Harga tiket peserta didik SMP )

Selanjutnya disubstitusikan dengan menggunakan salah satu persamaan misalnya persamaan yang ada pada diketahui :

1.550 = x + 3y

1.550 = x + 3(350)

1.550 = x + 1.050

1.550 - 1.050 = x

500 = x ( Harga tiket dewasa )

Kesimpulan :Jadi, harga tiket masuk untuk 1 dewasa adalah 500 yen dan 1 peserta didik smp adalah 350 yen

Apersepsi Materi : A) Persamaan linear dua variabel ( PLDV )

Persamaan linear dua variabel ( PLDV ) adalah konsep matematika dimana ada sebuah persamaan dengan 2 variabel dengan bentuk umumnya ax + by = c, contoh dalam kehidupan sehari-hari misalnya Adi membeli 2 pena dan 3 pensil dengan harga 100.

Untuk menjawab PLDV adalah apersepsi yang harus dipahami :

1) Contoh Soal PLDV :

Persamaan (i) ➞ x + ay = bPersamaan (ii) ➞ x + cy = d

2) Metode Substitusi :

Dengan mengganti salah satu variabelnya dengan variabel yang baru didapatkan contoh :

Persamaan (i) x + ay = bx = b - ayPersamaan (ii) x + cy = d( b - ay ) + cy = d-ay + cy = d - b

3) Metode Eliminasi :

Dengan menambahkan atau mengurangkan kedua persamaan secara langsung agar salah satu variabel menjadi habis

x + ay = bx + cy = d___________ _0 + ay - cy = b - d

jika konstanta (a,b,c..) belum sama maka dapat dikalikan satu persamaan agar sama konstantanya dengan persamaan lain.

4) Operasi bilangan ketika di pindahkan ruaskan

Operasi Penjumlahan (+) dipindahkan ruaskan menjadi Pengurangan (-) begitu juga sebaliknyaOperasi Perkalian (x) dipindahkan ruaskan menjadi Pengurangan (÷) begitu juga sebaliknya

26. Harga total pembelian gabungan perangko 52 yen dan 82 yen adalah 700 yen. (1) nyatakanlah banyaknya perangko 82 yen yang dibeli dalam x, jika adalah banyaknya perangko 52 yen yang dibeli. (2) berapakah banyaknya masing-masing perangko yang dibeli? buatlah persamaan menggunakan hubungan antara harga masing-masing perangko untuk menentukan penyelesaiannya. tolong bantu jawab ya pake cara pilss jangan ngasal besok mau dikumpulin ​


Adapun jawaban dari soal tersebut yaitu nilai x adalah 4 dan nilai y adalah 6. Berikut cara penyelesaiannya.

Penjelasan dengan langkah-langkah:

Diketahui:

Persamaan:

52x + 82y = 700 yen

Ditanya:

Nyatakanlah banyaknya perangko 82 yen yang dibeli dalam x, jika adalah banyaknya perangko 52 yen yang dibeli?Berapakah banyaknya masing-masing perangko yang dibeli? buatlah persamaan menggunakan hubungan antara harga masing-masing perangko untuk menentukan penyelesaiannya?

Jawab:

Misal

Banyak perangko 52 yen = x

Banyak perangko 82 yen = y

Maka,

52x + 82y = 700 yen

82y = 700 – 52x

y = [tex]\frac{(700 - 52 x)}{82}[/tex]

Jadi banyak perangko 82 yen yang dibeli sejumlah  [tex]\frac{(700 - 52 x)}{82}[/tex] dengan x adalah banyak perangko 52 yen yang di beli.

Misal x = 4

Maka

y = [tex]\frac{(700 - 52(4))}{82}[/tex] = 6

Jadi, nilai x adalah 4 dan nilai y adalah 6.
