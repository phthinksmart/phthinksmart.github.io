5. saya sudah pake root zenfone, tapi ko masih gagal... masih belum bisa root.. gimana solulsinya?


ini waktu saya me-root Asus Zenfone :
Pertama masuk ke Setting < Security < Centang “Unknown Sources”.
mematikan jaringan WiFi atau paket data anda atau juga bisa aktifkan Airplane Mode.
Jangan lupa, Download dulu Root ZenFone 1.3r APK , setelah selesai langsung install aplikasi tersebut.
Setelah diinstall, akan muncul aplikasi baru bernama Root ZenFone.
Buka aplikasi Root ZenFone dan centang kotak Do Not Ask. setelah itu pilih tombol sebelah kanan / OK.
Pilih SuperSU(By Chainfire) lalu pilih tombol Root yang berada di paling bawah.
Tunggu beberapa saat hingga Demonstrasi Mode Selesai lalu akan muncul pesan “You Have Activated this demo device OFFLINE successfully” dan setelah itu tutup aplikasi dan Reboot / Restart ponsel anda.
Setelah Asus Zenfone 5 anda kembali menyala, buka kembali Root Zenfone. Jika Root sukses, SuperSU akan meminta akses Root ke aplikasi tersebut dan dengan begitu Asus Zenfone 5 anda telah berhasil di Root.

Semoga bermanfaat :)
