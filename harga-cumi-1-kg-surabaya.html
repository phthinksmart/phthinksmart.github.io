22. Bu Santi membeli 5 kg bebek, 2 kg ayam, dan 1 kg cumi dengan harga Rp240.000,00. Bu Ami membeli 3 kg bebek, 1 kg ayam, dan 2 kg cumi dengan harga Rp250.000,00. SPLTV

Buat persamaan ayam (a), bebek (b), cumi (c):

Bu Santi beli 5 kg bebek, 2 kg ayam, dan 1 kg cumi seharga Rp240.000

2a + 5b + c = 240.000 ............................................................ (i)

Bu Ami beli 3 kg bebek, 1 kg ayam, dan 2 kg cumi seharga Rp250.000

a + 3b + 2c = 250.000  ............................................................ (ii)

Bu Mei beli 1 kg bebek, 3 kg ayam, dan 2 kg cumi seharga Rp270.000

3a + b + 2c = 270.000 ............................................................. (iii)

Eliminasi pers (iii) dan (ii):

3a + b + 2c = 270.000

a + 3b + 2c = 250.000

_________________-

2a - 2b = 20.000

a - b = 10.000  ............................................................................. (iv)

Eliminasi pers (i) dan (ii):

2a + 5b + c = 240.000   |×2|   4a + 10b + 2c = 480.000

a + 3b + 2c = 250.000   |×1 |       a + 3b + 2c = 250.000

                                                ___________________-

                                                 3a + 7b = 230.000 ............. (v)

Eliminasi pers (v) dan (iv):

3a + 7b = 230.000   |×1 |   3a + 7b = 230.000

a - b = 10.000           |×3|   3a - 3b = 30.000

                                          ______________-

                                                10b = 200.000

                                                    b = 200.000/10

                                                    b = Rp20.000

Substitusi b ke pers (iv):

a - b = 10.000

a - 20.000 = 10.000

a = 10.000 + 20.000

a = Rp30.000

Substitusi a dan b ke pers (ii):

a + 3b + 2c = 250.000

30.000 + 3(20.000) + 2c = 250.000

30.000 + 60.000 + 2c = 250.000

2c = 250.000 - 90.000

c = 160.000/2

c = Rp80.000

Jika bu Desi beli 3 kg bebek, 2 kg ayam, dan 1 kg cumi

Harga yang harus dibayar = 3b + 2a + c

= 3(20.000) + 2(30.000) + 80.000

= 60.000 + 60.000 + 80.000

= Rp200.000

Jadi, harga yang harus dibayar Bu Desi sebesar Rp200.000,00.
