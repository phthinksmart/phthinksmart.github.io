SOAL ↓

1) 2³ + 6!

Penjelasan dengan langkah-langkah:

Pembahasan :

Eksponen atau bilangan berpangkat itu operasi hitung yang berbentuk pangkat perkalian berulang .

Faktorial atau kaidah pencacahan itu bentuk hasil perkalian mundur yang berbentuk bilangan positif dan dilambangkan faktorial dengan ( n! ).

Soal :

2^3 + 6! =

Penyelesaian soal :

= 2^3 + 6!

= 2 x 2 x 2 + 6 x 5 x 4 x 3 x 2 x 1

= 4 x 2 + 6 x 5 x 4 x 3 x 2 x 1

= 8 + 6 x 5 x 4 x 3 x 2 x 1

= 8 + 30 x 4 x 3 x 2 x 1

= 8 + 120 x 3 x 2 x 1

= 8 + 360 x 2 x 1

= 8 + 720 x 1

= 8 + 720

= 728
