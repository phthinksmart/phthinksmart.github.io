8. Panjang penggaris adalah.......batang korek api


Korek api merupakan salah satu alat untuk menyalakan api secara terkendali. Sebatang korek api terdiri dari dua bagian. Bagian pertama adalah batang kayu pinus. Bagian kedua adalah bahan kimia, biasanya fosfor, yang berada di salah satu ujung batang kayu dan biasanya berbentuk elips. Untuk dapat menyalakan api, batang korek api harus digesekkan dengan cepat pada sisi kiri dan kanan kotak korek api. Kedua sisi ini telah disematkan bahan kimia tertentu yang dapat memicu terbakarnya fosfor pada ujung korek api.


Pembahasan

Pada kesempatan ini, soal meminta kita untuk menentukan perbandingan antara panjang sebatang korek api dengan panjang penggaris.

Sebagai catatan, soal ini tidak memberikan informasi detil tentang jenis penggaris dan korek api yang digunakan. Untuk itu, kakak akan menggunakan informasi tentang panjang penggaris secara umum yaitu 30 cm dan panjang sebatang korek api yang berkisar di angka 52 - 57 mm.

Dengan menggunakan informasi ini, berikut kakak akan mencoba menjawab pertanyaan tersebut.


Diketahui:

Panjang penggaris (p) = 30 cm

Panjang sebatang korek api (k) = 5,2 - 5,7 cm


Ditanya

Perbandingan/rasio panjang penggaris dengan korek api (r)


Jawaban

r = p/k

r = (30/5,2) - (30/5,7) -> tanda '-' bukan berarti kurang, tapi hingga atau sampai

r = 5,7692 - 5,26315 kali


Dengan demikian, jika panjang sebuah penggaris adalah 30 cm dan panjang sebatang korek api berkisar di 5,2 - 5,7 cm, maka perbandingan panjang penggaris adalah sekitar 5,2 - 5,7 kali panjang sebatang korek api.

Rangkuman Matematika: Satuan Tidak Baku

Pernahkah teman-teman mengukur benda yang ada di sekitarmu?

Kita bisa mengukur benda menggunakan satuan baku dan satuan tidak baku.

Satuan baku adalah satuan yang sudah ditetapkan secara internasional. Misalnya, untuk mengukur panjang, satuan internasionalnya adalah meter.

Alat yang digunakan untuk mengukur satuan panjang misalnya ada penggaris atau meteran.

Sedangkan, satuan tidak baku adalah satuan yang tidak ditetapkan dengan satuan pengukuran tertentu.

Contoh mengukur menggunakan satuan tidak baku adalah kita bisa mengukur panjang sebuah benda tanpa penggaris atau meteran, melainkan benda lain.

Misalnya sebuah sisir jika diukur menggunakan peniti hasilnya 5 peniti. Namun jika sebuah sisir diukur menggunakan baut, hasilnya 7 baut.
