tangga nya

#NoCopasDiketahuiNabati memiliki beras seberat ➡️ 135 kgTango memiliki beras seberat ➡️ 67 dagNissin memiliki beras seberat ➡️ 100.000 cg

Ditanya

Berat Beras keseluruhan ( hg )

Pendahuluan

Berikut Tangga Satuan Massa.

kg ( Kilogram )

hg ( Hektogram )

dag ( Dekagram )

g ( Gram )

dg ( Desigram )

cg ( Centigram )

mg ( Miligram )

Keterangan

Setiap naik 1 tangga ➡️ Dibagi 10 ( ÷ 10 )Setiap turun 1 tangga ➡️ Dikali 10 ( × 10 )

Penyelesaian Soal

Permisalan

Berat beras keseluruhan ➡️ xBerat beras Nabati ➡️ nBerat beras Tango ➡️ tBerat Beras Nissin ➡️ ns

Maka :

x = n + t + ns

x = 135 kg + 67 dag + 100.000 cg

x = ( 135 × 10 ) + ( 67 ÷ 10 ) + ( 10.000 ÷ 100.000 )

x = 1.350 hg + 6,7 hg + 10 hg

x = 1.356,7 hg + 10 hg

x = 1.367,7 hg

Kesimpulan

Jadi, Berat Beras keseluruhan adalah 1.367,7 hg.
