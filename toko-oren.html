10. nisbah bilangan epal kepada bilangan oren ialah 9:7 .Jumlah bilangan buah-buahan ialah 208. Hitung beza antara bilangan epal dan bilangan oren


Diketahui :
- Bilangan epal : Bilangan oren = 9 : 7
- Jumlah bilangan = 208

Maka :
• Beza antara kedua bilangan
= (9 - 7)/(9 + 7) x 208
= 2/16 x 208
= 26
