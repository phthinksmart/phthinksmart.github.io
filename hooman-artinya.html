Jika hooman membeli masing-masing satu bungkus wishkash, Royal canin dan Me-O di toko yang sama, maka ia harus membayar sebesar Rp9.850

PEMBAHASAN

Sistem Persamaan Linear Tiga Variabel atau yang dikenal dengan SPLTV adalah sistem persamaan yang memiliki 3 persamaan linear dan mempunyai 3 variabel ( x , y , z ) atau bisa yang lain.

Menentukan himpunan penyelesaian dari SPLTV ini ada beberapa metode , diantaranya Metode Substitusi (Pengganti dan Menggabung)Metode Eliminasi (Menghilangkan)Metode Gabungan (Subs dan Elim)Metode determinan

PENYELESAIAN

Misal

x = Wishkashy = Royal Caninz = Me-O

Buat persamaan

2x + 2y + 2z = 19.700 ... (1)

2x + y + 2z = 14.000 ... (2)

2x + 3y + z = 17.200 ... (3)

» Eliminasi pers. 1 dan 2

2x + 2y + 2z = 19.700

2x + y + 2z = 14.000

_______________ (-)

y = 5.700

» Eliminasi pers. 2 dan 3

2x + y + 2z = 14.000 | x 1|

2x + 3y + z = 17.200 | x 2|

2x + y + 2z = 14.000

4x + 6y + 2z = 34.400

_____________ (-)

-2x - 5y = -20.400 ... (4)

» Substitusi nilai y ke persamaan 4

-2x - 5y = -20.400

-2x - 5(5.700) = -20.400

-2x - 28.500 = -20.400

-2x = -20.400 + 28.500

-2x = 8.100

x = -4.050

» Substitusi nilai x dan y ke persamaan pertama

2x + 2y + 2z = 19.700

2(-4.050) + 2(5.700) + 2z = 19.700

-8.100 + 11.400 + 2z = 19.700

3.300 + 2z = 19.700

2z = 19.700 - 3.300

2z = 16.400

z = 8.200

» Jika hooman membeli masing-masing satu bungkus wishkash, Royal canin dan Me-O di toko yang sama, maka ia harus membayar sebesar

= x + y + z

= -4.050 + 5.700 + 8.200

= Rp9.850

KESIMPULAN

Jadi , jika hooman membeli masing-masing satu bungkus wishkash, Royal canin dan Me-O di toko yang sama, maka ia harus membayar sebesar Rp9.850

Jika hooman ingin membeli 8kg wetfood dan 20kg dryfood, maka total belanjaan Hooman adalah...Rules:- No Copas- No ngasal- No spam- Yg ngasal auto report- Usaha kan jawaban antara grade 4/5 ^^ ​


~Spldv~

pendahuluan :

persamaan linear dua variabel adalah persamaan yg mempunyai dua variabel.

pengertian² di Spldv

eliminasi = penghilangan salah satu variabelpersamaan = kalimat matematika yg dihubungkan dengan tanda sama dengan (=)subtitusi : mengganti salah satu variabel ke variabel yg lainvariabel : nilai yg belum di ketahui kebenarannyametode campuran : metode ini biasanya menggabungkan metode eliminasi dengan subtitusi

pembahasan :

diketahui :

Amenk membeli 6kg wetfood dan 10kg dryfood seharga Rp 84.000,00

Lupus membeli 10kg wetfood dan 5kg dryfood seharga Rp 70.000,00

ditanya :

Jika hooman ingin membeli 8kg wetfood dan 20kg dryfood, maka total belanjaan Hooman adalah

dijawab :

ubah wetfood menjadi variabel x

ubah dryfood menjadi variabel y

setelah itu kita bentuk persamaan linearnya

6x + 10y = 84.000 ....... ( persamaan 1 )

10x + 5y = 70.000 ......... ( persamaan 2 )

setelah itu cari nilai x dan y menggunakan metode campuran

eliminasi variabel y untuk mendapatkan nilai x

6x + 10y = 84.000 |x1| 6x +10y = 84.000

10x + 5y =70.000 |x2| 20x + 10y = 140.000

——————————— -

-14x = -56.000

x = -56.000/-14

x = 4.000

subtitusikan nilai x ke persamaan 1 untuk mendapatkan nilai y

6x + 10y = 84.000

6(4.000) + 10y = 84.000

24.000 + 10y = 84.000

10y = 84.000 - 24.000

10y = 60.000

y = 60.000/10

y = 6.000

jadi di dapatkan bahwa harga 1 wetfood adalah 4.000 dan harga 1 dryfood adalah 6.000

total yg harus di bayar hooman

8 kg wetfood = 8x

20 kg dryfood = 20y

bentuklah aljabar untuk mengetahui yg harus dibayar hooman

= 8x + 20y

= 8(4.000) + 20(6.000)

= 32.000 + 120.000

= 152.000

kesimpulan :

jadi yg harus di bayar hooman adalah Rp 152.000,00

Jawaban:

Pada sebuah toko, Amenk dan Lupus membeli wetfood dan dryfood dengan merek yg sama. Amenk membeli 6kg wetfood dan 10kg dryfood seharga Rp 84.000,00. Sedangkan Lupus membeli 10kg wetfood dan 5kg dryfood seharga Rp 70.000,00. Jika hooman ingin membeli 8kg wetfood dan 20kg dryfood, maka total belanjaan Hooman adalah Rp 152.000,00.

Pembahasan

sistem persamaan linier dua variable [SPLDV] adalah sistem persamaan yang memiliki dua buah variable, seperti x & y, ataupun a & b, dan dua pasangan variable lainnya. yang pangkat tertinggi dari variablenya adalah 1 (tidak bentuk pangkat 2 dan sebagainya)

Dalam penyelesaian soal SPLDV ini, saya memakai metode campuran (eliminasi & subtitusi). agar lebih mudah untuk menyelesaikan soal ini.

Metode campuran merupakan metode gabungan dari eliminasi dan subtitusi, langkah penyelesaian soal SPLDV dengan metode campuran sebagai berikut:

samakan koefisien dari salah satu variable bisa (x / y).eliminasi variable y untuk mendapat nilai xSubtitusikan nilai x  ke salah satu persamaan untuk mendapat nilai yJika ingin mencari nilai y terlebih dahulu juga tidak masalah.

Penyelesaian

Dimisalkan dengan variable:

wetfood = x

dryfood = y

Persamaan nya menjadi;

Amenk

⇔ 6x + 10y = 84.000 | ×1 |

⇔6x + 10y = 84.000 .... (pers. 1)

Lupus

⇔ 10x + 5y = 70.000 | ×2 |

⇔20x + 10y = 140.000 .... (pers. 2)

Eliminasi kedua persamaan;

disini saya balik antara pers (1) dan (2), jadinya:

20x + 10y = 140.000

 6x + 10y = 84.000

_________________ (—)

         14x = 56.000

             x = 56.000/14

             x = 4.000 ⇒ [Harga wetfood]

Subtitusikan nilai x ke persamaan 6x + 10y = 84.000

6(4.000) + 10y = 84.000

 24.000 + 10y = 84.000

                 10y = 60.000

                    y = 60.000/10

                    y = 6.000 ⇒ [Harga dryfood]

Sekarang kita hitung total belanjaan Hooman; hooman ingin membeli 8kg wetfood dan 20kg dryfood

persamaan nya ⇔ 8x + 20y = .. ?

Subtitusikan nilai x dan y untuk menghitung total belanjaan Hooman

8(4.000) + 20(6.000)

= 32.000 + 120.000

= Rp 152.000,00

Kesimpulan soal

Total belanjaan Hooman adalah Rp 152.000,00.
