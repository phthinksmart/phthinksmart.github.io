16. dik 40 HCl 0,2 diencerkan sehingga volumenya 200ml tentukan : a. PH larutan sebelun di encerkan b. konsentrasi HCl setelah di encerkan c.PH larutan setelah di encerkan


Jawaban:

Pengenceran = suatu cara untuk membuat konsentrasi suatu senyawa turun

Rumus pengenceran :

V₁ x M₁ = V₂ x M₂

V₁ = Banyak volume senyawa yang ingin diencerkan ( l )

M₁ = Molaritas suatu senyawa yang ingin diencerkan ( M )

V₂ = Volume campuran ( total ) antara volume senyawa yang ingin diencerkan dengan volume pengencer ( l )

M₂ = Molaritas suatu senyawa setelah diencerkan ( M )

Pertanyaan :

Diketahui  

V₁ = 40 ml

M₁ = 0,2 M

V₂ = 200 ml ( karena dikatakan di soal hingga volumenya 200 ml )

M₂ = .... M

a. Untuk mencari pH kita harus mencari [ H⁺ ] terlebih dahulu

[ H⁺ ] = M x n

[ H⁺ ] = 0,2 x 1

[ H⁺ ] = 0,2  

[ H⁺ ] = 2 x 10⁻¹

pH = 1 - log 2

b. Konsentrasi HCl setelah diencerkan

V₁ x M₁ = V₂ x M₂

0,04 x 0,2 = 0,2 x M₂

0,008 = 0,2 x M₂

M₂ = 0,04 M

c. pH larutan setelah di encerkan

M₂ = 0,04

[ H⁺ ] = M x n  

[ H⁺ ] = 0,04 x 1

[ H⁺ ] = 0,04

[ H⁺ ] = 4 x 10⁻²

pH = 2 - log 4

Mapel = Kimia

Kelas = 11

Materi = Pengenceran
