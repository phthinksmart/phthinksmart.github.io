---------------------------------
> KESIMPULAN SOAL <
---------------------------------

• Nilai Subtitusi X

• Nilai Y
3X + 2Y = 280000
3(210000 - 3Y) + 2Y = 280000
630000 - 9Y + 2Y = 280000
630000 - 7Y = 280000
-7Y = 280000 - 630000
-7Y = -350000
Y = -350000/-7
Y = 50000

• Nilai X (Celana)

X + 3Y = 210000
X + (3.50000) = 210000
X + 150000 = 210000
X = 210000 - 150000
X = 60000

30. Harga 3 celana dan 2 baju adalah rp280.000,00. sedangkan harga 1 celana dan 3 baju di tempat dan model yang sama adalah rp210.000,00. harga sebuah celana adalah … .?


Untuk menyelesaikan masalah ini, kita perlu menggunakan sistem persamaan linear dua variabel. Misalkan x adalah harga sebuah celana dan y adalah harga sebuah baju. Maka dari soal, kita dapat membentuk sistem persamaan:

3x + 2y = 280.000

x + 3y = 210.000

Kita dapat menyelesaikan sistem persamaan ini dengan menggunakan metode eliminasi:

-3( x + 3y = 210.000) ==> -3x - 9y = -630.000

3x + 2y = 280.000

------------------------

-7y = -350.000

y = 50.000

Setelah menemukan nilai y, kita dapat menghitung nilai x dengan memasukkannya ke salah satu persamaan. Misalnya, kita gunakan persamaan kedua:

x + 3y = 210.000

x + 3(50.000) = 210.000

x = 60.000

Jadi, harga sebuah celana adalah Rp60.000.
