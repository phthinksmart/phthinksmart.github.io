13. Jumlah usia honda dan bambang adalah 93. Jika usia honda 3 tahun yang lalu sama dengan usia bambang 8 tahun dari sekarang, berapa usia honda sekarang? sepi...


Usia Honda sekarang adalah 52 tahun

PENJELASAN

Misal:

Usia Honda = xUsia Bambang = y

Diketahui:

Jumlah usia x dan y adalah 93Usia x - 3 = y + 8

Ditanyakan:

Usia x

Jawab:

Jumlah x dan y adalah 93

=> x + y = 93  (I)

x - 3 = y + 8 (II)

=> x = y + 8 + 3

=> x = y + 11 (III)

Substitusikan III ke I

x + y = 93

y + 11 + y = 93

2y + 11 = 93

2y = 93 - 11

2y = 82

y = 82/2

y = 41

Nilai y (usia Bambang) adalah 41

Substitusikan nilai y ke III

x = y + 11

x = 41 + 11

x = 52

Nilai x (usia Honda) adalah 52

PEMBUKTIAN

x + y = 93

52 + 41 = 93

93 = 93  (Benar)

x - 3 = y + 8

52 - 3 = 41 + 8

49 = 49  (Benar)

KESIMPULAN

Usia Honda sekarang adalah 52 tahun

17. Negara XYZ pada tahun 2010 memiliki data pendapatan nasional sebagai berikut iuran ansuransi 3.000.000laba ditahan 2.000.000pajak perseorangan 1.000.000depresi 3.000.000trasfer payment 5.000.000pajak tidak langsung 1.000.000NNI 90.000.000pajak langsung 500.000 berdasarkan disposable income (DI) pada negara XYZ tahun 2010 adalah​


DIKETAHUI

iuran ansuransi: 3.000.000

laba ditahan 2.000.000

pajak perseorangan 1.000.000

depresi 3.000.000

trasfer payment 5.000.000

pajak tidak langsung 1.000.000

NNI 90.000.000

pajak langsung 500.000

DITANYA

Disposable Incomenya ?

DIJAWAB

DI = PI - Pajak langsung

PI = NNI - Iuran Asuransi - laba ditahan + transfer payment

PI = (90.000.000 - 3.000.000)( - 2.000.000 + 5.000.000)

PI = 87.000.000 + 3.000.000

PI = 90.000.000

DI = PI - Pajak langsung

DI = 90.000.000 - 500.000

DI = 89.500.000

Jadi, diperoleh disposible income di negara XYZ pada tahun 2010 adalah Rp 89.500.000,-

(a) Laba kena pajak untuk tahun 2010 dapat dihitung dengan menggunakan rumus: Laba kena pajak = Laba sebelum pajak - Beban pajak penghasilan.

Berdasarkan informasi yang diberikan, tarif pajak untuk tahun 2010 adalah 40%. Oleh karena itu, beban pajak penghasilan dapat dihitung dengan mengalikan laba sebelum pajak dengan tarif pajak tersebut:

Beban pajak penghasilan = Laba sebelum pajak x Tarif pajak

= Laba sebelum pajak x 40%

= $800,000 x 40%

= $320,000

Dengan demikian, laba kena pajak untuk tahun 2010 adalah:

Laba kena pajak = Laba sebelum pajak - Beban pajak penghasilan

= $800,000 - $320,000

= $480,000

(b) Laba sebelum pajak untuk tahun 2010 dapat dihitung dengan menambahkan beban pajak penghasilan dengan laba kena pajak:

= $480,000 + $320,000

= $800,000

Jawaban:

(a) Laba kena pajak untuk tahun 2010 dapat dihitung sebagai berikut:

Pendapatan kena pajak = Laba sebelum pajak penghasilan + Bunga obligasi - (Sewa yang diterima di muka - Pendapatan yang belum diterima)

= Laba sebelum pajak penghasilan + $10,000 - ($60,000 - $4)

= Laba sebelum pajak penghasilan + $6,004

= Laba sebelum pajak penghasilan + ($120,000 - $30,000) + $10,000 - ($60,000 - $4)

= Laba sebelum pajak penghasilan + $40,004

= Laba sebelum pajak penghasilan + ($1,000,000 - $400,000) + $10,000 - ($60,000 - $4)

= $550,004

Pajak penghasilan untuk tahun 2010 = $550,004 x 40% = $220,001.60

(b) Laba sebelum pajak penghasilan untuk tahun 2010 dapat dihitung sebagai berikut:

Laba sebelum pajak penghasilan = Pendapatan kena pajak - Bunga obligasi + Sewa yang diterima di muka - Pendapatan yang belum diterima

= $550,004 - $10,000 + ($60,000 - $4)

= $600,000
