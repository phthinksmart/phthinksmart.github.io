12. Sebuah koin dilempar sebanyak 100 kali. Jika mata koin Angka muncul 36kali, maka peluang empirik kemunculan mata koin bukan Angka adalah​


diket:

muncul 100x atau n (S)

angka 36x

ditanya:

peluang empirik bukan angka

jawab:

rumus peluang empirik

n (P) = n (A) / n (S)

n (S) = 100

n (A) = 100-36

= 64

n (P) = n (A) / n (S)

n (P) = 64 / 100

n (P) = 16 /25
