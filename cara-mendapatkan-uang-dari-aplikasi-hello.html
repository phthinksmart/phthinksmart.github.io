3. Hello dan Panda memiliki uang dengan perbandingan 3:5. Jika Hello dan Panda memiliki uang dengan jumlah Rp. 400.000,00 maka berapakah jumlah uang masing-masing Hello dan Panda?


Jawaban:

Hello = Rp 150.000

Panda = Rp 250.000

Penjelasan dengan langkah-langkah:

Hello : Panda = 3 : 5

Jumlah perbandingan = 3 + 5 = 8

Jumlah uang = Rp 400.000

Hello = 3/8 x Rp 400.000

= 3 x Rp 50.000

= Rp 150.000

Panda = 5/8 x Rp 400.000

= 5 x Rp 50.000

= Rp 250.000

Jawab:

hello : Rp.150.000.00

panda : Rp.250.000.00

Penjelasan dengan langkah-langkah:

[tex]dik :\\ H:P = 3:5 \\ jumlah :Rp400.000.00\\ \\ jawab:\\ H=\frac{3}{3+5} * 400.000\\ H=\frac{3}{8} *400.000\\ H = 150.000\\\\P=400.000-150.000=250.000[/tex]

Kode Program:

Imports System

Module ModuleKu

   Sub Main()

       Console.WriteLine("Hello World!")

   End Sub  

End Module

Penjelasan:

Ayo Kita Bahas Satu Persatu Artinya Ya.

1. Imports System

Imports System Berarti Kita Mengimpor Semua Kelas, Fungsi Dan Konstanta / Variabel Didalam Namespace system Yang Sudah Disediakan Oleh NET. Framework.

2. Module ModuleKu

Module ModuleKu Berarti Kita Mendefinisikan Module Yang Bernama "ModuleKu". Module Ini Berisi Semua Variabel Dan Fungsi Yang Kita Butuhkan.

3. Sub Main()

Ini Berarti Kita Mendefinisikan Fungsi Main() Didalam Module ModuleKu. Sub Main() Artinya Fungsi Utama. Program Akan Berjalan Mulai Dari Sini.

4. Console.WriteLine("Hello World!")

Console.WriteLine("Hello World!") Berarti Kita Akan Menulis Output "Hello World!" Ke Konsol.

5. End Sub

End Sub Menandakan Akhir Dari Fungsi Yang Kita Buat, Dalam Hal Ini Adalah Fungsi Main().

6. End Module

End Module Menandakan Akhir Dari Module Yang Kita Buat, Dalam Hal Ini Adalah Module ModuleKu.

24. cara buat hello world di java


Jawaban:

1.      Buka NetBeans

2.      Membuat New Project yang akan diberi nama Hello.Java

3.      Masuklah pada menu editor. Kemudian inputkan kode

System.out.println(“Hello World”) di papan editor/console yang telah disediakan tersebut.

4.      Terakhir jalankan kode dan tarraaaaa kode berhasil dijalankanJ

J NOTES

·       Saat menjalankan program java seperti pada program Hello world tersebut diharuskan untuk mengintallisasi JDK (Java Development Kit) atau JRE (Java Runtime Environtment) yang mempunyai fungsi untuk mengkompilasi kode-kode java menjadi aplikasi java. Lalu selanjutnya mengistall IDE NetBeans, Enclipse, Notepad, Notepad++, dsb. Setelah itu jalankan program sesuai index struktur bahasa Java itu sendiri dengan benar dan tepat

·         Aturan penulisan program pada java:

1.      Nama file harus sama dengan nama kelas pada program. Contohnya tadi kita membuat nama file berupa Hello.java , maka nama kelasnya menjadi Hello

2.      Nama file/Program Java yang dibuat harus selalu diakhiri dengan ekstensi file .java

3.      Hanya boleh ada satu kelas public pada sebuah file

4.      Kelas yang menjadi program harus memiliki metode public static void main(String[]args)

5.      Di setiap akhiran statemen pada java diberi tanda ; (titik koma)
