2. Presto menghasilkan tembahan tekanan uap sekitar 1 atm. Dengan demikian, tekanan uap di dalam presto sekitar 2 atm. Jika penutup presto memiliki luas penampang sekitar 300 cm2, berapa gaya tekan ke luar pada tutup presto.​


Jika penutup presto memiliki luas penampang sekitar 300 cm² , maka gaya tekan ke luar pada tutup presto adalah 6000 N

PEMBAHASAN

Untuk soal ini kita akan memakai rumus dasar dari tekanan berikut ini :

Tekanan = Gaya / Luas Penampang

atau di simbolkan :

P = F / A

P = Tekanan ( Pa )

F = Gaya Tekan ( N )

A = Luas Penampang ( m² )

Diketahui :

Tekanan Uap = P = 2 atm = 2 x 10⁵ Pa

Luas Penampang = A = 300 cm² = 0,03 m²

Ditanyakan :

Gaya Tekan Tutup Presto = F = ?

Penyelesaian :

Kita akan langsung memakai rumus diatas untuk mencari gaya tekan ke luar pada tutup Presto berikut ini :

F = P . A

F = 2 x 10⁵ . 0,03

F = 6000 N

Kesimpulan :

Tekanan uap yang di hasilkan oleh presto akan menekan ke luar pada tutup presto sebesar 6000 N
