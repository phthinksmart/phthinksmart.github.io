Misalkan,

x = 1 kg anggury = 1 kg delimaz = 1 kg jambu

Bentuk persamaan linear,

4x + y + 2z = 54.000 ..................... (1)x + 2y + 2z = 43.000 ..................... (2)3x + y + z = 37.750 .......................... (3)

Dieliminasikan nilai x pada persamaan (1) dan (2) untuk menyisakan nilai y dan nilai z.

4x + y + 2z = 54.000 >>>>>>>> dikalikan 1x + 2y + 2z = 43.000 >>>>>>>> dikalikan 44x + y + 2z = 54.0004x + 8y + 8z = 172.000 _>>> -7y - 6z = -118.000 ................. (4)

Dieliminasikan nilai x pada persamaan (1) dan (3) untuk menyisakan nilai y dan nilai z.

4x + y + 2z = 54.000 >>>>>>>> dikalikan 33x + y + z = 37.750 >>>>>>>>>> dikalikan 412x + 3y + 6z = 162.00012x + 4y + 4z = 151.000 _>>> -y + 2z = 11.000 ..................... (5)

Dieliminasikan nilai y pada persamaan (4) dan (5) untuk mengetahui nilai z.

-7y - 6z = -118.000 >>>>>>>>> dikalikan (-1)-y + 2z = 11.000 >>>>>>>>>>> dikalikan (-7)7y + 6z = 118.0007y - 14z = -77.000 _>>> 20z = 195.000>>> z = 195.000/20>>> z = 9.750

Dieliminasikan nilai z pada persamaan (4) dan (5) untuk mengetahui nilai y.

-7y - 6z = -118.000 >>>>>>>> dikalikan (-1)-y + 2z = 11.000 >>>>>>>>>> dikalikan 3 7y + 6z = 118.000-3y + 6z = 33.000 _>>> 10y = 85.000>>> y = 85.000/10>>> y = 8.500

Disubstitusikan nilai y = 8.500 dan nilai z = 9.750 pada persamaan (3) untuk mengetahui nilai x.

3x + y + z = 37.7503x + 8.500 + 9.750 = 37.7503x + 18.250 = 37.7503x = 37.750 - 18.2503x = 19.500x = 19.500/3x = 6.500

Jadi, himpunan penyelesaian tersebut adalah {x , y , z} = {6.500 , 8.500 , 9.750}.

22. harga 1 kilo jeruk dan 3 kilo Apel adalah Rp90.000 sedangkan harga 2 kilo jeruk dan 1 kilo Apel adalah Rp55.000 Berapakah harga 3 kilo jeruk dan 1 kilo apel​


Jawaban:

Diketahui :

1 kilo jeruk + 3 kilo Apel = Rp. 90,000

2 kilo jeruk + 1 kilo Apel = Rp. 55,000

Ditanyakan :

Harga 3 kilo jeruk + 1 kilo Apel ?

Jawab :

Soal ini membahas tentang sistem persamaan linear dua variabel atau SPLDV Didalam SPLDV terdapat beberapa metode yang digunakan yaitu metode eliminasi subsitusi grafik determinan dll..

Kita akan gunakan metode Eliminasi untuk mengerjakan ini , maka ;

Model matematika soal diatas adalah :

misal :

x = jeruk

y = Apel

maka,

x + 3y = 90,000

2x + y = 55,000

Metode Eliminasi :

eliminasi x untuk dapat kan nilai y maka :

x + 3y = 90,000 |×2

2x + y = 55,000 |×1

2x + 6y = 180,000

2x + y = 55,000

-------------------------- (-)

5y = 125,000

y = 125,000 : 5

y = 25,000

substitusi kan y = 25,000 ke :

2x + y = 55,000

2x + (25,000) = 55,000

2x = 55,000 - 25,000

2x = 30,000

x = 30,000 : 2

x = 15,000

jadi, harga

x = jeruk 1 kilonya Rp. 15,000

y = Apel 1 kilonya Rp. 25,000

maka, total harga dari :

= 3 kilo jeruk + 1 kilo Apel

= 3x + y

= 3(15,000) + (25,000)

= 45,000 + 25,000

= Rp. 70,000

semoga membantu

24. 2 kilo dan 1 kilo apel dengan harga Rp15.000 sedangkan Ayah membeli 1 kilo mangga dan 2 kilo apel seharga Rp18.000 Berapakah harga 5 kilo mangga dan 3 kilo apel


materi: Sistem persamaan linear dua variabel
kelas: 2 SMP

Diketahui:
2 kilo mangga + 1 kilo apel = 15.000
1 kilo mangga + 2 kilo apel = 18.000

Ditanya:
5 kg mangga + 3 kg apel

Jawab:

2 kilo mangga + 1 kilo apel = 15.000
1 kilo mangga = 18.000 - 2 kg apel

maka,

2 (18.000 - 2 kilo apel) + 1 kilo apel = 15.000
36.000 - 4 kilo apel + 1 kilo apel = 15.000
- 4 kilo apel + 1 kilo apel = 15.000 - 36.000
- 3 kilo apel = - 21.000
1 kilo apel = - 21.000 / 3
1 kilo apel = 7.000

2 kilo mangga + 7.000 = 15.000
2 kilo mangga = 15.000 - 7.000
2 kilo mangga = 8.000
1 kilo mangga = 8.000 / 2
1 kilo mangga = 4.000

jadi,
5 kilo mangga + 3 kilo apel =
5 (4.000) + 3 (7.000)
20.000 + 21.000
41.000

jadi, harga 5 kilo mangga dan 3 kilo apel adalah 41.000

29. harga 3 kilo Mangga 2 kilo jeruk dan 1 kilo anggur adalah 85 harga 1 kilo Mangga 2 kilo jeruk dan 2 kilo anggur adalah 86 harga 2 kilo Mangga 2 kilo jeruk dan 3 kilo anggur adalah 123 Berapakah harga per kilo masing-masing 2 tersebut jika ingin membeli 2 kilo Mangga 3 kilo jeruk dan 1 kilo anggur maka total biaya yang harus dikeluarkan adalah


Penjelasan dengan langkah-langkah:

Misalkan :

x = mangga

y = jeruk

z = anggur

maka akan didapatkan sistem persamaan sebagai berikut.

3x + 2y + z = 85 ... pers (1)

x + 2y + 2z = 86 ... pers (2)

2x + 2y + 3z = 123 ... pers (3)

Eliminasi pers 1 & 2

3x + 2y + z = 85

x + 2y + 2z = 86 -

2x - z = -1 ...pers (4)

Eliminasi pers 2&3

x + 2y + 2z = 86

2x + 2y + 3z = 123 -

-x-z = -37 ... pers (5)

Eliminasi pers 4&5

2x - z = -1

-x - z = -37 -

3x = 36

x = 12

Subtitusikan x = 12 ke pers (5)

-x-z = -37

-(12)-z = -37

-12+37 = z

25 = z

Subtitusikan x=12 dan z=25 ke pers (2)

x + 2y + 2z = 86

12 + 2y + 2(25) = 86

12 + 2y + 50 = 86

2y + 62 = 86

2y = 86-62

2y = 24

y = 12

harga 1 kilo mangga = 12.000

harga 1 kilo jeruk = 25.000

harga 1 kilo anggur = 12.000

2x + 3y + z = ?

2(12) + 3(25) + 12 = ..

24 + 75 + 12 = 111

jadi, total biaya yang harus dikeluarkan untuk membeli 2 kilo Mangga 3 kilo jeruk dan 1 kilo anggur adalah Rp. 111.000
