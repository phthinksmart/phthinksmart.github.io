1. sebuah perusahaan pelayaran akan mengangkut 160 sedan dan 330 bus dengan 2 kapal Feri. Jika feri a mengangkut 20 sedan dan 30 bus untuk sekali angkut dan feri b mengangkut 20 sedan dan 60 bus untuk sekali angkut. jika biaya biaya angkut menggunakan feri B setengah dari feri A, banyaj masing masing feri yang harus di sewa, dengan biaya minimum? A. 8 feri A saja. B. 8 Feri B saja. C. 3 Feri A dan 5 Feri B. D. 5 feri A dan 3 feri B. E. 3 Feri A dan 3 feri B


misal x = feri A, y = feri B
20x + 20y = 160 --> x + y = 8
30x + 60y = 330 --> x + 2y = 11
----------------------------------------- -
y = 3
x = 5
tipot (5,3)

x + y = 8
x = 0, y = 8
y = 0, x = 8
tikor (0,8) dan (8,0)

x + 2y = 11
x = 0, y = 11/2
y = 0, x = 11
tikor (0,11/2) dan (11,0)

biaya angkut y = 1/2x
misal biaya angkut x = 1 juta, y = 500 ribu
(5,3) --> (5 x 1 juta + 3 x 500 ribu) = 6,5 juta
(0,8) --> (8 x 500 ribu) = 4 juta ✓
(11,0) --> (1 juta x 11) = 11 juta

biaya minimum = 4 juta, berarti yg disewa 8 kapal feri B aja

30. sebuah perusahaan pelayaran hendak mengangkut 420 mobil sedan dan 120 bus dengan 2 kapal feri, yaitu feri A dan feri B. feri A mengangkut 30 bus dan 30 sedan, sedangkan feri B dapat mengangkut 70 sedan dan 10 bis. untuk sekali angkut, feri A memerlukan biaya 50juta, feri B memerlukan biaya 30jutatentukan biaya minimum yang dikeluarkan untuk mengangkut semua kendaraan.mohon bantuannya ya. caranya tolong dijabarkan


Pembahasan :

Misal :
x = kapal feri A
y = kapal feri B

Untuk memudahkan membuat model matematikanya kita buat tabel sebagai berikut
............... feri A (x) .. feri B (y)
sedan ....... 30x .......... 70y ..... 420
bus ............ 30x .......... 10y ..... 120
Biaya ......... 50x .......... 30y ..... ??? dalam juta rupiah

1) sedan
=> 30x + 70y ≥ 420
=> 3x + 7y ≥ 42 ........ (1)
(diarsir ke atas)
Jika x = 0 maka y = 42/7 = 6
(0, 6)
Jika y = 0 maka x = 42/3 = 14
(14, 0)

2) bus
=> 30x + 10y ≥ 120
=> 3x + y ≥ 12 .......... (2)
(diarsir ke atas)
Jika x = 0 maka y = 12
(0, 12)
Jika y = 0 maka x = 12/3 = 4
(4, 0)

3) x ≥ 0, y ≥ 0 ........... (3)

Fungsi sasaran => biaya
f(x, y) = 50x + 30y => dalam juta rupiah


Titik potong kedua garis :
3x + 7y = 42
3x + y = 12
----------------- -
...... 6y = 30
...... y = 5

3x + y = 12
3x + 5 = 12
3x = 7
x = 7/3

(7/3, 5)

Berdasarkan grafik (lihat lampiran)
titik-titik sudut yang memenuhi adalah (0, 12), (7/3, 5) dan (14, 0)

Kita substitusikan ke fungsi sasaran
f(x, y) = 50x + 30y
f(0, 12) = 50(0) + 30(12) = 360 juta
f(7/3, 5) = 50(7/3) + 30(5) = 266,67 juta
f(14, 0) = 50(14) + 30(0) = 700 juta

Jadi biaya minimum yang dikeluarkan untuk mengangkut semua kendaraan 266,67 juta rupiah
= Rp266.666.666,67
